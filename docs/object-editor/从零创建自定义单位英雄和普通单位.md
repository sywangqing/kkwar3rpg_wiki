# 从零创建自定义单位：英雄和普通单位

## 什么是单位？——英雄与普通单位的区别

在本节中，你将了解 Warcraft III 地图编辑器中最基本的概念——**单位（Unit）**，以及英雄单位和普通单位之间的核心区别。学完本节后，你将能够区分这两种单位类型，并理解为什么你的地图需要它们。

### 单位的定义与作用

**单位**是你在地图上可以选中、移动、攻击或控制的任何角色。在地图编辑器中，所有的人物、怪物、士兵都可以统称为"单位"[^2]。

单位的**作用**很简单：
- 推动游戏剧情发展
- 供玩家操控（玩家单位）
- 作为敌人或障碍（中立/敌方单位）

### 英雄单位的特殊属性

英雄是特殊的单位，它们拥有普通单位没有的关键特性：

1. **等级系统** — 英雄可以通过击杀敌人或完成任务获得经验值升级，每次升级会提升属性[^1]
2. **独立模型** — 每个英雄都有独特的3D模型，方便玩家区分[^5]
3. **可选取图标** — 英雄会在屏幕右下角显示头像，方便玩家操作[^3]
4. **复活机制** — 英雄死亡后会在基地自动复活，普通单位则永久死亡

> **💡 新手提示**：英雄单位和普通单位在编辑器中是由不同的分类管理的，但它们都使用**单位编辑器（Unit Editor）**来创建和修改[^2]。

> **⚠️ 常见错误**：很多新手把自定义英雄放到另一张地图时发现找不到，这是因为自定义单位需要**导出/导入**整个单位数据，不能直接复制模型文件[^7]。解决办法是使用地图管理器导出地图，然后在新地图中导入。

### 小结

现在你已经理解：
- 单位是地图中所有可操控角色的统称
- 英雄单位有等级、复活、独立模型等特殊功能
- 普通单位通常作为士兵、怪物或背景角色

下一步，我们将学习如何**在单位编辑器中创建你的第一个自定义单位**！

## Object Editor 物体编辑器入门

本节将带你认识**物体编辑器（Object Editor）**——这是制作自定义单位的核心工具。学完本节后，你将能够打开物体编辑器，并了解界面上各个按钮的作用，为创建第一个自定义单位做好准备。

### 如何打开物体编辑器

1. **第一步：启动世界编辑器** — 打开 Warcraft III 冰封王座，在主菜单点击"自定义游戏"，然后双击"World Editor"（世界编辑器）来启动[^2]

2. **第二步：找到物体编辑器入口** — 编辑器顶部有一排标签栏，点击**"物体编辑器"**按钮（一个立方体图标）[^2]

3. **第三步：等待加载完成** — 首次打开时，加载可能需要几秒钟，耐心等待即可

> **💡 新手提示**：如果你在标签栏找不到物体编辑器，可能是因为打开了其他编辑器窗口。此时可以点击菜单栏的**"视图"→"物体编辑器"**来重新打开。

### 编辑器界面布局介绍

物体编辑器打开后，你会看到分为**左右两个主要区域**：

- **左侧面板（单位列表）**：显示所有可用的单位类型。你可以在这里选择"英雄"或"普通单位"作为基础模板进行修改[^4]

- **右侧面板（属性面板）**：当你选中某个单位后，这里会显示该单位的所有属性参数，如生命值、攻击伤害、移动速度等

> **⚠️ 常见错误**：很多新手双击左侧单位后，发现右侧什么都没显示。这是因为你需要**单击选中**单位（蓝色高亮），而不是双击。双击通常是预览功能。

在属性面板顶部，有几个**标签页**用于切换不同类型的编辑内容：
- **单位** — 修改单位的基础属性
- **技能** — 为单位添加技能
- **护甲/攻击** — 调整战斗相关参数

> **💡 新手提示**：顶部工具栏的快捷键很有用！把鼠标悬停在按钮上，会显示提示文字和快捷键。例如按 `F6` 可以快速打开物体编辑器[^3]。

### 小结

完成以上步骤后，你应该已经成功打开了物体编辑器，并认识了界面的基本布局。现在你可以尝试在左侧列表中找到"步兵"或"死亡骑士"等单位，单击选中它们，观察右侧属性面板的变化。这些信息将帮助你在下一节学习如何修改单位的属性值。

## 创建第一个自定义普通单位

在这一节中，我们将学习如何通过复制现有单位来创建第一个自定义普通单位。学完本节后，你将能够根据自己的游戏需求，创造出独特的小兵、怪物或NPC。

### 操作步骤

1. **第一步：打开单位编辑器** — 在世界编辑器（World Editor）顶部菜单栏中，点击"模块"（Module）按钮，然后选择"单位编辑器"（Unit Editor）[^2]。这一步非常重要，因为所有关于单位的创建和修改都在这里完成。

2. **第二步：复制一个现有单位作为基础** — 在单位编辑器左侧面板中，找到你想要作为模板的单位（比如"步兵"），选中后点击左上角的"复制"按钮。这样可以保留原单位的所有基础设定，我们只需要修改想改变的部分[^4]。

3. **第三步：重命名你的新单位** — 复制完成后，新单位会出现在左侧面板的"自定义单位"列表中。选中它，然后在右侧面板中找到"名称"一栏，输入一个容易识别的名字，比如"我的小兵"[^3]。

4. **第四步：设置基础属性** — 在右侧面板中找到"生命值"（Maximum HP）和"攻击力"（Damage）两个重要属性。根据你的游戏设计，调整合适的数值——生命值越高单位越耐打，攻击力越高伤害越大。

5. **第五步：调整移动速度和护甲类型** — 继续在属性面板中找到"移动速度"（Move Speed）和"护甲类型"（Armor Type）。移动速度决定单位跑得快不快，护甲类型则影响单位受到不同攻击时的伤害减免效果[^1]。

> **💡 新手提示**：在修改数值时，不必担心"改坏了怎么办"。你可以随时重新复制一个新单位作为备份，或者使用Ctrl+Z撤销上一步操作。

> **⚠️ 常见错误**：新手经常忘记给重命名后的单位设置快捷键，导致测试地图时无法快速生产这个单位。解决方法是在单位编辑器的快捷键设置中，为单位分配一个英文字母作为快捷键[^3]。

### 小结

完成以上步骤后，你应该已经在单位编辑器中创建了一个带有自定义属性（生命值、攻击力、移动速度、护甲类型）的普通单位。接下来你可以将这个单位放置到地图上，开始测试它的实际表现了！

## 创建英雄单位

在本节中，你将学习如何把一个普通单位变成可升级、拥有技能和属性的英雄单位。学完本节后，你将能够创建属于自己的英雄角色，为后续的技能系统和RPG玩法打下基础。

### 操作步骤

1. **第一步：打开单位编辑器** — 在WE主界面顶部菜单栏中，点击「场景」→「单位编辑器」[^2]。如果你还没有创建过自定义单位，可以先复制一个现有的英雄单位作为模板（比如人族英雄「Paladin」），这样能省去很多基础设置的麻烦。

2. **第二步：设置单位为英雄** — 在单位编辑器中找到你想改造的单位，在右侧属性面板中找到「类别-英雄」选项，将其改为「True」或「是」[^1]。这一步非常关键——只有设为英雄的单位才能获得经验值并升级，否则它永远只是一个普通单位。

3. **第三步：添加技能** — 切换到「技能」标签页，点击「新建技能」按钮。为英雄添加你想要的技能（比如「风暴之锤」「重击」等）。每添加一个技能，记得在弹出的窗口中选择技能的等级范围[^1]。

4. **第四步：设置英雄属性** — 切换到「属性」标签页，你会看到三个核心属性：「力量」「敏捷」「智力」[^1]。每个属性会影响不同的战斗效果：
   - **力量**：增加生命值和生命恢复速度
   - **敏捷**：增加攻击速度和护甲
   - **智力**：增加魔法值和魔法恢复速度

   设置初始值和每级成长值，这两个数值决定了英雄升级时的属性增长幅度。

> **💡 新手提示**：建议先复制一个现有的英雄模板，然后在其基础上修改，这样能避免漏掉一些关键设置。复制方法是：在单位列表中右键点击原单位，选择「复制单位」。

> **⚠️ 常见错误**：很多新手只设置了「英雄=True」就以为完成了，但忘记在地图上实际放置这个单位。完成设置后，一定要回到地形编辑器，在地图中拖入你刚创建的英雄单位，否则游戏运行时根本找不到这个英雄[^7]。

### 小结

完成以上步骤后，你应该能看到地图上出现了一个可升级的英雄角色，它拥有你添加的技能，并且具备三项基础属性。当你进入游戏测试时，英雄击杀敌人可以获得经验，经验条满了会自动升级，属性也会按照你设置的成长值增加。

## 自定义单位外观

本节将教你如何给自定义单位换"皮肤"——包括更换3D模型、修改小图标和战斗头像。学完本节后，你的单位将不再使用默认的兽人或人类模型，而是呈现出你设计的独特外观。

### 更换单位模型

1. **打开单位编辑器** — 在主界面顶部菜单栏点击"模块"→"单位编辑器"[^2]，或者直接按 **F6** 快捷键
2. **选择目标单位** — 在左侧单位列表中找到你想修改的单位，点击选中（如果你还没创建单位，请先参考"创建自定义单位"章节）
3. **定位模型字段** — 在右侧属性面板中找到"美术 - 模型文件"一栏[^5]，这是控制单位外观的关键字段
4. **更换模型** — 点击该字段右侧的省略号按钮 `...`，在弹出的模型选择器中挑选你想要的模型。你也可以直接输入模型路径（如 `war3mapImported\MyHero.mdx`）
5. **预览效果** — 关闭单位编辑器后，在地形编辑器中选中该单位，按 **F9** 可以快速预览模型的显示效果

> **💡 新手提示**：模型文件必须先导入到地图中才能使用！导入模型的方法是在编辑器中按 **Ctrl+M** 打开"导入资源"面板，把 `.mdx` 或 `.mdl` 格式的模型文件拖进去[^5]。

> **⚠️ 常见错误**：很多新手换了模型后发现单位变成"透明"或"不见了"，这通常是因为模型文件路径写错了。请确保路径大小写完全一致，并且文件已经成功导入。

### 修改单位图标和头像

1. **找到图标字段** — 在单位编辑器的属性面板中，定位到"游戏界面 - 图标（编队）"字段[^3]，这是单位在屏幕下方编队栏显示的小图标
2. **选择或更换图标** — 点击省略号按钮，在弹出的图标列表中选择一个替换图标
3. **设置战斗头像** — 找到"游戏界面 - 头像"字段[^3]，这个头像会在单位被选中时显示在屏幕左上角。点击省略号选择合适的头像图片
4. **导入自定义图标** — 如果想用自己设计的图标，需要先在"导入资源"面板（**Ctrl+M**）中添加 `.blp` 或 `.tga` 格式的图片，然后在图标字段中通过"自定义"选项找到你导入的图片

> **💡 新手提示**：图标和头像的尺寸很重要！一般小图标推荐使用 256×256 像素，头像推荐使用 512×512 像素，否则图片可能会被编辑器自动拉伸模糊。

> **⚠️ 常见错误**：新手经常混淆"图标"和"头像"——图标是那个 **64×64** 大小的小图，头像是选中单位时左上角显示的大图。如果你的单位在编队栏显示不对，请检查"图标（编队）"字段；如果选中后看不到头像，请检查"头像"字段。

### 小结

完成以上步骤后，你的自定义单位应该拥有了独特的外观：3D模型在地图上栩栩如生，小图标在编队栏清晰可见，头像在选中时准确显示。建议回到地形编辑器，在不同角度和距离下观察你的单位，确保外观效果符合预期。

## 测试你的自定义单位

恭喜你完成了自定义单位的创建！现在你需要测试一下它是否工作正常。这一节我们将学习如何保存地图、进入测试模式，以及当出现问题时如何排查。

### 保存地图并进入测试模式

1. **第一步：保存地图** — 按下键盘的 `Ctrl + S` 组合键，或点击顶部菜单栏的「文件」→「保存」[^2]。首次保存时，系统会弹出对话框让你选择保存位置和文件名，建议起一个容易记住的名字，比如"我的第一个RPG"。

2. **第二步：打开测试模式** — 保存完成后，按下键盘的 `F9` 键，这是进入测试模式（也称为"模拟游戏"）的快捷键[^3]。你也可以点击菜单栏的「测试」→「开始游戏」选项。编辑器会弹出一个新窗口，启动魔兽争霸3游戏并加载你的地图。

3. **第三步：召唤你的单位** — 游戏加载后，按 `F9` 打开作弊码输入框（测试专用的功能）[^6]。输入 `-create` 然后按回车，如果你的单位设置正确，它应该会出现在屏幕上。你可以控制这个单位四处移动，观察它的属性和技能是否显示正常。

4. **第四步：退出测试** — 测试完毕后，直接关闭窗口或按 `Esc` 键，选择「退出」，即可返回到编辑器界面继续修改。

> **💡 新手提示**：测试时建议使用单人游戏（默认就是单人），不要选择多人模式，因为多人模式需要其他玩家连接，会导致无法正常测试。

> **⚠️ 常见错误**：很多新手发现测试时根本找不到自己的自定义单位。这是因为单位虽然创建好了，但还没有把它放到地图上！你需要在左侧的「单位面板」中找到你的单位，然后把它拖到地图的某个位置，否则游戏里当然看不到它。

### 常见问题与调试方法

**问题一：单位不显示或显示为紫色问号**

这是最常见的问题之一，通常是因为单位的"编辑器前缀"设置不正确[^7]。打开单位编辑器，检查你创建的单位，确保它的「名称」字段填写了内容，而不是留空。如果显示紫色问号，说明游戏找不到这个单位的显示数据。

**问题二：单位没有技能**

你可能在单位编辑器里设置了技能，但游戏里看不到。检查以下两点：第一，技能是否正确添加到了单位的「技能」列表中（不是只创建了技能就完了）；第二，技能图标是否正确加载，如果图标路径错误，也会导致技能不显示[^5]。

**问题三：单位属性没有生效**

你可能设置了攻击力和生命值，但实际测试时发现数值不对。请仔细检查「数据」选项卡中的数值是否填在了正确的字段里。有时候你修改了某个字段，但编辑器里显示的是另一个值——这种情况下，建议把光标悬停在字段上，查看提示信息确认这个字段的含义。

> **⚠️ 常见错误**：新手经常反复修改数值，但测试时发现数值没有变化。这是因为每次修改后都必须重新保存地图（Ctrl+S），然后重新进入测试模式！游戏是加载你上次保存时的数据，不会实时反映编辑器的修改。

### 小结

完成以上步骤后，你应该：
- ✅ 成功保存并进入测试模式
- ✅ 在地图上看到你的自定义单位
- ✅ 能够控制单位移动和释放技能
- ✅ 学会通过检查设置来排查常见问题

如果测试过程中遇到任何问题，不要气馁——这是每个地图创作者的必经之路。多尝试几次，你会发现问题越来越容易解决！
