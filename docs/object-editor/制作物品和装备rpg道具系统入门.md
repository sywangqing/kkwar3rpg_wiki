# 制作物品和装备：RPG道具系统入门

## 概述

在RPG游戏中，道具是玩家体验的重要组成部分。想象一下，你刚刚击败了一只怪物，屏幕上弹出"获得铁剑"的提示——这就是道具系统的作用。在本节中，你将学习什么是道具系统，以及如何在地图编辑器中创建你的第一个自定义物品[^4]。

### 什么是道具？

简单来说，**道具就是游戏中可以使用的物品**。根据用途不同，道具大致分为三种类型[^4]：

- **使用类道具**：用完就会消失的物品，比如恢复生命值的药水。玩家点击使用后，数量会减少。
- **装备类道具**：可以长期佩戴的物品，比如武器和防具。装备后会持续提供属性加成（如增加攻击力、防御力等）[^2]。
- **情节类道具**：用于推动剧情发展的物品，比如某个任务需要的关键物品。

> **💡 新手提示**：理解这三种类型的区别很重要！在设计游戏时，先想清楚你要创建的道具属于哪一类，因为不同类型的道具在后续设置上有所不同。

### 在编辑器中找到物品面板

在正式开始之前，我们需要先找到物品编辑的入口。打开你的地图编辑器后，按以下步骤操作：

1. **第一步：打开自定义物品面板** — 在编辑器左下角找到【自定义物品】选项[^1]，将鼠标移动到上面
2. **第二步：新建物品** — 右键点击【自定义物品】，在弹出的菜单中选择【新建自定义物品】[^1][^3]

> **⚠️ 常见错误**：很多新手找不到物品面板在哪里。记住，物品面板在编辑器**左下角区域**，不是在上方的菜单栏里。如果你的编辑器界面被缩小了，可能看不到这个面板，可以尝试按 `Ctrl+鼠标滚轮` 来调整缩放比例。

### 小结

完成以上步骤后，你应该能在左侧面板中看到新建的物品条目，接下来我们就可以开始设置物品的名称、图标和属性了。准备好你的第一个自定义道具吧！
