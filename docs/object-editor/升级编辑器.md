# 升级编辑器

## 什么是升级编辑器

升级编辑器是魔兽争霸3地图编辑器中用于创建和管理"升级"系统的工具。升级可以让单位在满足特定条件后获得属性提升或解锁新能力——比如让英雄的攻击力+5，或者让士兵学会新技能。

通过学习本节，你将理解升级编辑器在对象编辑器中的位置，以及升级系统与其他游戏对象之间的关联。

### 升级编辑器在对象编辑器中的位置

升级编辑器位于对象编辑器（Object Editor）的左侧工具栏中，与单位编辑器、技能编辑器等并列，属于WE的核心组件之一[1]。在WE界面顶部菜单栏找到"模块"（Modules）选项，点击后在展开的列表中即可看到"对象编辑器"。

> **💡 新手提示**：对象编辑器是管理所有游戏内"对象"（单位、技能、物品、升级等）的地方，升级只是其中一小部分功能。

进入对象编辑器后，你会看到一排标签页按钮，升级编辑器（Upgrade）的图标通常是一个带有向上箭头的盾牌形状。点击这个标签，当前窗口会切换到升级列表视图。

### 升级系统与其他对象的关系

升级不是孤立存在的，它与单位、技能等多个对象有密切联系：

1. **升级与单位绑定** — 每个升级都需要指定"哪个单位可以研究这个升级"，比如让步兵营可以研究"强化护甲"
2. **升级影响单位属性** — 当升级生效时，可以修改单位的攻击力、护甲、生命值等数值
3. **升级可解锁技能** — 部分技能需要先研究对应升级才能使用，比如某些终极技能需要先升到特定科技等级

> **⚠️ 常见错误**：新手经常忘记为升级设置"允许研究的单位"，导致玩家无论如何都无法研发这个升级。如果你的升级不生效，首先检查这个设置！

### 小结

现在你应该已经了解：

- 升级编辑器在对象编辑器的哪个位置
- 升级如何与单位、技能等其他对象产生关联

下一步，你可以尝试打开升级编辑器，查看游戏自带的预设升级（如"防御护甲"、"长管火枪"等），了解它们的结构配置方式。

## 升级编辑器基础操作

本节将带你学习如何在魔兽争霸3的**对象编辑器**中创建和修改自定义升级。完成学习后，你将能够为你的RPG地图添加独特的升级系统，让玩家通过升级来强化自己的单位或建筑。

### 操作步骤

1. **打开对象编辑器** — 在主工具栏点击“对象编辑器”按钮（一个立方体图标）[^1]，然后在对象类型列表中选择“升级”类别。你会看到右侧列出了所有预设升级，如“防御卷轴”和“畜牧”。

2. **创建自定义升级** — 在升级列表的空白处**右键点击**，选择“新建升级”。系统会自动生成一个新升级，名字默认为“新建升级（0）”。你也可以按“Ctrl+D”快速复制一个现有的升级作为起点。

3. **设置升级名称和图标** — 选中新建的升级后，在左侧属性面板中找到“名称”一栏，输入你想要的升级名字（如“火焰强化”）。点击“编辑器图标”可以选择一个合适的图标，图标会在游戏内显示给玩家。

4. **配置升级效果** — 在“升级”属性栏中，找到“应用到”选项，点击后可选择这个升级会影响哪些单位。比如你想制作一个“攻击强化”升级，就需要勾选你想强化的单位类型。

5. **设置资源消耗（可选）** — 在属性面板中你可以设置升级需要的**黄金**和**木材**数量，以及升级所需的**时间**。如果不设置，默认消耗为0。

> **💡 新手提示**：在设置升级效果前，先在左侧属性中展开“数据”分支，确保你修改的属性对应当前选择的升级类型。不同类型的升级显示的属性可能略有不同。

> **⚠️ 常见错误**：新手经常忘记在“应用到”中勾选单位，导致升级创建后却无法对任何单位生效。解决方法：完成升级属性设置后，再次检查“应用到”列表是否已正确勾选目标单位。

### 小结

完成以上步骤后，你应该能够在对象编辑器中成功创建一个自定义升级，并将其关联到特定单位身上。你可以尝试为你的RPG角色制作一个简单的“属性强化”升级，测试它在游戏中是否正常工作。

## 升级编辑器核心字段解析

本节我们将学习如何在升级编辑器中设置升级项目的基础参数，包括消耗资源、升级效果和前置条件。完成学习后，你就能为你的RPG地图创建自定义升级系统，让玩家通过消耗资源获得属性提升。

### 操作步骤

1. **第一步：打开升级编辑器** — 在主界面菜单栏点击「编辑器」→ 选择「对象编辑器」[^1]，在对象类型列表中找到「升级」并点击。此时右侧窗口会显示所有已创建的升级项目。

2. **第二步：设置成本字段（金币、木材）** — 选中一个升级项目后，找到「花费 - 金币」和「花费 - 木材」两个字段[^1]。在右侧数值框中输入数字，例如输入 500 表示消耗500金币。木材字段同理，填0表示不需要木材。

3. **第三步：设置升级时间** — 找到「升级时间」字段，输入数值（单位为秒）。数值越大，玩家等待升级完成的时间越长。RPG地图中建议设置为10-30秒，避免玩家等待过久。

4. **第四步：配置升级效果字段** — 在「属性」列表中找到「生命值上限加成」「攻击力加成」「护甲加成」等字段[^1]。例如要增加50点攻击力，在对应字段输入50即可。注意：这些数值是「每次升级」的增量，不是最终总值。

5. **第五步：设置前置科技和等级要求** — 找到「前置升级」字段，点击选择已有的升级项目作为前置条件。找到「等级要求」字段，填入数字表示玩家单位需要达到的最低等级才能研发此升级。

> **💡 新手提示**：如果你想设置「每级消耗递增」，需要在「花费 - 金币」字段中填写多个数值，用逗号分隔，如"100,150,200"表示1-3级分别消耗100/150/200金币。

> **⚠️ 常见错误**：新手经常忽略「可重复研发」字段。默认是「是」，但如果你希望每个升级只能研发一次，必须手动将其改为「否」，否则玩家可以无限叠加属性导致数值失衡。

### 小结

完成以上五个步骤后，你应该已经掌握了一个完整升级项目的核心配置方法。现在你可以回到地图中测试：在触发器中添加「研发升级」动作，或为兵种单位设置默认可研发的升级项目。试试创建一个"力量强化"升级，看看玩家是否需要消耗200金币并等待15秒才能获得+20攻击力的效果。

## 升级编辑器高级应用

本节将带你学习升级编辑器（Upgrade）的三个高级功能：让多个单位共享升级效果、设置科技研究的前置条件、以及自定义升级的图标和名称。掌握这些技巧后，你的RPG地图将拥有更专业的科技树系统，让玩家感受到更丰富的养成乐趣。

### 多个单位同时升级

在RPG地图中，你可能想让多个职业都能通过同一个升级变强，比如“力量训练”能让战士、骑士、野蛮人都获得攻击力提升。

1. **打开对象编辑器（Object Editor）** — 在顶部菜单栏点击“编辑”→“对象编辑器”[^1]
2. **定位到升级分类** — 在左侧列表中找到“升级”类别，点击展开
3. **创建或选择一个升级** — 右键选择“新建升级”，或双击已有升级打开其属性面板
4. **设置适用单位** — 找到“可用种族（Allowed Units）”或类似字段，点击右侧按钮打开单位选择窗口
5. **勾选目标单位** — 在弹出的对话框中，勾选所有你想要让这个升级生效的单位，然后点击“确定”[^1]

> **💡 新手提示**：如果你的升级没有生效，第一时间检查这个字段是否包含了目标单位。很多新手创建了升级却忘了设置适用单位，导致升级对任何单位都不起作用。

> **⚠️ 常见错误**：新手容易混淆“单位可用性”和“升级效果”。前者决定哪个单位能看到并购买这个升级，后者才是实际增加的属性值。两者都需要正确设置！

### 设置升级前置科技

有时候你需要设计“必须先学会A，才能研究B”的科技树逻辑。比如玩家必须先研究“初级魔法”才能研究“中级魔法”。

1. **选中需要设置前置条件的升级** — 在升级列表中双击目标升级
2. **找到前置条件字段** — 在属性面板中找到“前置升级（PreReq）”或“前置科技”字段[^2]
3. **点击编辑按钮** — 该字段右侧通常有一个放大镜或编辑图标
4. **输入前置升级的原始代码** — 在弹出的输入框中，输入玩家必须先完成的那个升级的原始代码（如“Hpal”代表“神圣之光”）
5. **保存并测试** — 关闭编辑器，在游戏中测试：没有研究前置科技时，这个升级应该显示为灰色不可点击状态

> **💡 新手提示**：多个前置条件可以用逗号分隔。例如输入"Hpal,Hcst"表示必须同时完成两个前置升级才能解锁当前升级。

### 升级图标和名称自定义

默认的升级图标可能不符合你地图的主题，比如你想把“防御力+5”改成更直观的图标和中文名称。

1. **在升级属性面板中找到“名称（Name）”字段** — 直接输入你想要的汉字名称，如“铁甲强化”
2. **定位到“图标（Icon - Enabled）”字段** — 这是升级可用时显示的图标
3. **点击右侧按钮打开资源选择器** — 从列表中选择一个适合的图标文件（.blp格式）[^3]
4. **找到“图标 - 禁用（Icon - Disabled）”字段** — 这是升级不可用时（前置条件未满足）显示的灰色图标
5. **同样方式设置禁用状态图标** — 建议选择一个与启用状态相似但颜色偏暗的版本，让玩家一眼看出状态差异

> **⚠️ 常见错误**：新手经常忘记修改“禁用图标”，导致升级不可用时图标仍然显示正常颜色，容易误导玩家。一定要记得同时修改这两个图标！

### 小结

完成以上三个高级功能的设置后，你的RPG地图应该已经具备：
- 一个能让多个相关职业共享的升级效果系统
- 层次分明的科技树，满足“先学A再学B”的设计逻辑
- 专业美观的升级界面，让玩家一目了然

建议你先从最简单的“多个单位同时升级”开始练习，熟悉后再尝试前置科技的设计，最后再加上自定义图标和名称。一步一步来，你一定能做出专业的RPG地图！

## 常见问题与故障排除

本节将帮助你解决编辑器升级配置中最常见的两个问题：升级在游戏中不生效、以及科技树配置错误。通过学习这些排查方法，你将能够独立诊断和修复大部分升级相关的故障。

### 升级在游戏中不生效的解决方法

当你在对象编辑器中配置了单位升级，但测试地图时发现升级选项没有出现或无法生效时，请按以下步骤排查[^1]：

1. **第一步：检查单位是否正确添加了升级按钮** — 打开对象编辑器，点击左侧的"单位"选项，找到你的单位后切换到"命令"标签页，确保"中立敌对"或对应玩家颜色的升级命令按钮已启用[^1]。

2. **第二步：确认科技树（升级）对象已创建** — 在对象编辑器左侧选择"科技树（升级）"类型，点击新建（绿色+号），配置好升级的名称、所需资源和前置科技[^1]。

3. **第三步：建立升级与单位的关联** — 切换到你的单位详情页，找到"科技树"标签页，点击"允许"列下的空格，添加你刚才创建的升级ID（通常是类似"Amim,Amil"的代码格式）[^1]。

4. **第四步：检查玩家是否拥有所需建筑** — 很多升级需要特定的建筑作为前提条件，切换到"科技树（升级）"对象，确保"建筑要求"字段填写了正确的建筑ID。

> **💡 新手提示**：如果你不确定升级的ID是什么，可以选中该升级对象，ID会显示在窗口顶部的"对象名称"后面。

> **⚠️ 常见错误**：很多新手在"科技树"标签页中填错了升级ID格式。正确格式是用逗号分隔的多个ID，例如"AUam,AUav"表示允许两种不同的升级。别忘了每行只能有一个前缀（如"AUam"），后面的逗号和空格要保持一致。

### 科技树配置错误的检查步骤

科技树配置错误会导致玩家无法研发升级或出现游戏崩溃[^2]，请按以下步骤系统检查：

1. **第一步：验证升级的基础属性** — 打开"科技树（升级）"对象，检查"资源耗费"中的黄金和木材数值是否为正整数，负数会导致升级无法正常触发。

2. **第二步：检查前置科技要求** — 在"前置科技"字段中，确保前置升级已经正确创建且ID无误。如果前置科技不存在，后面的升级永远不会出现。

3. **第三步：确认研究时间** — "研究时间"应该设置为一个合理的秒数（建议60-120秒），设置为0可能导致升级瞬间完成但触发异常。

4. **第四步：测试各玩家的科技树** — 使用地图测试功能，按F6打开"游戏开始"触发器，为每个玩家单独测试科技树是否正确显示。

> **💡 新手提示**：创建一个简单的测试触发器：当玩家按ESC键时，给当前玩家添加所有升级。这样可以快速验证升级配置是否有效，而不需要手动建造所有建筑。

> **⚠️ 常见错误**：新手经常忽略"升级分类"设置。如果设置为"魔法"类别但单位没有魔法值上限，升级效果可能无法生效。确保升级效果与目标单位的属性类型匹配。

### 小结

完成以上排查步骤后，你应该能够定位并修复大部分升级配置问题。记住，最常见的三个错误来源是：升级ID格式错误、前置科技未满足、以及单位未正确关联升级[^1][^2]。遇到问题时，从这三个方向逐一检查，通常能找到答案。

## 实战案例：创建自定义升级

本节将通过两个完整案例，教你使用**对象编辑器（Object Editor）** 创建自定义升级效果。学完后，你将能够为你的RPG地图添加独特的升级系统，让玩家通过升级获得属性提升。

### 案例一：创建攻速提升升级

#### 操作步骤

1. **打开对象编辑器** — 在顶部菜单栏点击「文件」→「新建」，或直接按 **F6** 键打开对象编辑器[^1]

2. **定位升级对象** — 在左侧树形菜单中找到「升级」选项，点击展开，你会看到许多预设升级，如「攻击速度提升」「护甲增强」等

3. **创建新升级** — 右键点击「升级」文件夹，选择「新建自定义升级」，系统会生成一个名为「升级0000」的新条目

4. **重命名并设置图标** — 在右侧属性面板中，找到「名称」一栏，将其改为「攻速强化」，同时设置一个易于辨认的图标路径

5. **配置属性加成** — 在「效果 - 攻击速度」属性中，填写你希望的加成数值（如 10、20、30）。这里的数值代表**百分比**，所以填10就代表提升10%攻击速度

6. **设置升级费用和条件** — 调整「升级需要的人口」「黄金费用」「木材费用」等，确保升级成本与游戏平衡相符

> **💡 新手提示**：在设置攻击速度加成时，建议从小数值开始测试（如每次+5%），因为攻速加成效果非常明显，太高的数值可能导致游戏数值失衡。

> **⚠️ 常见错误**：新手容易忽略「升级适用对象」属性。如果你不设置这个选项，升级可能对所有单位生效，导致游戏平衡被破坏。一定要在「可用性 - 升级适用」中指定需要此升级的单位类型。

#### 小结

完成以上步骤后，你已经创建了一个基础的攻击速度升级。接下来需要通过**触发器（Trigger）**或**技能链接**，让玩家能够实际使用这个升级。我们将在后续章节讲解如何将升级与技能树系统连接。

### 案例二：制作多级连锁科技

#### 操作步骤

1. **创建一级升级** — 按照案例一的方法，创建「连锁科技 I」，设置攻击速度+5%的效果，并将升级费用设为较低数值（黄金50、木材0）

2. **创建二级升级** — 创建「连锁科技 II」，设置攻击速度+10%的效果。此升级的关键是添加**前置条件**：右键点击「连锁科技 II」，在「前置升级」属性中选择「连锁科技 I」[^1]

3. **创建三级升级** — 创建「连锁科技 III」，设置攻击速度+20%，前置条件设为「连锁科技 II」，并增加费用

4. **设置升级互斥（如需要）** — 如果你希望这些升级属于同一科技树且只能选择一条路线，可以在「前置条件」中设置互斥逻辑

5. **测试连锁效果** — 按 **F9** 打开测试地图，进入游戏后验证：玩家必须先完成「连锁科技 I」，才能看到「连锁科技 II」的选项

> **💡 新手提示**：连锁科技的前置条件逻辑非常强大。你可以创建复杂的科技树，例如：A → B/C → D/E/F，形成多条升级路线，让玩家有更多策略选择。

> **⚠️ 常见错误**：新手忘记设置「最大等级」属性。默认情况下，升级只能升一级。如果你想让玩家能重复升级同一科技（如攻速+5%可以叠加3次），记得把「最大等级」改为 3 或更高。

#### 小结

通过以上两个案例，你已经掌握了创建单级升级和多级连锁科技的方法。这些基础技能是构建RPG游戏技能树系统的核心。建议你先从简单的单级升级开始练习，熟悉后再尝试创建复杂的连锁科技树。
