# 升级编辑器

## 升级编辑器简介

本节将带你认识**升级编辑器**（Upgrade Editor），这是魔兽争霸III地图编辑器中用于创建和管理"升级系统"的核心工具。学完本节后，你将能够理解什么是升级系统，以及它如何为你的RPG地图增加深度和可玩性。

### 什么是升级系统

在开始使用工具之前，我们先理解"升级"这个概念。

想象一下你玩的RPG游戏：英雄可以通过击败敌人获得经验值，升到更高等级后，生命值、攻击力等属性会变强。这种"属性变强"的机制，就是**升级系统**。[^2]

在魔兽争霸III的地图编辑器里，**升级编辑器**是Object Editor（对象编辑器）中的一个模块，专门用来定义和管理这些属性提升。简单来说，它告诉游戏："当单位升到3级时，攻击力+10；升到5级时，护甲+5。"[^3]

> **💡 新手提示**：升级编辑器不是用来给单位"升级经验值"的，而是用来定义"升级后属性如何变化"。经验值和等级的计算，通常需要配合**触发器**（Trigger）来实现，我们会在后续章节详细介绍。

### 升级编辑器在地图编辑中的用途

升级编辑器在RPG地图制作中非常实用，主要用于以下场景：

1. **创建单位强化路径** — 让你的兵种可以通过升级变得更强，比如"初级步兵"升一级后变成"强化步兵"
2. **设计科技树** — 类似于原版魔兽的科技系统，玩家可以研究升级来解锁新能力
3. **英雄属性成长** — 为RPG地图中的英雄定义每级获得哪些属性加成[^4]

魔兽争霸III World Editor 允许你操作游戏的几乎所有方面[^2]，而升级编辑器正是其中的关键一环。通过它，你可以在不写代码的情况下，为地图添加丰富的成长要素。

> **⚠️ 常见错误**：新手常把"升级"和"技能"混淆。升级（Upgrade）影响的是单位的**基础属性**（如攻击、护甲、移动速度），而技能（Ability）是单位可以使用的**主动或被动效果**（如治疗、闪电链）。两者功能不同，不要混用！

### 小结

现在你已经了解：
- **升级系统**是让单位属性随等级或条件变化而提升的机制
- **升级编辑器**是定义这些变化规则的专用工具
- 它主要用于创建属性成长、科技强化等系统

下一步，你可以尝试打开升级编辑器，浏览预设的升级模板，了解它们是如何工作的！

## 打开升级编辑器

在本节中，你将学习如何打开升级编辑器（Upgrade Editor），这是用于修改单位、科技和物品等属性升级效果的工具。完成本节后，你将能够独立进入升级编辑器界面，并认识基本的界面布局，为后续的属性修改打下基础。

### 操作步骤

1. **第一步：启动 Warcraft III 世界编辑器** — 在你的电脑中找到 `World Editor.exe` 并双击打开[²]。等待编辑器加载完成后，你会看到一个包含多个菜单栏和面板的主界面。

2. **第二步：进入对象编辑器** — 在顶部菜单栏中，找到「文件」旁边的「模块」菜单（Modules）[^3]。将鼠标悬停在此处，会弹出一个下拉列表。选择列表中的「对象编辑器」（Object Editor）选项并点击。这是管理所有游戏对象属性的核心工具[^5]。

3. **第三步：选择升级编辑器** — 对象编辑器打开后，在上方的标签页区域中找到并点击「升级」（Upgrades）标签[^3]。这将切换到升级编辑器视图，在这里你可以看到所有可用的升级项目列表。

### 小结

完成以上步骤后，你应该能看到升级编辑器的主界面，左侧是升级列表（包含单位升级、科技升级等），右侧是所选升级的详细属性面板。你可以点击列表中的任意升级项，在右侧查看和修改其具体数值。

## 创建新升级

在本节中，你将学习如何在魔兽争霸3编辑器中创建一个全新的升级（Upgrade）。学完本节后，你将能够创建属于自己地图的科技升级，并将其应用到你想要的单位身上。

### 操作步骤

1. **第一步：打开对象编辑器** — 在顶部菜单栏点击「高级」→ 选择「对象数据」→ 点击「对象编辑器」[^2]。这是 Warcraft 3 编辑器中专门用来编辑所有游戏对象的地方，包括单位、技能、升级等。

2. **第二步：定位到升级分类** — 在对象编辑器左侧的列表中，找到「升级」分类。点击「升级」左边的「+」展开列表，你会看到编辑器自带的许多预设升级，比如「铁剑」和「防御」等。

3. **第三步：创建新升级** — 确保「升级」分类被选中，然后在右侧的列表面板中点击最上方的空白行。按「Ctrl + D」键即可复制一个预设升级作为模板[^3]。这比从零开始创建要容易得多！

4. **第四步：设置基础属性** — 选中你新建的升级后，在右侧面板中找到以下关键字段：
   - **科技名称** — 这是升级的内部ID，可以改成你自己定义的名称
   - **别名** — 这是玩家在游戏中看到的实际名称
   - **类型** — 选择这个升级影响的是什么属性（如攻击速度、生命值、护甲等）

5. **第五步：配置升级图标** — 向下滚动找到「挂接图标」字段，双击该字段可以打开资源选择器。你可以从编辑器自带的图标库中选择一个代表此升级的图片，或者导入自己设计的自定义图标。

6. **第六步：设置资源消耗** — 找到「gold cost」（金子消耗）和「lumber cost」（木材消耗）两个字段，将它们的值改为你想要的数字。如果不需要某种资源，将其设为「0」即可[^5]。

> **💡 新手提示**：新建的升级默认是关闭状态的！你还需要在「可用升级」字段中为你的单位勾选这个升级，升级才会出现在该单位的科技面板中。

> **⚠️ 常见错误**：很多新手创建完升级后直接去测试，发现单位无法研究该升级。这是因为忘记了给单位分配升级权限！解决方法是在单位属性中找到「可用升级」字段，点击右侧的「...」按钮，然后勾选你刚才创建的升级。

### 小结

完成以上步骤后，你已经成功创建了一个自定义升级。现在你可以在「触发器」中配合使用「研究科技」动作，让单位在满足条件时自动升级，或者添加升级按钮让玩家手动研究。你可以根据需要继续调整升级的效果数值、添加多个等级（只需修改「升级等级最大数量」字段），让你的RPG地图拥有更丰富的科技树！

## 将升级关联到单位

本节你将学习如何在对象编辑器中把升级（也叫"科技"）关联到具体单位，让你的单位能够使用这些升级功能。学完本节后，你就能控制哪些单位可以研究某个升级，以及研究升级需要满足什么前置条件。

### 操作步骤

1. **第一步：打开对象编辑器** — 在顶部菜单栏找到"高级"（或"Advanced"）菜单，点击"对象编辑器"[^2]。在左侧列表中找到并点击"升级"（Upgrades）类别，选中你想要配置的那个升级。

2. **第二步：找到"允许升级的单位"选项卡** — 在升级的属性面板中，切换到"允许升级的单位"（Allowed Units）选项卡[^3]。这里有一个列表，默认可能是空的。你需要点击列表区域，手动输入或选择允许使用这个升级的单位类型代码（比如 "hfoo" 代表人族步兵）。如果留空，则表示没有单位可以使用这个升级！

3. **第三步：设置升级等级和前置科技** — 在升级属性中找到"前置升级"（Requirements）字段[^3]。如果这个升级需要先研究其他科技才能解锁，就在这里填写前置科技的代码。例如"要研究'铁盾'升级，必须先研究'防御'科技"。同时，"等级"（Level）字段决定了升级的最高可研究次数。

4. **第四步：为建筑添加升级选项（可选）** — 如果你想让某种建筑（比如农场）能提供这个升级，需要在"单位编辑器"中找到该建筑单位，在其"升级 - 允许使用"字段中添加这个升级的代码[^5]。这样玩家在建造该建筑后才能研究对应升级。

> **💡 新手提示**：单位类型代码可以在选中单位时看顶部标题栏，比如 "Footman (hfoo)" 中括号里的就是代码。输入时要用代码而非中文名称！

> **⚠️ 常见错误**：很多新手在"允许升级的单位"中留空了，以为默认所有单位都能用。实际上留空意味着**没有任何单位**能用这个升级，必须手动添加单位代码！

### 小结

完成以上步骤后，你配置的升级就已经成功关联到指定的单位了。玩家在使用这些单位时，就能在建筑处看到并研究对应的升级选项。记得检查前置科技是否也正确配置了，否则玩家可能会发现升级选项是灰色的无法点击。

## 升级效果设置

本节将教你在魔兽争霸3编辑器中**设置单位升级效果**，让单位通过升级变得更强。完成后，你的单位可以拥有攻击力、护甲、速度等属性变化！

### 修改攻击力和护甲

在"升级效果设置"中，你可以让单位升级后变得更强壮——比如攻击力从20提升到25，护甲从2变成5。升级效果是通过**升级（Upgrades）** 系统实现的，它决定了单位属性如何变化。

> **💡 新手提示**：升级效果和单位基础属性是两回事！基础属性是单位天生的能力，升级效果是在此基础上的额外加成。

### 操作步骤

1. **打开对象编辑器** — 在编辑器顶部菜单栏点击"对象编辑器"（Object Editor）[^3]

2. **切换到升级选项卡** — 在对象编辑器左侧找到并点击"升级"（Upgrades）图标[^3]

3. **创建一个新升级** — 右键空白区域，选择"新建升级"，给它起一个有意义的名字，比如"重甲训练"

4. **设置影响的单位** — 在右侧面板找到"可升级的单位"字段，点击添加你希望拥有这个升级的单位（比如骑士）

5. **配置攻击力和护甲变化** — 找到"优先级"和"效果"区域，设置具体数值[^5]
   - 护甲：在护甲字段输入"+"号开头的数值，如"+2"表示每级护甲增加2
   - 攻击力：在攻击力字段同样输入"+3"表示每级增加3

6. **设置升级费用和时间** — 向下滚动，找到"升级时间"和"升级费用"，这些决定玩家需要多少资源来研究这个升级[^3]

> **⚠️ 常见错误**：新手常把升级效果值设成绝对值而非增加值！如果你写"10"，编辑器会认为护甲固定是10，而不是在原基础上+10。**务必加上"+"符号**。

### 移动速度和生命值调整

升级不仅可以改变攻击力和护甲，还能调整移动速度和生命值。

1. **继续在升级设置中操作** — 找到"生命值"和"移动速度"字段[^5]

2. **设置生命值提升** — 输入"+100"让单位每级增加100点生命

3. **设置移动速度** — 输入"+20"或"+50"来提升单位移动速度（注意：数值不要设太大，否则单位会跑得"飘"起来）

> **💡 新手提示**：移动速度的增加值是绝对数值，不是百分比！如果想让所有单位按相同比例提速，需要手动计算合适的数值。

### 多个升级等级的效果叠加

当你的升级拥有多个等级时，属性变化会**逐级累加**。这是什么意思呢？

| 升级等级 | 攻击力变化 | 最终攻击力（假设基础20） |
|---------|----------|----------------------|
| 1级     | +5       | 25                   |
| 2级     | +10      | 30                   |
| 3级     | +15      | 35                   |

### 操作步骤

1. **在升级详情中找到"等级数量"** — 设置这个升级最多有几级（比如3级）[^3]

2. **设置每级效果** — 编辑器通常让你分别设置每级的加成数值，或者设置"每级增量"

3. **配置高级升级条件** — 在"前置科技"字段中，可以设置2级需要先升1级，3级需要先升2级[^5]

> **⚠️ 常见错误**：忘记设置前置升级要求！如果不设置，玩家可能直接从1级跳到3级，导致属性跳跃过大。**建议养成设置前置条件的习惯**。

### 小结

完成以上步骤后，你已经学会了：
- 如何在对象编辑器中创建和管理升级
- 如何设置攻击力和护甲的变化效果
- 如何让升级影响移动速度和生命值
- 如何配置多级升级的效果叠加

**立刻试试看**：打开编辑器，为一个兵种创建一个"攻击强化"升级，设置为3级，每级增加5点攻击力，然后测试效果！

## 实战示例与常见问题

本节将通过一个完整的攻防升级创建示例，带你一步步掌握升级编辑器的基本用法，并学习如何排查常见的升级不生效问题。完成学习后，你将能够独立为你的RPG地图添加属性提升功能。

### 创建攻防升级的完整示例

假设我们想创建一个"攻击强化"升级，让玩家能够消耗金币永久提升单位的攻击力。

1. **第一步：打开物体编辑器** — 在顶部菜单栏点击【场景】→【物体编辑器】[^2]，或者直接按快捷键 **F6**。物体编辑器是管理所有游戏内单位、道具、技能的强大工具。

2. **第二步：创建新升级** — 在左侧分类栏中找到【科技】选项并点击展开，你会看到许多预设的升级。右键空白处，选择【新建科技】（也可以直接复制一个现有的升级，然后修改它）。科技就是 Warcraft 3 中用于升级系统的核心概念。

3. **第三步：设置升级属性** — 选中新建的科技后，在右侧面板找到以下关键字段：
   - **名称**：输入"攻击强化 I"
   - **研发费用**：设置需要的金币和木材数量
   - **研发时间**：设置多少秒完成
   - **最大等级**：设为 3（表示可以升级3次）

4. **第四步：配置升级效果** — 找到【数据 - 字段】中的相关属性：
   - **攻击力提升**：设置每次升级增加多少攻击力
   - **影响单位**：选择哪个单位会获得这个提升效果

5. **第五步：放置升级建筑** — 返回地图编辑器界面，在左侧【建筑】单位分类中，找到支持该科技的建筑（如"神秘圣地"），或者创建一个新的升级建筑并放置到地图中。

> **💡 新手提示**：如果找不到升级效果字段，记得在物体编辑器菜单中点击【视图】→【显示高级属性】，这样会显示更多详细选项。

> **⚠️ 常见错误**：很多新手只创建了科技，却没有把它和某个建筑关联起来！一定要在建筑的"科技树"或"可研发升级"字段中添加这个科技，否则玩家根本看不到这个升级选项。

### 升级不生效的排查方法

当你发现升级没有效果时，按以下顺序逐项检查：

1. **检查科技是否被正确关联** — 打开建筑单位的属性，找到【科技 - 可研发升级】字段，确认你的升级ID在列表中。

2. **检查触发器（触发器是一种自动化脚本）** — 如果升级需要触发器激活，确认触发器的条件是否满足。在触发器编辑器中按 **F6**，检查相关触发是否启用。

3. **检查单位类型** — 确认你升级的单位是否在科技的"影响单位"范围内。比如你想升级"步兵"，但科技只关联了"骑士"，那就不会生效。

4. **测试地图** — 保存地图后点击【测试】（快捷键 **F10**），进入游戏实际测试升级功能。有时编辑器预览并不准确。

> **⚠️ 常见错误**：新手经常忽略"影响单位"字段！即使科技关联了建筑，如果这个字段为空或者没有包含你的目标单位，升级也不会有任何效果。

### 新手常见错误与避免方法

| 常见错误 | 避免方法 |
|---------|---------|
| 升级在编辑器里看不到 | 确认科技已添加到建筑的"可研发升级"列表中 |
| 升级后属性没有提升 | 检查"影响单位"字段是否包含目标单位 |
| 升级按钮位置混乱 | 在建筑的"按钮位置"字段中设置正确的X/Y坐标 |
| 忘记设置研发费用 | 每个科技至少设置一项资源（金币或木材）作为费用 |
| 多人游戏中只有主机生效 | 使用"玩家 - 动作"设置升级时针对所有玩家执行 |

> **💡 新手提示**：养成每次修改后都用 **F10** 实际测试的好习惯！很多问题只有进入游戏才能发现，编辑器预览并不总是准确的。建议创建一个简单的测试地图专门用来验证各种功能。
