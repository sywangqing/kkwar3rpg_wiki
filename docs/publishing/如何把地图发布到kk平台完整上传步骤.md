# 如何把地图发布到KK平台：完整上传步骤

## 概述

本节将带你了解**发布地图到KK平台**的完整流程。学完本节后，你将知道如何从打开编辑器开始，一步步把做好的地图上传到KK对战平台，让其他玩家可以玩到你的作品！

> **💡 新手提示**：在开始上传之前，请确保你已经**注册了KK开发者账号**，并且**地图已经保存完毕**。如果还没准备好，建议先完成地图制作再来上传。

### 操作步骤

1. **找到并打开Y3编辑器** — 进入你的KK对战平台安装文件夹，找到 `kkduizhan/Games/y3/game` 目录，双击 `Editor.exe` 打开编辑器[^1]

2. **登录开发者平台账号** — 打开编辑器后，点击登录按钮，输入你的开发者账号信息[^1]

3. **核对编辑器版本号** — 登录后，注意检查编辑器左上角显示的版本号，确保与你制作地图时使用的版本**完全一致**[^1]

4. **打开你的地图文件** — 点击“文件”→“打开”，找到你地图包体内的 `header.map` 文件并打开[^1]

5. **等待地图加载完成** — 打开地图后耐心等待加载，直到编辑器完全进入可操作状态

6. **进入发布流程** — 点击编辑器**左上角的“发布”**按钮，再点击“上传项目”[^1]

### 小结

完成以上步骤后，你应该能看到一个**发布信息填写界面**，接下来就可以输入作品名称、版本号、更新日志等内容了。恭喜你，已经成功迈出了发布地图的第一步！
