# 🛤️ 学习路径

不同经验水平的开发者，可以参考以下路径循序渐进地学习 War3 地图编辑器。

适合从未接触过 War3 地图编辑器的同学，目标是独立制作一张简单的 RPG 地图。

1. **[编辑器安装与配置](/getting-started/editor-installation)** — 安装 YDWE，配置开发环境
2. **[编辑器界面功能全览](/getting-started/editor-ui-overview)** — 了解各功能区域
3. **[地图属性与基础设置](/getting-started/map-settings)** — 创建你的第一张地图
4. **[地形编辑器基础操作](/terrain/terrain-basics)** — 绘制地形、放置装饰物
5. **[触发器系统概述](/triggers/trigger-intro)** — 了解事件-条件-动作模型
6. **[触发器事件类型详解](/triggers/trigger-events)** — 学会响应游戏事件
7. **[触发器动作大全](/triggers/trigger-actions)** — 控制游戏行为
8. **[单位编辑器详解](/object-editor/unit-editor)** — 创建自定义英雄和单位
9. **[创建第一张 RPG 地图](/getting-started/first-rpg-map)** — 综合实践
10. **[地图测试与调试方法](/getting-started/test-and-debug)** — 发布前验证
