# 灯光与氛围：让地图有场景感

## 为什么灯光和氛围很重要

在本节中，你将了解到灯光和氛围对地图的重要性，以及它们如何影响玩家的游戏体验。学完本节后，你会在设置地图灯光时有明确的目标，而不是凭感觉乱调。

### 灯光如何影响玩家体验

想象一下你走进一个漆黑的房间，和走进一个阳光明媚的房间，感觉完全不同对吧？灯光就是这样一种"无声的语言"，它会在第一时间告诉玩家：这个地图是危险的还是安全的，是白天的还是夜晚的[^1]。

当你设置好地图的灯光后，玩家会不自觉地被明亮的地方吸引，同时对阴暗的角落产生好奇或紧张感。这种"引导"不需要任何文字说明，玩家自己就会跟着感觉走[^1]。

好的灯光设计还能让你的地图看起来更有"层次感"。比如在一片森林里，阳光透过树叶洒下来，形成明暗交错的效果——这种细节虽然小，但会让玩家觉得这个地图很精致、很用心。

### 氛围感对地图品质的影响

氛围指的是玩家进入地图后感受到的"整体感觉"——是紧张的、轻松的、神秘的，还是欢快的？灯光是营造这种氛围最直接的工具之一。

一个优秀的地图，灯光和场景会完美融合。比如：
- 阴森的山洞 → 使用暗淡的蓝紫色灯光
- 热闹的城镇广场 → 使用明亮的暖黄色灯光
- 神秘的魔法塔 → 使用忽明忽暗的闪烁灯光

如果灯光和场景不匹配，玩家会觉得"哪里怪怪的"，但又说不出原因。这就是新手地图和专业地图的区别之一——专业地图会用灯光"讲故事"，让玩家沉浸其中。

> **💡 新手提示**：别小看灯光的作用！即使你的地图功能很简单，好的灯光也能让它看起来更专业。玩家第一眼看到的往往是视觉效果，所以这是提升地图品质最"划算"的一步。

> **⚠️ 常见错误**：新手容易犯的一个错误是整张地图都用同一个亮度。正确的做法是根据不同区域设置不同的灯光——比如室内暗一些、室外亮一些，这样才符合真实世界的逻辑。

### 小结

学完本节后，你应该明白：
- 灯光能在第一时间给玩家"感觉上的引导"
- 灯光和场景匹配能大幅提升地图的专业度
- 氛围是通过灯光、颜色、亮度等细节综合营造的

下一节我们将学习如何在地图编辑器中实际调整灯光设置！

## World Editor 灯光基础设置

本节将带你学习如何在World Editor中调整地图的**天色**、**灯光方向**和**阴影**设置。这些设置决定了玩家进入你的地图时，第一眼看到的"氛围感"——是明媚的白天、神秘的黄昏，还是阴森的夜晚？学完本节，你将能够为自己的地图设定独特的光影风格。

### 操作步骤

1. **打开场景设置面板** — 在World Editor菜单栏中，点击 **"场景"（Scenario）**，然后选择 **"场景设置"（Scenario Settings）**[^1]。这一步是所有灯光调整的入口，就像打开控制台的"总开关"。

2. **设置天色（Ambient Color）** — 在场景设置窗口中，找到"环境光颜色"（Ambient Color）选项。点击颜色方块，弹出颜色选择器。这里的颜色会渗透到整个场景，模拟天空反射到物体上的光。**白天用淡蓝色**，**夜晚用深紫色**，**黄昏用橙黄色**，分别营造不同的情绪氛围。

3. **调整灯光方向与强度** — 在同一面板中，找到"阴影开关"和"环境光强度"滑块。灯光方向由"太阳角度"控制，强度则决定整体画面的明暗程度。**强度值越高，画面越亮**；调低则会呈现更戏剧化的明暗对比。

4. **配置阴影显示** — 点击 **"高级"（Advanced）** 标签页，可以设置是否显示动态阴影。初学者建议**先关闭阴影**以提高运行流畅度，等地图优化完成后再开启细节阴影。

### 小结

完成以上四步后，你的地图将拥有基本的氛围感。你可以在编辑器中按 **F9** 预览地图，观察天色和光影是否符合预期。如果想要更高级的昼夜变化效果，建议进一步学习"Day and Night模型"的用法[^1]。

> **💡 新手提示**：别忘了不同地形的"高度"也会影响阴影效果——高的建筑或树木会在低地上投下阴影，这能为你的场景增添层次感。

> **⚠️ 常见错误**：很多新手把环境光调得太高，导致整个地图看起来"灰蒙蒙的"，失去了对比度。建议初始值设在 **0.5-0.7** 之间，然后再根据实际效果微调。

## 不同场景的灯光方案

在本节中，你将学习如何为地图中的不同场景设置合适的灯光效果。通过掌握白天、黄昏、夜间和地下城洞穴的灯光配置技巧，你能让地图的每个区域都有独特的视觉氛围，提升玩家的沉浸感。

### 操作步骤

1. **第一步：打开环境设置面板** — 在WE菜单栏点击"场景" → "环境"（或使用快捷键F4）打开环境设置窗口[^1]

2. **第二步：了解DNC系统** — DNC（Day and Night Cycle）是魔兽争霸的昼夜循环系统，它控制着场景中灯光随时间变化的规则[^1]

3. **第三步：设置白天场景** — 在环境设置中选择"白天"预设，调节"环境光"强度为80-100%，让整个场景明亮清晰

4. **第四步：设置黄昏场景** — 切换到黄昏预设，将灯光颜色调整为橙红色（RGB值约255,100,50），环境光强度降至50-60%，模拟夕阳效果

5. **第五步：设置夜间场景** — 选择"夜间"预设，环境光强度调至20-30%，并增加"Omnilight"泛光灯来模拟月光[^1]

6. **第六步：创建地下城/洞穴氛围** — 关闭DNC系统，手动放置点光源（Point Light），将环境光降至10%以下，营造封闭黑暗的感觉

> **💡 新手提示**：地下城场景中，不同区域的灯光颜色可以略有差异，比如矿洞用蓝绿色、熔岩区域用红橙色，能增加场景的真实感。

> **⚠️ 常见错误**：新手容易把所有场景的灯光都调成最亮，这样反而让地图失去层次感。记住：对比度越强，场景感越明显。

### 小结

完成以上步骤后，你的地图应该具备清晰的昼夜氛围区分。白天场景明亮开阔，黄昏场景温暖柔和，夜间场景幽暗神秘，地下城场景则充满压迫感。建议在地图的不同区域多测试几组灯光参数，找到最合适的搭配。

## 环境音效与氛围营造

在地图中添加合适的背景音乐和环境音效，能让玩家更有代入感。学会这节内容后，你可以为不同区域设置专属音乐，并让音效配合灯光变化，营造出白天、夜晚等不同的场景氛围。

### 操作步骤

1. **第一步：创建音效文件并导入** — 在WE（World Editor地图编辑器）的资源管理器中，选择"音效"标签页，右键点击导入你的音乐或环境音效文件（支持.mp3或.wav格式）[^1]

2. **第二步：设置区域音效** — 在地形编辑器中创建一个"区域"（Region），然后打开触发编辑器，新建一个"地图初始化"事件，添加"播放音效"动作，选择你导入的音效文件

3. **第三步：配置区域触发器** — 新建一个触发器，设置为"单位进入区域"事件，配置当玩家单位进入该区域时播放对应音效，离开时停止或切换音乐

4. **第四步：音效与Day/Night灯光同步** — 在触发器中使用"时间条件"判断当前是白天还是夜晚，白天播放轻快的音乐，夜晚播放紧张或神秘的音乐[^1]

> **💡 新手提示**：每个区域只设置一个主背景音乐即可，音效文件不要太大（建议单文件不超过5MB），否则会影响地图加载速度。

> **⚠️ 常见错误**：新手经常忘记在触发器中设置"停止音效"动作，导致玩家离开区域后音乐还在继续播放，解决方案是在"单位离开区域"事件中添加"停止音效"动作。

### 小结

完成以上步骤后，你的地图将拥有动态的环境音效系统，玩家进入不同区域时会听到专属音乐，昼夜切换时音效也会随之变化，场景感将大幅提升！

## 地形设计与氛围的关系

在地形编辑器中，你放置的每一样东西都会影响玩家感受到的情绪和氛围。森林、树木、花草属于**自然元素**，能让地图显得生机勃勃；城墙、房屋、路灯属于**人工元素**，则给人城市或废墟的感觉。学会把这两类元素搭配好，你的地图就会有"场景感"，玩家一看就知道这是森林、城镇还是废墟。[^1]

### 植被与自然元素

在编辑器左侧工具栏找到"地形"选项，里面有各种植被贴图和模型。

1. **第一步：选择地形刷** — 点击工具栏中的"地形刷"图标[^1]，在下方面板中选择"草地"类贴图
2. **第二步：绘制草地** — 在地图空白区域按住鼠标左键拖动，草地会自动铺上去
3. **第三步：添加树木** — 切换到"装饰物"选项卡，找到"Tree"开头的模型[^1]，在草地区域点击放置
4. **第四步：点缀细节** — 可以再放一些蘑菇、石头、灌木等，让自然场景更丰富

> **💡 新手提示**：树木不要放得太整齐，自然界的树是随机分布的。尝试用"随机旋转"功能（选中树木后按R键），能让场景更真实。

> **⚠️ 常见错误**：新手容易把植被贴图铺满整张地图，这样反而显得单调。正确做法是分区——有的地方草地多，有的地方石头多，形成对比。

### 建筑与人工元素

人工元素能快速建立"这是什么地方"的印象。

1. **第一步：打开建筑库** — 在"装饰物"选项卡中，选择带有"Bldg"或"City"标签的模型
2. **第二步：放置主要建筑** — 先放1-2个大房子或城墙作为"主角"[^1]
3. **第三步：添加辅助道具** — 路灯、箱子、栅栏等小物件能让场景更有生活气息
4. **第四步：调整高度** — 有时建筑会"浮"在地上，选中后用Shift+上下方向键微调位置

### 小结

完成以上步骤后，你应该能区分自然元素和人工元素的用法，并尝试在一张地图上组合它们。建议先做一个小区域做实验，比如"村口"或"森林入口"，反复调整直到感觉对为止。多观察真实照片或游戏截图，模仿其中的布局，慢慢你就会形成自己的设计思路。

## 新手常见错误与改进建议

本节将介绍新手在使用灯光时最常犯的错误，以及具体的改进方法。学完本节后，你将能够避免常见的灯光陷阱，让地图拥有更专业的视觉效果。[^1]

### 避免灯光过于均匀

很多新手在设置灯光时，为了让地图"每个角落都能看清"，会把所有灯光的**强度**都设为相同的数值，**角度**也均匀分布。他们认为这样最"安全"。

> **⚠️ 常见错误**：把所有方向光、点光源的强度都设为100，角度也均匀分布。这样做会让地图看起来很"平"，没有层次感，就像一张没有阴影的平面画，玩家会感觉场景缺乏真实感。

**改进步骤**：

1. **设置主光源** — 选择一个方向作为"太阳光"或"月光"，将强度设得较高（如80-100），这是地图中最亮的光源[^1]
2. **添加补光** — 在主光源照射不到的阴影区域，添加强度较低的补光（30-50即可），让暗处不完全漆黑
3. **利用明暗对比** — 把某些区域故意调暗，另一些区域保持明亮，这样能创造出视觉焦点

> **💡 新手提示**：试着把地图某个区域的灯光关掉，只保留另一区域的光源。你会发现对比度高的地图反而更有"代入感"，就像现实中的手电筒效果。

### 氛围一致性的重要性

灯光不仅仅是照明工具，更是**传递情绪**的重要手段。一个阴森的地牢如果用了暖黄色灯光，玩家会感到"违和"；一个温馨的村庄如果用了冷蓝色灯光，玩家会感到"奇怪"。

> **⚠️ 常见错误**：一个地图里混用太多不同颜色的灯光，导致氛围混乱。比如森林场景同时出现橙色、紫色、青色的灯光。

**改进方法**：

1. **先确定场景情绪** — 在设置灯光前，先问自己：这个场景想表达什么？（恐怖、温馨、神秘？）
2. **统一色调** — 整个场景使用相近色系的灯光（暖色系或冷色系），不同场景之间再做明显区分
3. **参考真实世界** — 回忆现实中的类似场景：夜晚的森林是深蓝偏黑的，日落的沙漠是橙红色的

> **💡 新手提示**：可以截取真实照片的截图作为参考，用取色工具吸取照片中的主要颜色，再应用到你的灯光设置中。

### 小结

完成以上步骤后，你的地图灯光应该已经告别了"一片灰白"的问题，拥有了清晰的主光源和合理的明暗对比。记住：**好的灯光不是让玩家看清一切，而是引导玩家注意你想让他们看的地方。**
