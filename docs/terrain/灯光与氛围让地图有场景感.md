# 灯光与氛围：让地图有场景感

## 为什么灯光和氛围很重要？

在学习完本节后，你将理解**灯光和氛围**如何把一张普通地图变成一个有"代入感"的冒险世界。你会知道为什么玩家一进入你的地图就能感受到紧张、神秘或温暖，以及如何在World Editor中设置基本灯光[^1]。

### 灯光对玩家体验的影响

你有没有注意到，当你走进一片阴暗的森林时，会不由自主地感到紧张？灯光的作用正是如此——它能**操控玩家的情绪**。想象一下，同一个山洞场景，用明亮的金色阳光照亮，玩家会觉得安全；换成深蓝色的幽暗灯光，立刻就变成了危险的地下迷宫[^2]。

灯光还帮助玩家**理解时间和环境**。比如：
- 暖黄色的午后阳光 → 告诉玩家这是白天、温暖的区域
- 冷蓝色的月光 → 表示夜晚或室内
- 暗红色的雾气 → 暗示危险或魔法区域

### 新手常忽视的视觉细节

很多新手花费大量时间调整地形、种植树木，却完全忽略了灯光设置。结果地图看起来"平"，缺乏层次感和代入感[^1]。

### 操作步骤

1. **第一步：打开灯光设置** — 在World Editor顶部菜单中点击"场景" → "环境设置"（或直接使用快捷键 Shift + F7）[^2]
2. **第二步：选择基础光照** — 在弹出的"环境"面板中，找到"环境光"选项，点击下拉菜单选择预设的灯光方案，如"黎明"、"正午"、"黄昏"等
3. **第三步：调整氛围颜色** — 拖动"Ambient"（环境光）滑块，或点击颜色框自定义环境光的色调
4. **第四步：预览效果** — 点击"应用"后，进入地图测试（F10），观察不同时间下的氛围变化

> **💡 新手提示**：不要只设置一种灯光！如果你的地图包含多个区域（森林、平原、地下城），每个区域可以用不同的"区域灯光"来营造各自独特的氛围。

> **⚠️ 常见错误**：新手常把环境光调到最亮，认为"看得清楚就是好"。但这样会让地图失去层次，像一张没有阴影的纸。正确的做法是保留适度的明暗对比，让重要区域自然"突出"出来。

### 小结

完成以上步骤后，你的地图将拥有基本的环境光和氛围设置。玩家进入你的地图时，会根据灯光感受到"这是一个温暖的村庄"或"这是一片阴暗的沼泽"，从而更投入地体验你的RPG故事[^1][^2]。

## 认识World Editor灯光系统

在本节中，你将学习如何在World Editor中找到并调整灯光设置，了解影响地图氛围的基本灯光属性。完成本节学习后，你将能够为不同场景（森林、沙漠、夜晚等）设置合适的灯光效果，让玩家一眼就能感受到地图想表达的情绪。

### 操作步骤

1. **打开场景设置面板** — 在World Editor顶部菜单栏中，点击 **"场景"** 菜单，然后选择 **"场景设置"** 选项[^1]。这是一个经常被新手忽略的入口，但它控制着地图的整体环境参数。

2. **定位到灯光选项卡** — 在弹出的"场景设置"窗口中，你会看到多个选项卡（地形、内存资源、灯光等）。点击 **"灯光"** 选项卡[^2]。如果你的界面是英文版本，这个选项卡通常叫做"Lighting"。

3. **认识三种基本灯光属性** — 在灯光面板中，你会看到以下核心参数：
   - **环境光**：指照亮整个地图的基础光线强度，影响阴影的暗度[^1]
   - **太阳光/方向光**：模拟太阳的光线方向，决定高光和阴影的位置
   - **雾效**：添加距离感的重要参数，让远处的物体逐渐融入背景色

4. **切换预设灯光方案** — 面板下方有一个下拉菜单，里面列出了Warcraft III内置的多种预设灯光（如"默认"、"荒漠"、"夜晚"、"地下城"等）。选择一个预设后，点击 **"应用"** 按钮[^3]。

> **💡 新手提示**：每次调整灯光后，记得点击"应用"按钮，否则设置不会保存。你可以反复尝试不同的预设组合，找到最适合你地图风格的灯光。

> **⚠️ 常见错误**：新手常误以为调整一次灯光就能完美呈现氛围，实际上灯光需要与地形材质、背景颜色配合使用。建议先确定地图的整体色调（暖色/冷色），再相应调整灯光。

### 小结

完成以上步骤后，你应该已经找到了灯光设置面板，了解了环境光、太阳光和雾效的基本作用，并能应用预设灯光方案。接下来，你可以尝试同时调整多个参数，创建完全属于你的自定义灯光组合！

## 颜色理论：如何用灯光塑造场景情绪

在地图编辑器中，灯光不仅仅是"照亮地图"这么简单！不同颜色的灯光可以传达完全不同的情感——比如温暖的金色灯光让人感到安全，而冰冷的蓝色灯光会让人紧张。本节你将学会根据场景需求选择合适的灯光颜色，让玩家在进入地图的第一秒就能感受到你想要的氛围。

### 暖色调灯光的适用场景

暖色调灯光通常指偏黄、橙、红等颜色，这类灯光给人**温暖、亲切、活力**的感觉。

**什么时候用暖色调？**

- 🏠 **主城/安全区**：玩家出生点、商店聚集区 — 暖光让人感觉这里是"家"，可以安心交易和休息
- 🔥 **火山/熔岩地区**：配合岩浆地形，橙红色灯光能增强"炙热"的视觉冲击
- 🌅 **白天场景**：模拟清晨或黄昏的阳光，金黄色调营造舒适的户外感
- 🏕️ **营地篝火**：玩家休息点周围用橙色光晕，传递"这里很安全"的信息

### 冷色调灯光的适用场景

冷色调灯光通常指偏蓝、青、紫等颜色，这类灯光给人**神秘、寒冷、紧张**的感觉。

**什么时候用冷色调？**

- 🏔️ **雪山/冰原**：蓝色或白色灯光模拟冰雪反射的冷光
- 👻 **鬼屋/地下墓穴**：紫色或深蓝色灯光营造诡异、恐怖的氛围
- 🌙 **夜间地图**：深蓝色为主，配合少量星光点光源
- 💀 ** Boss战区域**：深红色或暗紫色暗示危险，提醒玩家"即将面临挑战"

### 灯光颜色与地形风格的搭配

灯光和地形必须**协调一致**，否则会让玩家觉得"哪里不对劲"。记住这个原则：**灯光颜色应该"强化"地形的自然属性**，而不是"对抗"它。

> **💡 新手提示**：搭配口诀 —— "火地用暖光，雪地用冷光；主城用暖有安全感，副本用冷有压迫感"

**常见搭配误区：**

| 错误做法 | 问题 | 正确做法 |
|---------|------|---------|
| 在雪地用地暖橙光 | 违反常识，像"化工厂" | 用淡蓝色或白色 |
| 在熔岩地用蓝光 | 视觉冲突，没有逻辑感 | 用橙红渐变 |
| 主城用深紫色光 | 让人不敢靠近 | 用暖黄或米白色 |

### 操作步骤

1. **打开场景控制面板** — 在世界编辑器顶部菜单栏找到"场景"选项，点击"场景控制"[^1]
2. **进入灯光设置界面** — 在场景控制面板中选择"灯光"标签页，你会看到颜色选择器[^2]
3. **调整灯光颜色** — 点击颜色方块，会弹出颜色选择器；向左拖动滑块偏向冷色，向右偏向暖色
4. **设置灯光强度** — 调整"亮度"数值（0-100），数值越高灯光越亮；建议安全区用70-80，危险区用40-60
5. **添加环境光** — 勾选"使用环境光"选项，这样即使玩家单位不在光源附近，也有基础照明

> **⚠️ 常见错误**：新手往往只设置一盏主灯就完事了！正确做法是添加**多盏辅助灯**，比如在地图四角各放一盏微型补光灯，这样可以避免"中间亮、边缘黑"的尴尬效果。

### 小结

完成以上设置后，你的地图应该能够通过灯光颜色**自动传达场景情绪**：玩家走进主城会感到温暖安心，进入冰洞会下意识紧张起来。灯光是地图的"隐形叙事者"，用好它，你的地图专业度瞬间提升一个档次！

## 迷雾效果：营造空间深度和神秘感

通过设置迷雾效果，你可以让地图看起来更有层次感，营造出神秘的探索氛围。这一步不需要写任何代码，只需要调整几个参数就能实现。

### 操作步骤

1. **第一步：打开地图设置面板** — 在菜单栏中点击“场景”（Scenario）菜单，然后选择“地图设置”（Map Options）[^1]
2. **第二步：定位迷雾选项** — 在弹出的设置窗口中，找到“迷雾”（Fog of War）选项卡[^2]
3. **第三步：启用迷雾效果** — 勾选“启用迷雾”复选框，这样迷雾功能才会生效[^2]
4. **第四步：设置迷雾密度** — 拖动“迷雾密度”滑块调整浓度，数值越高迷雾越浓[^2]
5. **第五步：选择迷雾颜色** — 点击颜色选择器，根据你的地图主题选择合适的迷雾色调[^2]
6. **第六步：测试效果** — 点击“确定”保存后，按F9进入测试模式，查看迷雾效果是否符合预期

> **💡 新手提示**：迷雾密度建议设置在0.3到0.5之间，这样既能保持足够的层次感，又不会影响玩家观察已探索的区域。如果设置过厚，玩家会感觉视野被严重遮挡。

> **⚠️ 常见错误**：很多新手只调整迷雾密度，却忽略了颜色设置。默认的白色迷雾不一定适合你的地图场景——沼泽地图适合深绿色迷雾，沙漠地图适合土黄色迷雾，合理的颜色搭配能让地图更有代入感。

### 小结

完成以上步骤后，你的地图已经成功添加了迷雾效果。玩家在探索地图时，未探索的区域会被迷雾遮挡，只有单位到达过的地方才会逐渐显示。这样不仅增加了神秘感，还让玩家有探索的动力。如果想进一步增强氛围，可以结合之前学习的灯光设置，让迷雾在灯光照射下呈现不同的颜色层次。

## 天气与时段：动态氛围营造

本节将教你为地图添加动态天气效果（雨、雪、雾等）以及设置白天、黄昏、黑夜的时段切换。学完本节后，你的地图将不再是“死板”的静态画面，而是能根据场景变化而呼吸的动态世界。

### 添加雨、雪、雾等天气效果

天气效果是魔兽争霸3地图编辑器自带的功能模块，无需编写代码即可使用。以下是具体步骤：

1. **第一步：打开氛围面板** — 在编辑器顶部菜单栏中，点击“场景”菜单，然后选择“氛围”选项[^1]。这会打开一个控制全局视觉效果的面板。

2. **第二步：选择天气类型** — 在氛围面板中找到“天气”选项卡。这里列出了多种预设天气效果，包括“雨”、“雪”、“雾”、“暴风雪”等。点击你想要的效果即可选中[^2]。

3. **第三步：调整天气强度和范围** — 选中天气后，可以在右侧面板中调整“密度”（控制天气效果的浓密程度）和“持续时间”。建议新手先保持默认设置，熟悉后再自行调整。

4. **第四步：预览效果** — 按F9或使用“测试地图”功能，在游戏中预览天气效果是否符合预期。

> **💡 新手提示**：天气效果默认会覆盖整张地图。如果你想让特定区域（如山洞内）不受天气影响，需要在该区域的边缘添加“遮蔽物”（使用地形工具中的“装饰物”类别），这样雨水或雪花就会被阻挡。

> **⚠️ 常见错误**：新手经常一次性叠加多种天气效果（如雨+雾+暴风雪），导致画面过于杂乱、看不清游戏内容。建议一张地图最多同时使用1-2种天气效果，保持视觉清晰。

### 时段系统：白天、黄昏、黑夜的切换

时段系统决定了地图的光照颜色、阴影方向和整体色调。通过调整时段，你可以营造出不同的情绪和故事氛围。

1. **第一步：进入时段设置** — 在“场景”菜单的“氛围”面板中，点击“环境”选项卡。这里你可以看到“当前时段”和“时段列表”两个选项[^3]。

2. **第二步：选择预设时段** — 编辑器提供了四种基础时段预设：白天（明亮暖色调）、黄昏（橙红渐变）、夜晚（深蓝冷色调）以及黎明。新手可以直接选择这些预设时段[^4]。

3. **第三步：设置时段持续时间** — 如果你想让时段自动循环变化，可以设置每个时段的“持续时间”（单位为游戏秒）。例如，将白天设为120秒、黄昏设为60秒、夜晚设为180秒。

4. **第四步：启用时段循环** — 在时段列表下方，勾选“循环”选项。这样地图中的时间会按照你的设置不断重复循环。

> **💡 新手提示**：时段切换时会伴随光照颜色渐变，这个渐变过程本身就是一种氛围效果。比如在恐怖主题地图中，可以设置超长的黑夜时段，增强压迫感。

> **⚠️ 常见错误**：忘记关闭时段循环。有些新手做完测试后发现时间一直变化，无法稳定在特定时段检查地形。解决方法是在不需要循环时取消勾选“循环”选项。

### 小结

完成以上步骤后，你应该已经掌握了：
- 如何在地图中添加雨、雪、雾等天气效果
- 如何调整天气强度以适应不同的场景需求
- 如何设置白天、黄昏、黑夜的时段切换
- 如何让时段系统自动循环运行

现在你可以打开编辑器，尝试为你的地图添加第一套天气和时段设置吧！建议先从“白天+雨天”或“夜晚+浓雾”这样简单的组合开始练习。

## 实战技巧：优秀灯光预设推荐与调整

本节将为你推荐几个经过验证的优秀灯光预设，并手把手教你如何根据自己的地图风格微调这些预设。学完以后，你将能够为不同类型的地图场景（森林、洞穴、城镇等）选择最合适的灯光方案。

### 常用灯光预设介绍及使用场景

1. **第一步：打开灯光编辑器** — 在顶部菜单栏点击"场景"→"环境设置"→"灯光"[^1]
2. **第二步：浏览预设列表** — 在左侧面板你会看到多个预设，常见的有"白天"、"黄昏"、"夜晚"、"地牢"等[^2]
3. **第三步：选择适合场景的预设** — 森林地图推荐使用"白天"或"黄昏"，洞穴/地牢使用"夜晚"或"地牢"，城镇可以使用"黄昏"营造温暖氛围
4. **第四步：应用预设到地图** — 双击你选择的预设，或点击"应用"按钮[^1]

> **💡 新手提示**：不同预设对整体氛围影响巨大！先花5分钟尝试每个预设的视觉效果，比你凭想象猜测要有效得多。

### 如何微调预设达到理想效果

1. **第一步：选中已应用的预设** — 在灯光面板中点击预设名称[^2]
2. **第二步：调整环境光颜色** — 向右拖动"环境光"滑块可以提亮阴影区域，向左则加深
3. **第三步：调整直射光方向** — 拖动"太阳高度角"改变光源位置，模拟早晨、中午、傍晚的不同光线
4. **第四步：预览效果** — 切换到"测试地图"模式（按F9测试），在不同时间段走动感受真实氛围[^4]

> **⚠️ 常见错误**：新手经常只调一个参数就以为完成了。正确做法是同时微调"环境光"、"直射光"、"雾效"三个参数，让它们互相配合，否则画面会显得不自然。

### 新手常见灯光错误与修正方法

**错误一：灯光太亮或太暗**
修正：检查"整体亮度"设置，白天场景建议在70%-80%，夜晚场景建议在30%-40%[^1]

**错误二：阴影区域全黑**
修正：将"环境光"值调高到40%以上，让阴影中也能看清物体轮廓[^2]

**错误三：雾效颜色与灯光不匹配**
修正：雾效颜色应与灯光色调协调——夜晚灯光偏蓝时，雾效也应为淡蓝色[^3]

### 小结

完成以上步骤后，你应该能够为你的地图选择合适的灯光预设，并通过微调环境光、直射光和雾效参数，让地图呈现出独特的场景氛围。建议先从预设入手，逐步增加个人调整，最终形成自己独特的灯光风格！
