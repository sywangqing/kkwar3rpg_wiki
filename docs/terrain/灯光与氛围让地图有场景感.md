# 灯光与氛围：让地图有场景感

## 概述

在Warcraft III地图编辑器中，**灯光**是塑造地图"氛围感"的核心工具。简单来说，灯光就是决定你的地图"看起来是什么感觉"的重要因素——是阳光明媚还是阴雨绵绵，是温暖舒适还是阴森恐怖，这些都离不开灯光的设置。

也许你会想："我已经把地形铺好了，把树木房屋放好了，为什么还要管灯光？"这是很多新手会问的问题。答案是：**灯光是让静止的场景"活"起来的关键**。想象一下，同样是一片森林，在不同的光线下可以显得神秘、浪漫、恐怖或欢乐。灯光就像画家调色板上的颜料，让你的地图有了灵魂。

对于新手来说，最容易理解的概念是"**全局光照**"——它决定了整张地图的整体色调。打个比方，就像给照片加滤镜一样，全局光照会让你的地图统一偏暖（橙黄色）或者偏冷（蓝紫色）。这就是为什么有些地图看起来"暖暖的"，有些看起来"冷冷的"。

> **💡 新手提示**：不要小看灯光的作用！即使你的地形和模型很简单，调整好灯光也能让地图看起来专业很多。很多新手地图看起来"很平"、"没感觉"，往往就是灯光没设置好。

> **⚠️ 常见错误**：新手容易犯的一个错误是让灯光"太亮"或"太平均"。以为灯光越亮越好，或者所有地方亮度都一样。其实真正好看的地图，灯光应该有明有暗、有主有次，这样才能产生层次感和立体感。

### 小结

通过本节的学习，你将理解什么是灯光，以及为什么它对地图的氛围至关重要。完成后，你将能够在地图编辑器中找到灯光设置的位置，并初步调整出你想要的基本色调。
