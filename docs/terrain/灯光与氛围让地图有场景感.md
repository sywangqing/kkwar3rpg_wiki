# 灯光与氛围：让地图有场景感

## 概述

灯光是让地图“有灵魂”的秘密武器。想象一下，同样是一片森林，在明亮的日光下显得清爽活泼，而在昏暗的蓝紫色月光下却充满神秘感——灯光就像一位魔术师，用光线和阴影就能改变玩家的情绪[^1]。在本节中，你将学习 Warcraft III 中调整灯光的基础操作，学会为你的地图营造不同的场景氛围，让玩家一进入地图就能感受到你想表达的情感。

### 操作步骤

1. **第一步：打开地图属性面板** — 在 World Editor（世界编辑器）顶部菜单栏点击“场景”，在下拉菜单中选择“地图属性”。这一步是所有地图全局设置的入口，灯光设置也在其中[^2]。

2. **第二步：找到灯光预设选项** — 在地图属性窗口中，找到“灯光”相关的设置区域。编辑器会提供几种内置的灯光预设（Preset），比如“白天”“黄昏”“夜晚”等，每个预设都包含了一组预先配置好的光照颜色和强度组合[^1]。

3. **第三步：尝试不同的预设** — 点击不同的预设名称，观察右侧预览窗口（或直接保存后在游戏中测试）的效果差异。暖色调的灯光通常给人温暖舒适的感觉，冷色调则容易营造神秘或紧张的氛围[^2]。作为新手，你可以先从预设中选择一个最接近你想要的风格的。

4. **第四步：微调灯光颜色（可选）** — 如果预设不够满意，点击预设旁边的“自定义”按钮，可以手动调整灯光的“环境色”和“直接光”颜色。这里不需要懂编程，只要用取色器选择你喜欢的颜色即可。

> **💡 新手提示**：别小看“预览”功能！很多新手调完灯光后直接保存进游戏才发现效果不对，其实在编辑器里多预览几次可以节省大量调试时间。

> **⚠️ 常见错误**：新手经常把灯光调得太亮或太暗，导致地图细节看不清或氛围全无。建议先从预设的80%亮度开始尝试，逐步调整到自己满意的效果。

### 小结

完成以上步骤后，你应该已经掌握了为地图选择和调整灯光的基础方法，能够：
- 打开地图属性面板找到灯光设置
- 使用预设快速改变地图的整体氛围
- 对灯光进行简单的自定义调整
