# 灯光与氛围：让地图有场景感

## 一、灯光基础概念：为什么氛围很重要

本节我们将了解什么是"场景氛围感"，以及灯光如何影响玩家的情绪。学完本节后，你就能理解为什么好的地图会让人有身临其境的感觉，并能初步认识到灯光在地图设计中的重要作用。

### 1.1 什么是场景氛围感

场景氛围感是指玩家进入一张地图后，感受到的整体"感觉"。举个例子，同样是一片森林，昏暗神秘的森林会让人紧张恐惧，阳光明媚的森林则让人心情舒畅——这两种完全不同的感受，就是氛围的体现。

在《魔兽争霸III》地图编辑器中，氛围主要通过以下几个元素来实现：

- **地形纹理**：草地、沙地、岩石等不同地面会传递不同情绪
- **迷雾效果**：迷雾可以让场景显得更神秘或有压迫感[^5]
- **高度落差**：陡峭的山崖和低洼的盆地能制造空间层次感

编辑器提供三种地形显示模式：质材模式（显示真实纹理）、线体框架模式（仅显示高度线）、隐藏模式（完全不显示地形）[^3]。其中质材模式能最直观地展现场景氛围。

### 1.3 灯光对玩家情绪的影响

灯光（或更准确地说"明暗对比"）是控制氛围的最直接工具。虽然《魔兽争霸III》的地图编辑器没有独立的灯光系统，但通过巧妙搭配地形、迷雾和单位阴影，依然可以营造出丰富多样的情绪氛围。

> **💡 新手提示**：初学者容易犯的一个错误是只关注地形好不好看，却忽略了整体氛围的协调。其实即使地形略显粗糙，只要氛围营造得当（比如阴暗的地下城配上适当的迷雾），玩家依然会沉浸其中。

**不同场景对应的情绪设计建议**：

| 场景类型 | 建议氛围 | 实现方法 |
|---------|---------|---------|
| 野外草原 | 开阔、明朗 | 使用浅绿色草地纹理，保持无迷雾 |
| 阴暗地牢 | 压抑、紧张 | 使用深色地面纹理，添加浓重迷雾 |
| 废墟遗迹 | 苍凉、神秘 | 混合多种地形纹理，使用中等迷雾 |

> **⚠️ 常见错误**：很多新手把所有区域的迷雾都调成一样，这是最常见的"氛围失误"。正确的做法是根据每个区域的剧情设定，独立调整迷雾参数。

### 小结

通过本节学习，你应该理解了三件事：

1. **场景氛围感**是玩家对地图整体的"感觉"，它比单个地形元素更重要
2. **迷雾和地形**是编辑器中营造氛围的主要工具[^5]
3. **不同场景需要不同氛围**，不能一刀切地设置所有参数

掌握了这些基础概念后，我们就可以开始学习如何在编辑器中具体调整这些设置了。

## 二、天气系统：雨雪雾营造沉浸感

天气是营造游戏场景氛围的重要工具。在魔兽争霸3地图中添加雨、雪、雾等天气效果，可以让荒凉的废墟显得更加萧瑟，让森林地图更有神秘感。本节你将学会用触发器创建天气效果，并掌握雨、雪、雾的配置与调试方法。

### 操作步骤

**1. 打开触发编辑器** — 在主界面菜单栏点击“触发编辑器”按钮[^2]，或者直接按F4键进入。触发编辑器是控制天气系统的核心工具。

**2. 创建天气触发的变量** — 右键点击左侧的“变量”列表，选择“新建变量”[^4]。将变量命名为"WeatherType"，类型选择“整数”。这个变量用来存储你想要的天气类型编号。

**3. 添加天气动作** — 在触发器中添加“环境 - Set Weather”动作[^3]。这个动作接受三个参数：整数（天气类型）、实数（强度0-1）、布尔值（是否显示特效）。

**4. 设置天气类型编号** — 魔兽争霸3中，1=雨、2=雪、3=雾。在动作参数中填入对应的数字即可[^3]。

**5. 调整天气强度** — 第二个参数控制天气明显程度，0表示无效果，1表示最大强度。建议根据场景需求设置在0.3-0.7之间[^5]。

**6. 测试天气效果** — 按F9保存并运行地图，观察天气是否正常显示。如需修改，随时回到触发编辑器调整参数。

> **💡 新手提示**：天气效果对电脑性能有一定影响。如果地图中出现卡顿，可以适当降低天气强度或减少同时出现的天气单位数量。

> **⚠️ 常见错误**：忘记在新建触发器时选择正确的事件条件，导致天气效果无法触发。确保事件设置为“地图初始化”或你需要的游戏事件类型。

### 小结

完成以上步骤后，你已经掌握了使用触发器创建天气效果的基本方法。现在可以为你的地图添加不同的天气状态，通过调整强度参数找到最适合你场景的氛围效果。

## 三、地形雾气设置：创造纵深与神秘感

本节我们将学习如何为地图添加雾气效果。掌握雾气设置后，你能让平坦的地图看起来有深度，让阴暗的森林显得更神秘，玩家的探索欲望也会更强！

### 操作步骤

1. **打开地形设置面板** — 在世界编辑器顶部菜单栏中，点击"地形"（Terrain）菜单，然后选择"地形属性"（Terrain Attributes）[^3]。这一步是所有地形美化的起点，很多新手会跳过这个菜单直接去找其他设置。

2. **找到雾效开关** — 在地形属性面板中，向下滚动找到"启用雾效"（Enable Fog）复选框，勾选它[^5]。如果这一步没做，后面调什么都看不到效果，这是最常见的新手错误！

3. **选择雾效类型** — 雾效分为两种：
   - **战争迷雾**（War3 Fog）：模拟传统RTS游戏的视野限制效果，远处会被半透明的灰色覆盖，常用于RPG和对抗地图
   - **环境雾**（Environment Fog）：更自然的渐变雾气效果，可以调节颜色，常见于营造氛围的场景地图[^5]
   点击下拉菜单选择你需要的类型。

4. **调节雾的颜色** — 点击"雾颜色"（Fog Color）旁边的色块，会弹出颜色选择器[^5]。建议根据地图主题选择：
   - 森林地图 → 绿色或深灰色
   - 沙漠地图 → 黄色或米色
   - 地下城 → 蓝紫色
   - 雪地 → 白色或浅蓝色

5. **调节雾的密度** — 找到"雾密度"或"雾起始/结束距离"滑块[^5]。数值越大，雾气越浓，地图看起来越"深"。建议设置在30-60之间，太高会看不清单位，太低又缺乏氛围感。

6. **预览效果** — 按F9或点击"测试地图"按钮，进入游戏测试你的雾气效果。编辑器预览和实际游戏效果可能有差异，一定要实际测试！

> **💡 新手提示**：雾气设置不是"一劳永逸"的！建议先设置一个基础值，在地图的不同区域使用**区域（Region）**功能配合触发器，可以实现"走到某处雾气变浓"的动态效果，这是高级地图常用的技巧。

> **⚠️ 常见错误**：很多新手把雾密度调得过高（比如80以上），结果自己都看不清单位在哪里了。解决方法是从低数值开始慢慢调，找到"看得清但有氛围"的平衡点。

### 小结

完成以上步骤后，你的地图应该已经拥有了基础的雾气效果。玩家进入游戏后，会感觉地图有纵深感，远处的内容若隐若现，神秘感油然而生。建议保存地图后实际测试几局，观察雾气是否影响了正常游戏体验，再做微调。

## 四、日夜循环与光照强度

想象一下，你的地图不再是一成不变的白天，而是会随着游戏进程从清晨慢慢变成黄昏，再进入宁静的夜晚——这种**日夜循环**效果能让玩家真正感受到时间的流逝，大幅提升地图的沉浸感。在这一节，你将学会两种核心技能：一是让光照强度自动变化，营造白天与黑夜的交替；二是调节光照的颜色，让不同场景拥有不同的氛围。

### 4.1 创建动态光照变化

> **⚠️ 常见错误**：很多新手以为日夜循环需要写复杂的代码，实际上魔兽争霸3的世界编辑器（World Editor）内置了专门控制时间的功能，不需要任何编程知识。

**操作步骤**

1. **打开场景控制面板** — 在世界编辑器界面，找到顶部菜单栏，点击“剧情”（Scenario）选项，然后在下拉菜单中选择“地图设置”（Map Options）[^2]。这一步就像打开电视机的设置菜单一样，是你调整游戏整体参数的地方。

2. **找到日夜循环开关** — 在地图设置窗口中，你会看到一个名为“启用日夜循环”（Enable Day/Night Cycle）的复选框，用鼠标左键点击它，在前面打上勾。这个开关就像是给地图安装了一个"时间控制器"，开启后游戏会自动按照设定的时间节奏切换白天和黑夜[^2]。

3. **设置一天的长度** — 在同一个设置区域，找到“一天的长度”（Day Duration）选项。这里需要你输入一个数字——数字越大代表现实中的1分钟对应的游戏天数越长，也就是说黑夜和白天交替得越慢。如果你想要玩家慢慢感受时间变化，建议设置为12或更大；如果想要快节奏的紧张感，可以设置为4或5[^2]。

4. **调整起始时间** — 找到“开始时间”（Starting Time of Day）选项，这里你可以选择游戏开始时是白天还是夜晚。比如你想让玩家从黄昏开始冒险，就选择"黄昏"选项。这个设置决定了玩家第一眼看到的画面是什么氛围[^2]。

> **💡 新手提示**：设置完日夜循环后，记得同时调整“雾化效果”（Fog）的开关。合适的雾气能让人物边缘更加柔和，让整个场景看起来更有层次感，否则白天和黑夜之间的过渡会显得很生硬[^5]。

### 4.2 光照颜色与冷暖色调运用

> **💡 新手提示**：光照的"颜色"不是指灯光本身的颜色，而是指整个场景被什么颜色的光线所笼罩。想象一下在黄色灯泡下看东西和白色日光下看东西的区别——这就是冷暖色调的区别。

**操作步骤**

1. **进入地形编辑器** — 在世界编辑器中，按下键盘上的快捷键 `Ctrl+T`，或者点击顶部工具栏的“地形”（Terrain）按钮，进入地形编辑模式[^3]。地形编辑器是制作地图的核心工作区，所有的视觉效果都在这里调整。

2. **打开光照控制面板** — 在地形编辑器界面，找到右侧面板中的“环境”（Environment）选项卡并点击展开。里面会有“环境光”（Ambient Light）和“日光”（Sun Light）两个滑块[^2]。环境光控制的是整个场景的底色亮度，日光控制的是来自"太阳"方向的光线强度。

3. **调节白天与夜晚的对比** — 拖动环境光滑块向左边（调暗），向右边（调亮）。一般白天环境光设置在80%到100%之间，夜晚则设置在20%到40%之间。这样设置后，当日夜循环运行时，两个时间段的亮度差异会非常明显，玩家能清晰感受到白天变黑夜的过程[^2]。

4. **使用预设氛围方案** — 世界编辑器内置了几种常见的光照预设（Lighting Presets），比如“北极光”“沙漠黄昏”“血色黎明”等。你可以直接点击这些预设快速应用到地图上，无需手动一个一个调整参数[^2]。建议先尝试几个预设，看看哪种最适合你的地图主题。

> **⚠️ 常见错误**：新手经常把白天和夜晚的光照强度设置得太接近，导致日夜循环几乎看不出变化。正确的做法是：白天至少比夜晚亮2到3倍，这样玩家才能明显感受到差异。记住，你是在做游戏效果，不需要"真实"，需要的是"明显"。

### 小结

完成以上两个小节的学习后，你应该已经掌握了为地图添加日夜循环的基础方法，能够调整光照强度让白天和黑夜形成明显的视觉对比，并尝试使用不同的光照预设来营造各种氛围。现在你可以打开你的地图，打开地图设置中的日夜循环开关，把一天长度设置为8，然后进入地形编辑器调整环境光参数，亲眼看看效果——试着在不同时间段暂停游戏，观察你的地图是否有了"活着"的感觉。

## 五、环境音效与视觉的配合

本节将教你让地图的"声音"和"画面"和谐搭配。完成学习后，你的地图将拥有身临其境的氛围感——比如阴雨天地图会响起淅沥雨声，篝火旁能听到噼啪火焰声。

### 5.1 添加背景音乐和环境音

背景音乐能第一时间告诉玩家"这是一个什么地方"，而环境音则让场景更加真实可信。

#### 操作步骤

1. **打开场景参数面板** — 在顶部菜单栏点击「场景」→「场景参数」[^4]，或直接按 `F4` 键

2. **进入音效设置** — 在弹出的「场景参数」窗口中，找到「环境音效」标签页

3. **添加背景音乐** — 点击「背景音乐」旁的「浏览」按钮，选择你准备好的 .mp3 或 .wav 格式音乐文件[^2]。建议选择循环播放的版本，这样音乐不会突然中断

4. **添加环境音效** — 在「环境音效」列表中点击「添加」，从下拉菜单选择适合你地图主题的音效[^3]，例如：
   - 森林地图 → 选择「森林」「鸟鸣」
   - 洞穴地图 → 选择「洞穴回音」「滴水」
   - 海边地图 → 选择「海浪」「海鸥」

5. **调节音量比例** — 将「背景音乐音量」设为 30%–50%，「环境音效音量」设为 50%–70%[^4]。背景音乐太响会盖过脚步声、对话声等重要音效

> **💡 新手提示**：环境音效不要贪多！同时播放 3 种以上的环境音会让玩家觉得嘈杂。选择 1–2 种最符合当前场景的音效即可。

> **⚠️ 常见错误**：新手经常忘记把音乐文件放在地图的「自定义资源」中。发布地图时，如果别人没有同等的音乐文件，是听不到音乐的！记得在「文件」→「导入资源」里把音乐文件导入到地图中[^3]。

---

### 5.2 音效与天气、光照的同步技巧

如果你的地图有动态天气变化（下雨、起雾等），让音效配合天气效果会大大提升沉浸感。这需要用到「触发器」功能——你可以把它理解成"如果...那么..."的自动执行规则。

#### 操作步骤

1. **打开触发器编辑器** — 在 WE 界面按 `F4`，或点击工具栏的「触发器」按钮[^4]

2. **创建新触发器** — 点击左上角「新建触发器」图标，将其命名为「天气音效同步」

3. **设置触发条件** — 双击触发器的「事件」部分，选择「游戏 - 天气改变」[^2]。这意味着：当天气发生变化时，下面的动作会自动执行

4. **添加音效动作** — 双击「条件」部分（留空即可），然后双击「动作」部分，选择「声音 - 播放声音」[^5]。设置如下：
   - 雨天 → 播放「雨声」音效
   - 雾天 → 播放「风声」音效
   - 晴天 → 播放「鸟鸣」或保持安静

5. **配合光照效果** — 如果你想让视觉和听觉同步，可以同时在「地形属性」中调整「光照强度」和「环境色」[^5]。比如：夜晚光照调暗 → 触发器播放「狼嚎」音效，玩家会自然联想到黑暗中的危险

> **💡 新手提示**：触发器是从上往下、从「事件」→「条件」→「动作」依次执行的。新手常搞混这个顺序，记住了！

> **⚠️ 常见错误**：忘记在触发器开头加上「关闭此触发器」动作又忘了最后「开启」。这样音效可能会重复播放。正确做法是在动作末尾加上「关闭触发器(当前触发器)」，并在最开头加上「开启触发器(当前触发器)」来重置状态。

### 小结

通过以上两个部分，你已经掌握了：
- ✅ 为地图添加背景音乐和环境音效
- ✅ 使用触发器让音效跟随天气自动变化
- ✅ 调节音量比例避免音效互相干扰
- ✅ 正确导入音乐文件确保发布后能正常播放

下一步，你可以尝试为地图的不同「区域」（Region）设置不同的环境音效，让玩家走进森林和走进沙漠时听到完全不同的声音！

## 六、实战案例：不同主题的氛围营造

学完本节后，你将掌握为不同类型地图场景设置合适灯光和雾效的方法，让玩家一眼就能感受到"这是森林"或"这是地下城"。我们用两个具体案例来演示操作步骤。

### 6.1 森林/自然场景的明亮氛围

森林场景需要让人感到开阔、温暖、有生命力。关键在于使用明亮的浅绿色雾气和暖色调灯光。

**操作步骤：**

1. **打开场景设置面板** — 在顶部菜单栏点击"场景"→"场景控制"（或直接按快捷键 Shift+S）[^3]
2. **设置雾气颜色** — 在弹出的面板中找到"雾气颜色"，选择浅绿色或淡蓝色，模拟阳光穿透树叶的效果
3. **调整雾气密度** — 将"雾气起点"设为500，"雾气终点"设为3500，数值越大雾气越薄
4. **设置环境光颜色** — 将"环境光"调为暖黄色（如RGB: 255, 230, 180），模拟阳光照射
5. **选择合适的地表材质** — 使用"森林"或"草地"地表格子，编辑器的默认绿植会比较适合[^2]

> **💡 新手提示**：森林场景的雾气颜色不要选得太深，否则玩家会感觉像"起雾的森林"而不是"阳光明媚的森林"。如果想让森林更有层次感，可以在不同区域设置不同的雾气密度——比如湖泊附近雾气浓一些，高地雾气稀薄。

### 6.2 地下城/废墟的阴暗氛围

地下城需要营造压抑、神秘的感觉。关键在于深色雾气、冷色调灯光和限制视野范围。

**操作步骤：**

1. **设置雾气颜色** — 选择深灰色或暗紫色（如RGB: 50, 50, 80），这种颜色会让整个场景看起来阴暗[^5]
2. **调近雾气起点** — 将"雾气起点"设为200，"雾气终点"设为1500，雾气会很快笼罩玩家视野
3. **降低环境光亮度** — 将"环境光"调为冷色调的深蓝色（如RGB: 80, 80, 120），模拟微弱火把或完全没有光源
4. **添加迷雾效果** — 在地图上绘制"迷雾区域"（通过区域工具），在区域内手动放置迷雾装饰物
5. **放置光源单位** — 从单位编辑器创建"火把"或"灯笼"等发光单位，放置在关键位置[^4]

> **⚠️ 常见错误**：新手做地下城时最容易犯的错误是——雾气太淡或环境光太亮，导致地下城看起来像"普通房间"。如果玩家抱怨"这地下城不够暗"，请检查雾气终点数值是否低于2500，或者环境光RGB值是否都超过100。

### 6.3 常见错误与调试方法

| 错误现象 | 原因 | 解决方法 |
|---------|------|---------|
| 场景看起来"灰蒙蒙"没有层次 | 雾气颜色太深或起点太近 | 调远雾气终点或选择更浅的雾气颜色 |
| 明明设置了灯光但场景还是很暗 | 环境光被设为全黑 | 环境光至少保留RGB值(20,20,20)的微弱光 |
| 不同区域雾气效果突然跳变 | 在区域边界没有使用过渡 | 使用"编辑"→"地形迷雾设置"中的过渡工具 |
| 玩家视野范围过大，缺少探索感 | 雾气终点设置太远 | 将雾气终点设为1200-2000之间 |

### 小结

完成以上两种场景的设置后，你应该已经掌握了：
- 如何通过雾气颜色、密度和环境光来塑造不同氛围
- 森林场景需要暖色调、远雾距；地下城需要冷色调、近雾距
- 通过调试雾气参数来解决"太暗/太亮/太灰"的问题

> **🎯 课后练习**：现在打开你的地图，尝试为一个副本入口设置"由明转暗"的过渡效果——入口区域用明亮雾气，进去后逐渐变成阴暗雾气。这个技巧会让你的地图场景更有代入感！
