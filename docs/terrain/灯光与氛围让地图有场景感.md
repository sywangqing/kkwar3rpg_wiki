# 灯光与氛围：让地图有场景感

## 灯光基础概念：理解WC3的光照系统

在制作地图时，你有没有注意到同样是一片森林，有的地图看起来阴森恐怖，有的却温暖舒适？这种给玩家带来的"感觉"，我们称之为**场景感**。而灯光，就是创造场景感最重要的工具之一。本节我们将学习WC3的光照系统是什么，以及它如何影响你的地图氛围。学完，你就能理解为什么有些地图看起来专业，有些看起来"灰蒙蒙"的。

### 操作步骤

1. **第一步：了解什么是场景感** — 场景感就是玩家进入你的地图时，整体感受到的氛围和情绪。比如黑暗的洞穴让人紧张，阳光明媚的村庄让人放松。[^1] 即使两块地图的地形完全相同，只要灯光设置不同，给人的感觉就会完全不同。

2. **第二步：认识WC3的三种基础光照类型** — WC3编辑器中有三种主要的光照类型[^2]：
   - **环境光照**：这是整个地图的"底色亮度"，类似阴天时室外的自然光
   - **方向光照**：模拟太阳或月亮，有明确的照射方向，会产生阴影
   - **点光源**：像火把、灯笼这样的点状光源，照亮周围一小片区域

3. **第三步：在编辑器中查看当前光照设置** — 打开World Editor，点击顶部菜单 **"场景"→"光照"**（或使用快捷键），你可以看到当前地图使用的光照配置文件[^3]。试着切换不同的光照预设，观察地图整体亮度和色调的变化。

4. **第四步：理解"氛围"是如何通过灯光实现的** — 当你选择了"夜晚"光照，地图会变暗，色调偏蓝紫色；当选择"黄昏"光照时，会呈现橙红色的暖色调。这就是为什么选择正确的光照，是创造逼真场景的第一步。[^1]

> **💡 新手提示**：别小看基础光照设置！很多新手地图看起来"不专业"的原因不是地形不好，而是使用了错误的光照预设。比如森林地图用了"沙漠"光照，看起来就会很违和。

> **⚠️ 常见错误**：新手容易犯的错是认为"地图越亮越好"。其实不然！适当降低环境光照亮度，反而能让方向光和点光源的效果更突出，营造出更有层次感的场景。

### 小结

完成以上步骤后，你应该已经理解了什么是场景感，以及WC3中三种基础光照类型（环境光、方向光、点光源）各自的作用。下一步，我们将学习如何具体调整这些光照参数，让你的地图拥有独特的氛围！

## 地图照明设置：环境光与主光源

本节将教你如何调整地图的灯光设置，包括环境光颜色和主光源方向。学会这些技巧后，你的地图将拥有电影般的场景氛围，不再是单调的"白开水"画面。

### 环境光(Ambient)颜色调节技巧

1. **打开场景设置菜单** — 在顶部菜单栏点击"场景"选项[^1]
2. **找到环境光设置** — 在下拉菜单中选择"环境光颜色"（英文版显示为"Ambient"）
3. **选择合适的基础色** — 点击颜色方块打开调色板
4. **调整RGB数值** — 拖动红、绿、蓝三个滑块，或者直接输入0-255的数值[^2]

> **💡 新手提示**：环境光就像一层薄纱笼罩在整个地图上，它决定了"底色"。森林地图建议用淡绿色（RGB: 180,200,150），沙漠用暖黄色（RGB: 220,200,150），地下城用深紫色（RGB: 80,60,100）。

> **⚠️ 常见错误**：新手容易把环境光调得太亮或太白，导致整个地图看起来"灰蒙蒙的"。记住：环境光的亮度应该配合你的主光源，不要喧宾夺主！

### 主光源方向与阴影角度设置

1. **进入光照设置** — 在"场景"菜单中选择"光照"选项[^3]
2. **设置太阳高度角** — 拖动"方向"滑块（0°=正午太阳在头顶，负值=早晨/黄昏）
3. **调整阴影角度** — 根据太阳位置，阴影会向相反方向延伸
4. **预览效果** — 切换到地形查看阴影投射是否自然

> **💡 新手提示**：制作"永恒之夜"地图时，把太阳角度调低（如-60°）并用蓝色环境光；制作"正午战场"则用0°左右的正上方太阳，配合白色主光源。

### 小结

完成以上设置后，你的地图应该拥有统一的氛围色调和自然的阴影效果。建议先选择一个预设方案作为起点，再根据实际效果微调环境光与主光源的搭配。

## 氛围与天气效果：让地图活起来

在地图上添加天气效果和设置迷雾，是让玩家感受到"身临其境"的重要手段。一个阴雨绵绵的森林和一个阳光明媚的沙滩，给人的感觉完全不同。这一节你将学会如何为地图添加雨、雪、雾等天气效果，以及如何设置迷雾和玩家的视野范围。

### 天气效果添加：雨、雪、雾

1. **打开场景管理器** — 在顶部菜单栏找到 **"场景"** 菜单，点击 **"场景控制"**（或使用快捷键 `F4`）[^1]
2. **切换到环境标签** — 在弹出的场景控制面板中，找到并点击 **"环境"** 这一页签
3. **选择天气类型** — 在环境设置中，找到 **"天气"** 下拉菜单。这里列出了各种预设天气：
   - 雨（Rain）
   - 雪（Snow）
   - 雾（Fog）
   - 暴风雪（Storm）
   - 无（None）
4. **调整天气参数** — 选中一种天气后，下方会出现几个可调节的滑块：
   - **影响范围**：天气覆盖多大区域（数值越大，覆盖越广）
   - **密度**：天气效果有多强烈（数值越大，雨雪越密集）
5. **点击确定应用** — 设置完成后，点击 **"确定"** 按钮保存[^2]

> **💡 新手提示**：天气效果只在游戏运行时可见，在编辑器界面你只能看到设置面板。如果想预览效果，需要点击 **"测试地图"**（快捷键 `F9`）进入游戏测试。

> **⚠️ 常见错误**：很多新手设置了天气后忘记点"确定"，结果退出面板后发现天气没有变化。一定要确认点击了确定按钮！

### 迷雾与视野范围设置

1. **打开场景管理器** — 同样通过 **"场景" → "场景控制"**（或按 `F4`）打开面板[^1]
2. **选择迷雾标签** — 在场景控制面板中，点击 **"迷雾"** 标签页
3. **设置迷雾颜色** — 点击 **"迷雾颜色"** 色块，可以选择迷雾的基本色调。比如森林地图常用深绿色，洞穴常用深蓝色
4. **调整迷雾密度** — 拖动 **"密度"** 滑块控制迷雾的浓厚度。数值越高，画面越模糊
5. **设置玩家视野** — 找到 **"视野范围"** 选项，设置玩家的基本可视距离。默认值通常在 1800 左右，你可以根据地图大小调整
6. **点击确定保存**

> **💡 新手提示**：迷雾和天气效果可以叠加使用。比如在下雨天设置浓雾，会营造出非常压抑的氛围！但要注意，迷雾太浓可能会影响玩家操作，设置前先问自己：这样设置对游戏体验有帮助吗？

### 小结

完成以上步骤后，你应该能够为地图添加各种天气效果（雨、雪、雾），并根据需要调整迷雾的浓淡和颜色。试着打开一张新地图，分别尝试不同的天气组合，然后用测试模式体验一下效果差异。找到最适合你地图风格的天气设置吧！

## 地形配色与视觉层次营造

你有没有注意到，同样是一片草地，为什么有的地图看起来"有feel"，有的却像一块绿布？这节课我们就来学习如何通过**地形配色**（地图上不同地形元素的颜色搭配）和**视觉层次**（让画面有前中后景的立体感）来提升地图的"颜值"。学完后，你的地图会告别"草稿感"，开始有专业地图的样子了！

### 操作步骤

1. **第一步：打开地形设置面板** — 在世界编辑器菜单栏点击"场景"（Scenario），选择"地形"（Terrain）选项[^1]。这里可以调整整张地图的基础色调。为什么要从这里开始？因为基础色调决定了整张地图的"情绪底色"，后面的一切都基于它调整。

2. **第二步：选择符合主题的灯光色调** — 在地形设置中，你会看到"灯光"（Lighting）选项。如果你的地图是阴暗的地下城主题，选择偏冷偏暗的蓝灰色调；如果是阳光明媚的草原，选择暖黄色调[^2]。这里有个小技巧：灯光色调要和你的地图故事背景匹配——想象一下，如果一张恐怖地图用了粉红色调，玩家会不会出戏？

3. **第三步：调整对比度营造深度感** — 地形工具栏中找到"调色板"（Palette），选择"变化"（Variation）功能[^3]。用深色地形覆盖远处的区域，浅色覆盖近处区域。简单来说就是：**近处亮、远处暗，中间过渡要平滑**。这和我们人眼看东西的规律一样——远处的物体看起来颜色会更淡、更蓝（空气透视原理）。

4. **第四步：添加视觉层次点缀** — 在你觉得"太平"的地方（比如大草原中间），用不同的地形变体或装饰物（如树木、岩石）打破单调。这些点缀物要**前小后大、前疏后密**，帮助建立立体感[^1]。

> **💡 新手提示**：配色不是越多越好！新手常犯的错误是恨不得把调色盘里所有颜色都用上，结果画面变得杂乱。记住"**三色原则**"——主色调、辅助色、点缀色各选一种，整体效果会更和谐。

> **⚠️ 常见错误**：很多新手把整张地图的对比度调得过高，导致"假假的塑料感"。正确的做法是**让对比度渐变过渡**，不要在相邻的两个区域突然从最亮跳到最暗。解决方法是每次调整时把变化幅度控制在20%以内，然后多预览几次。

### 小结

完成以上步骤后，你应该能看到你的地图有了明显的"场景感"——玩家能一眼分辨出哪里是近景、哪里是远景，整张地图不再"平"，而是有了呼吸感。多尝试不同的灯光色调组合，找到最适合你地图主题的风格！

## 实用案例：从零开始打造场景氛围

本节将通过两个具体案例（森林地图和地下城），手把手教你设置灯光和氛围。学完之后，你将能够为不同类型的地图营造出独特的视觉风格，让玩家一眼就能感受到地图想表达的情绪。

### 森林地图灯光设置步骤

1. **第一步：选择基础地形** — 打开地图后，点击地形工具栏（Terrain Palette），选择草地类贴图[^2]。对于森林地图，推荐使用颜色偏深绿的草地作为底层，这样后续添加的树木和光影效果会更加明显。

2. **第二步：调整整体光照颜色** — 在菜单栏选择"场景"（Scenario）→"光照"（Lighting）。森林场景建议使用**淡黄绿色**或**暖白色**作为环境光颜色[^3]，这样能模拟阳光穿过树叶的效果。强度建议设置在0.8-1.0之间，太亮会失去森林的神秘感。

3. **第三步：添加方向光（太阳光）** — 点击"添加方向光"，将角度调整为从**左上方45度**照射[^2]。这个角度最符合自然规律，能在地上投射出树木和地形的影子。颜色选择金黄色，强度设为0.6左右。

4. **第四步：设置雾气效果** — 打开"高级"（Advanced）菜单中的"环境"（Environment）设置。森林地图推荐开启**浅绿色雾气**，密度设置为0.003-0.005[^3]。雾气能让远处的树木变得朦胧，增加空间深度感。

> **💡 新手提示**：森林地图的灯光颜色要和草地贴图协调！如果草地偏暗，灯光就不要选太亮的白色，否则会显得不真实。

### 地下城/洞穴氛围营造技巧

1. **第一步：关闭方向光** — 地下城不需要太阳光。进入光照设置，将方向光的强度调整为**0**，只保留环境光照明。

2. **第二步：设置冷色调环境光** — 地下城的氛围应该是阴冷潮湿的。将环境光颜色调整为**深蓝色**或**灰紫色**[^3]，强度设置为0.3-0.5。光线越暗，神秘感越强。

3. **第三步：添加局部光源** — 地下城不适合全局照明，而是需要用**点光源"火把"**来照亮关键区域。在需要放置火把的位置，使用"装饰物"（Doodads）中的火把模型，它们自带发光效果。

4. **第四步：调整深色雾气** — 地下城的雾气应该是浓重的深灰色或黑色[^3]。在环境设置中，将雾气颜色改为深灰（RGB值约30,30,40），密度提高到0.01以上。这样玩家只能看清火把周围的区域，远处完全隐藏在黑暗中。

> **⚠️ 常见错误**：很多新手把地下城的雾气颜色设置成和外面一样的浅色，这会让洞穴看起来像白天！记住，洞穴雾气一定要足够暗才能营造压迫感。

### 常见错误与修正方法

| 错误类型 | 问题表现 | 解决方法 |
|---------|---------|---------|
| 灯光颜色与地形不协调 | 蓝色灯光配黄色草地，看起来很假 | 灯光颜色应该和地形主色调相近或为互补色 |
| 雾气太淡 | 雾气形同虚设，地图没有纵深感 | 森林雾气密度至少0.003，洞穴至少0.01 |
| 方向光角度错误 | 影子方向不一致，看着别扭 | 所有地形区域的方向光角度保持一致 |
| 忽略预览测试 | 在编辑器里看着挺好，进游戏很暗 | 养成习惯：每次调整完灯光都按F9实际测试一下 |

> **💡 新手提示**：如果你不确定某种灯光设置的效果，可以先在一个**小测试地图**里单独练习，不要直接在正式地图上改来改去。保存一个"灯光参数备份"文档，记录你用过的满意参数组合。
