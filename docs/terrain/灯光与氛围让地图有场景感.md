# 灯光与氛围：让地图有场景感

## 什么是灯光与氛围？

在本节中，我们将学习什么是灯光与氛围，以及它们为什么能让你的地图从"一堆模型"变成"一个让人想探索的世界"。学完本节后，你会理解灯光如何影响玩家的心情，并为后续学习调整灯光的具体操作打下基础。

### 为什么灯光对地图很重要

当你打开一张地图时，第一眼看到的不是触发器（触发器：让游戏事件自动执行的东西，比如玩家进入区域时自动战斗），也不是脚本代码，而是**整体的视觉感受**。灯光就是塑造这种感受的核心工具。

在魔兽争霸III中，灯光系统通过"DayandNight模型"（日夜照明模型）来实现白天和黑夜的切换[^1]。简单来说，模型会告诉游戏："在白天，灯光应该是暖黄色的；在夜晚，灯光应该是冷蓝色的。"当你学会调整这些设置，你的地图就会有**时间变化**的感觉，而不只是静态的一成不变。

### 灯光对玩家情绪的影响

你可能玩过这样的RPG：明明是紧张的战斗场景，但灯光却是暖洋洋的橙色，让你完全感受不到紧迫感——这就是灯光和氛围没有配合好的例子。

好的灯光设计能帮助玩家**下意识地感受到当前场景的情绪**：

- **明亮、暖色调的灯光** → 让玩家感到安全、轻松，适合村庄、商店等日常场景
- **昏暗、冷色调的灯光** → 让玩家感到紧张、神秘，适合地下城、战斗场景
- **红色或紫色的特殊灯光** → 暗示危险或魔法力量的存在

> **💡 新手提示**：不要小看灯光的力量！很多新手制作的地图内容很丰富，但看起来"不够专业"，往往就是因为忽略了灯光的细节调整。

> **⚠️ 常见错误**：新手常犯的错误是完全不调整灯光，直接使用地图编辑器的默认设置。这样会导致你的地图和成百上千张其他地图看起来一模一样，没有任何个人特色。

### 小结

灯光与氛围不仅仅是"让地图变亮或变暗"，而是**通过颜色、亮度、时间变化来传递情感和故事信息**的设计工具。下一节我们将学习如何实际调整这些灯光设置，让你的地图真正拥有独特的场景感。

## World Editor中的灯光工具

本节将教你找到并使用World Editor中的灯光设置面板，学会后你可以为地图添加基础的氛围效果，让你的世界看起来有白天、黑夜的区别。

### 操作步骤

1. **第一步：打开高级设置菜单** — 在World Editor顶部菜单栏中，点击「视图」→「高级」选项[^1]。这会展开一个包含多种高级设置的下拉菜单，包括我们现在要用的灯光工具。

2. **第二步：找到灯光设置** — 在展开的「高级」菜单中，选择「切换日光」选项[^2]。这个功能允许你调整地图的光照效果，包括颜色、强度以及白天与黑夜之间的过渡方式。

3. **第三步：理解基础灯光参数** — 打开灯光面板后，你会看到几个关键参数[^1]：
   - **环境光颜色**：控制整个地图的基础照明色调，比如暖黄色代表白天，蓝色代表夜晚
   - **方向光**：模拟太阳光的角度和颜色，决定场景中物体阴影的方向
   - **日夜切换**：调整白天、黑夜、黎明、黄昏四个时段的灯光配置

> **💡 新手提示**：调整灯光时，记得先在地图上放置一个简单的建筑或单位作为参照物，这样你能更直观地看到灯光变化对场景的影响。

> **⚠️ 常见错误**：很多新手调整完灯光后发现没有变化，这是因为忽略了「日夜切换」按钮旁边的「启用日夜循环」选项。一定要勾选这个选项，灯光效果才会真正应用到游戏中。

### 小结

完成以上步骤后，你应该能够打开灯光设置面板，并调整基础的环境光颜色和方向光设置。建议现在就去尝试把环境光从默认的白色改成暖黄色，看看你的地图瞬间变得多么有"黄昏感"！

## 灯光类型详解

本节将带你认识魔兽争霸3地图编辑器中的三种主要灯光类型，了解它们各自的作用，并学会如何调整它们来营造你想要的场景氛围。学完本节后，你将能够为不同区域设置不同的环境光颜色，以及让地图实现真实的昼夜变化效果。

### 环境光（Ambient Light）

环境光是指照亮整个地图的"天空光线"，它决定了地图的基础色调。你可以把环境光想象成一层薄薄的彩色玻璃覆盖在整个地图上。

**环境光设置步骤：**

1. 打开魔兽争霸3地图编辑器，点击顶部菜单的 **"场景"** 选项[^1]
2. 在下拉菜单中找到 **"地形属性"** 或 **"地图设置"**，点击进入[^2]
3. 在弹出的面板中，你会看到 **"环境光"** 或 **"Ambient Color"** 的颜色选择器
4. 点击颜色块，打开调色板，选择你想要的颜色
5. 通常白天使用偏蓝或偏白的颜色，夜晚使用偏紫或偏蓝的颜色[^2]
6. 调整完毕后点击 **"确定"** 保存设置

> **💡 新手提示**：环境光的颜色决定了地图的"心情"——暖色调（橙/黄）给人温暖快乐的感觉，冷色调（蓝/紫）则营造神秘或悲伤的氛围。试试把白天设成淡蓝色，你会发现整个地图立刻有了清晨的感觉。

> **⚠️ 常见错误**：很多新手把环境光调得太暗，导致整个地图看起来灰蒙蒙的。建议白天环境光值保持在70%以上，夜晚也不要低于20%，否则地形纹理会看不清楚。

### 单位光（Unit Light）

单位光是一种特殊的灯光系统，它可以让单位（英雄、小兵、建筑等）在白天和夜晚呈现出不同的颜色效果。这需要使用特殊的"日夜模型"（DayandNight Model）来实现[^1]。

**日夜模型工作原理：**

魔兽争霸3的单位模型文件内部包含了多个颜色层，分别对应白天、黄昏和夜晚三个时段[^1]。当昼夜系统运行时，游戏会自动切换这些颜色层，让单位看起来与当前时段的光照条件协调。

**创建日夜灯光效果的步骤：**

1. 首先需要一张支持日夜变化的贴图或模型文件（通常后缀带有"Day"和"Night"的模型）
2. 使用 **War3 Model Editor** 打开你想要的模型文件[^1]
3. 在编辑器中找到 **Node Manager（节点管理器）** 面板
4. 展开灯光相关的节点，找到 **Omnilight（泛光灯）** 或类似选项
5. 双击节点，可以调整不同时间段的 **光颜色设置**[^1]
6. 调整完毕后保存模型，替换到你地图的单位上

> **💡 新手提示**：如果你不想手动制作日夜模型，也可以去HIVEworkshop等资源网站下载现成的日夜单位模型[^2]。这些模型已经设置好了白天发暖光、夜晚发冷光的效果。

### 昼夜循环与灯光变化

如果你想让地图有真实的白天→黄昏→夜晚→清晨的循环，光设置环境光是不够的，还需要配合触发器或使用编辑器内置的昼夜系统来实现灯光的动态变化。

**启用昼夜循环系统的步骤：**

1. 在地图编辑器中，点击 **"场景"** → **"地形属性"**
2. 找到 **"启用昼夜循环"** 的勾选框，打上勾
3. 设置 **"一天的长度"** 参数（数值越大，一天过得越慢）
4. 设置 **"黎明"**、**"正午"**、**"黄昏"**、**"午夜"** 四个时段的环境光颜色

**高级技巧——用触发器控制灯光：**

如果你想让特定区域在特定时间呈现不同灯光效果，可以：

1. 新建一个 **触发器**（触发器是一种"如果...那么..."的逻辑系统）
2. 添加 **"时间"** 相关的事件条件
3. 在动作中使用 **"设置环境光颜色"** 的动作
4. 这样当游戏时间到达黄昏时，所有区域的环境光会自动变成黄昏色调[^2]

> **⚠️ 常见错误**：新手容易忽略日夜模型需要在模型编辑器中转换格式才能正常显示[^2]。如果你的日夜效果在编辑器里看起来很奇怪（比如颜色错乱），记得用专门的转换工具处理一下模型文件。

### 小结

完成以上学习后，你应该掌握了：
- ✅ 如何调整地图的基础环境光颜色
- ✅ 了解日夜模型如何让单位产生时段感
- ✅ 开启昼夜循环系统让地图有动态光效
- ✅ 使用触发器实现更精细的灯光控制

现在你可以打开编辑器，试着把一张普通地图的环境光从默认改成偏暖的橙黄色，你会发现地图的氛围立刻从"普通"变成了"沙漠黄昏"！多尝试不同的颜色组合，找到最适合你RPG剧情的灯光风格吧。

## 色彩理论与情绪营造

灯光的颜色直接影响玩家感受地图的氛围。学会用颜色讲故事，你的地图就不再是"空白场地"，而是一个有生命的世界。

### 不同颜色传达的情绪

色彩心理学在游戏设计中非常重要。暖色调（红、橙、黄）让人感到温暖、危险、充满活力；冷色调（蓝、绿、紫）则让人感到神秘、安静或阴森[^1]。

| 颜色类型 | 适合的场景 | 传达的情绪 |
|---------|-----------|-----------|
| 暖色系 | 村庄、篝火、战斗区域 | 安全、热情、紧张 |
| 冷色系 | 地下城、森林深处、夜晚 | 神秘、压抑、未知 |
| 中性色 | 平原、道路、中立区域 | 平静、过渡 |

### 常见场景的配色方案

**第一步：打开地图的"环境设置"**[^2]
- 在WE（World Editor）顶部菜单找到 **场景 → 环境**
- 这里可以调整天空颜色、地面光照颜色和强度

**第二步：选择适合场景的光照颜色**
- 白天战斗地图：使用**橙黄色**光照（#FFD700 左右）
- 夜晚/地下城：使用**深蓝色**或**紫色**光照（#4169E1 或 #8A2BE2）
- 森林场景：使用**浅绿色**光照（#90EE90）
- 沙漠/荒原：使用**橙褐色**光照（#D2691E）

**第三步：预览并微调**
- 按 **F9** 测试地图，观察实际效果
- 如果感觉颜色太刺眼，降低"光照强度"数值
- 如果想要更明显的氛围感，增加"环境光"比例

> **💡 新手提示**：不要只改一个区域就完事！试着同时调整"天空盒"颜色（场景 → 环境 → 天空），两者配合才能营造出完整的氛围。很多新手只调光照，忽略了天空，导致白天场景看起来"不蓝"。

> **⚠️ 常见错误**：新手常用纯白色光照（RGB 255,255,255）作为默认设置。这样会让地图看起来"平"，没有层次感。正确的做法是让光照带有轻微的色彩倾向——即使是"白色"场景也应该是暖白或冷白。

### 小结

完成以上步骤后，你的地图应该有了明显的场景感：白天温暖明亮，夜晚神秘幽暗。记住，**颜色是玩家进入地图第一眼看到的东西**，用心调配色准，能让你的地图从"试玩版"瞬间提升到"成品感"！

## 天气效果与雾气设置

本节将教你为地图添加天气效果和雾气设置，让你的地图场景更有层次感和沉浸感。完成学习后，你将能够为不同区域设置独特的天气类型和雾气效果。

### 天气类型与灯光配合

在魔兽争霸III中，天气效果不仅仅是装饰，它能与灯光系统配合创造出白天、夜晚、黎明、黄昏等不同时间段的氛围感[^1][^2]。

### 操作步骤

1. **打开场景控制面板** — 在编辑器顶部菜单栏找到"场景"选项，点击展开后选择"环境"（Environment）。这是所有天气和雾气设置的"大本营"[^2]。

2. **设置基础天气类型** — 在环境面板中，你会看到"天气"下拉菜单。常见的天气类型包括：无（None）、雨（Rain）、风暴（Storm）、雪（Snow）等。选择一种天气后，地图会自动播放对应的粒子效果。

3. **选择与灯光匹配的天气** — 点击天气下拉菜单，选择适合你地图氛围的天气类型。比如阴沉的暴风天气配合深蓝色的夜间灯光，能营造出紧张的战斗氛围[^1]。

4. **微调天气强度** — 找到"天气缩放"参数，将其调整为0.5~1.5之间的数值。低于1会减少粒子数量，高于1会增强粒子密度。适合不同大小的地图区域。

### 雾气效果增强场景感

雾气是创造空间纵深感的利器。合理设置雾气可以让远处的景物若隐若现，增加神秘感或远景模糊效果。

### 操作步骤

1. **启用雾气效果** — 在环境面板中找到"使用雾气"复选框，确保其处于勾选状态。如果不勾选，后续的雾气颜色和密度设置都不会生效。

2. **设置雾气颜色** — 点击"雾气颜色"旁边的色块，选择你想要的颜色。浅色雾气适合明亮的白天场景，深灰色雾气则更适合阴郁的地下城或夜间环境。

3. **调整雾气起始距离** — "雾气起始距离"控制从哪个距离开始物体被雾气遮挡。数值越小，雾气从越近的地方就开始遮挡视线。

4. **调整雾气结束距离** — "雾气结束距离"控制雾气完全遮挡的距离。这个值与起始距离的差值越大，雾气过渡越平缓。

> **💡 新手提示**：雾气颜色最好与整体灯光色调保持一致！比如夜晚用深蓝色雾气，白天用浅灰白色雾气，这样看起来更自然协调。

## 新手常见问题与优化

本节将帮助新手识别制作地图时最常见的灯光错误，并提供实用的性能优化建议。学完本节后，你将能够避免90%的灯光设置陷阱，并让地图运行更流畅。

### 新手易犯的灯光错误

1. **忽视昼夜系统设置** — 许多新手只调整了白天的灯光，却忘记在"地图设置"中启用昼夜循环功能[^2]。这会导致辛苦设置的灯光效果在游戏中完全看不到。
   - 解决方法：在编辑器顶部菜单 → 场景 → 昼夜交替，选择"启用"
2. **灯光颜色过于刺眼** — 新手常把环境光亮度调到100%，使整个地图显得"发白"[^1]。正确做法是保持适度对比度，让阴影区域有足够深度。
3. **忘记保存为白天/夜晚模型** — 设置好灯光后，如果要让不同时间段呈现不同效果，必须使用"DayandNight"模型[^1]。普通模型无法实现昼夜光效变化。

> **⚠️ 常见错误**：有人在调整灯光时频繁修改数值却不预览，这样效率很低。正确做法是先在编辑器中实时预览效果，确认满意后再保存。

> **💡 新手提示**：建议制作多张截图对比不同时间段的灯光效果，便于发现设置不当的地方。

### 灯光性能优化建议

1. **减少同时使用的光源数量** — 每个额外的灯光都会消耗游戏性能。普通地图控制在10-15个点光源内即可。
2. **避免使用过大的光照范围** — 灯光范围过大会增加渲染负担。将光照范围设置为刚好覆盖目标区域即可。
3. **合并相似光源** — 如果地图中有多个位置需要相同的光照效果，可以考虑用触发器（一种自动执行命令的脚本系统）在不同区域循环切换同一光源，而不是同时开启所有灯光。

> **💡 新手提示**：制作完地图后，记得用"测试地图"功能实际运行一次，观察在不同时间段切换时是否出现卡顿。

### 小结

完成本节学习后，你应该能够识别并修复常见的灯光配置问题，包括启用昼夜系统、控制灯光亮度以及使用DayandNight模型。同时，你学会了通过限制光源数量、优化光照范围和合并相似光源来保持地图流畅运行。建议现在打开你的地图，逐一检查是否存在上述问题，并进行优化。
