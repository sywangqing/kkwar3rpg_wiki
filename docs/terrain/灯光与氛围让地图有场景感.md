# 灯光与氛围：让地图有场景感

## 基础概念：什么是灯光与氛围

本节将带你了解什么是灯光与氛围，以及它们为什么能让你的地图从"草稿"变成"作品"。学完后，你将能在地图中设置基本的灯光和氛围效果，让玩家一进入你的地图就感受到独特的场景感。

### 为什么灯光和氛围很重要

当你玩一张地图时，有没有注意过有些地图让你觉得很"冷"、很"荒凉"，而有些地图却让你觉得很"温暖"、很"有生机"？这种感觉很大程度上就是灯光和氛围带来的。

**氛围**指的是一个场景给玩家带来的整体感受——它由灯光颜色、雾气浓度、阴影强度等多种元素共同构成。就像拍电影时，导演会用不同的光线和色调来表达不同的情绪，游戏地图也是一样。

场景设计需要大量的美术知识，包括构图的技巧、空间关系、远景里的拉伸、大气透视等[^1]。好的氛围设计能让玩家沉浸其中，差的氛围则会让玩家觉得"哪里不对但说不出来"。

> **💡 新手提示**：别小看氛围的作用！很多新手做地图时只关注地形是否好看，却忽略了氛围，结果地图看起来"平"、"假"，玩家也说不上来为什么不喜欢。

### 氛围如何影响玩家体验

在魔兽争霸III地图编辑器中，氛围主要通过以下元素实现：

- **灯光颜色**：冷色调（蓝色、青色）给人寒冷、神秘的感觉；暖色调（橙色、黄色）给人温暖、舒适的感觉
- **雾气效果**：浓雾可以营造神秘感或远距离感，薄雾则让场景更清晰
- **环境色调**：整体画面的颜色倾向会影响玩家的情绪

使用材质相近但造型不同的模型可以制作出不同效果[^1]。比如同样是石头模型，用暖色灯光照射显得古朴温馨，用冷色灯光照射就显得冰冷肃穆。

> **⚠️ 常见错误**：新手容易把所有地图都用默认的"绿色草地+蓝天白云"，完全不做氛围调整。这样做出来的地图千篇一律，毫无特色。请记住：即使是森林地图，根据故事背景不同，也可以做成阴森黑暗的黑暗森林，或阳光明媚的精灵家园。

### 操作步骤

在地图编辑器中设置氛围的基本方法：

1. **打开地形编辑器** — 在World Editor主界面点击"地形编辑器"按钮[^3]
2. **进入场景设置** — 在顶部菜单栏找到"场景"选项，点击"场景控制"或直接使用快捷键（通常是Ctrl+Shift+E）
3. **调整基本环境参数** — 在弹出的面板中，你可以设置：
   - **环境光颜色**：控制整体光照基调
   - **雾效颜色**：设置远处场景被什么颜色"染上"
   - **雾效密度**：调整雾气浓淡
   - **光照强度**：调整明暗程度
4. **设置白天/夜晚循环**（可选）— 如果你的地图需要时间变化，可以在这里开启昼夜循环

### 小结

完成以上步骤后，你应该能看到你的地图整体色调发生了变化。原本"灰蒙蒙"的地图现在有了自己的"性格"。你可以多尝试不同的灯光颜色和雾效组合，找到最适合你地图主题的氛围设置。

> **💡 新手提示**：设置好后，记得按F9预览地图，从玩家视角看看氛围效果是否满意。很多时候在编辑器里看着还行，实际预览时感觉就不一样了。

## Warcraft III 照明系统详解

本节将带你了解 Warcraft III 自带的三大照明功能：环境光、单位阴影和昼夜周期。学完本节后，你能让地图告别"灰蒙蒙"的单调感，营造出有层次的视觉氛围。

### 环境光与全局照明

环境光是指照亮整个地图的基础光源，它决定了地图的整体色调和明暗基调。[^1]

1. **打开场景设置**：在编辑器菜单栏中点击「场景」→「场景控制」（或使用快捷键 F4）[^3]
2. **调整环境光亮度**：在弹出的面板中找到「环境光」滑块，向右拖动让地图更明亮，向左则更昏暗
3. **选择环境光颜色**：点击颜色框可以自定义环境光的颜色——暖黄色营造黄昏感，蓝灰色则适合阴森氛围
4. **预览效果**：拖动过程中可以实时看到地图变化，找到满意的效果后点击确定

> **💡 新手提示**：环境光颜色要与地图主题匹配。比如森林地图适合用绿色调，沙漠地图用金黄色调，冰霜地图则适合冷蓝色调。[^1]

### 单位阴影设置

单位阴影让游戏中的单位和建筑在地面上投下阴影，大幅提升真实感。[^2]

1. **进入高级设置**：在「场景控制」面板中点击「高级」标签页
2. **启用单位阴影**：勾选「单位阴影」复选框
3. **选择阴影类型**：
   - 「软阴影」：边缘模糊柔和，效果更真实但消耗更多性能
   - 「硬阴影」：边缘清晰锐利，性能友好
4. **调整阴影密度**：通过「阴影半径」滑块控制阴影大小，半径越大阴影越柔和

> **⚠️ 常见错误**：新手容易忽略阴影对性能的影响——开启所有单位的软阴影后，地图单位较多时会明显卡顿。建议在单位密集的地图中使用硬阴影或只给重要单位开启阴影。

### 昼夜周期系统

昼夜周期能让地图模拟白天、黑夜的时间流逝，为战斗增添紧迫感。[^2]

1. **启用昼夜周期**：在「场景控制」面板中勾选「启用昼夜交替」
2. **设置一天时长**：在「一天时长」输入框中填入数值（单位为秒），数值越大表示一天时间越长
3. **调整晨曦与黄昏颜色**：点击「黎明色」和「黄昏色」可以分别设置这两个过渡时段的氛围
4. **设置关键时间点亮度**：拖动「正午」「午夜」等时间点的光照强度滑块

> **💡 新手提示**：昼夜周期最适合带有任务/剧情的 RPG 地图，比如设置夜晚怪物攻击力增强、商店关门等设定，让玩家感受到时间的紧迫。

### 小结

完成以上设置后，你的地图应该拥有了：
- 与主题匹配的整体色调（环境光）
- 立体感更强的单位和建筑（阴影）
- 随着时间变化的动态光照（昼夜周期）

现在打开编辑器，选一张旧地图试试这些设置，观察不同参数带来的视觉效果变化吧！

## 地形编辑中的氛围营造

在这节课里，我会教你三种方法来让地图更有"场景感"——就像电影里的布景一样，让玩家一进去就想探索。学完之后，你的地图就不再是光秃秃的"对战图"，而是能让人沉浸其中的故事世界。

### 地形高度与阴影的配合

为什么有些地图看起来很"平"，有些却很有立体感？秘密就在于**地形高度**的运用。

1. **打开高度工具** — 在编辑器左侧工具栏找到"地形塑造"图标（一个带波浪线的小方块）[^1]
2. **选择画笔大小** — 按住鼠标左键，在地图上点击并拖动，地面就会升高；按住**Shift**再拖动，地面就会降低
3. **调整高度差** — 试试把山崖、河流两岸做成不同高度，视觉上立刻就有"层次感"了[^1]
4. **观察阴影变化** — 地形高度改变后，阴影会自动生成。你不需要手动画阴影，编辑器会根据高度差帮你计算好

> **💡 新手提示**：一开始别把高度差做得太夸张！新手常常把山坡做成"悬崖"，结果玩家角色根本爬不上去。除非你故意要做悬崖地形，否则建议每次调整高度时幅度小一点，多试几次找到合适的感觉。

### 如何使用地形雾气

**地形雾气**听起来很专业，其实就是给地图加上一层朦胧的"空气感"——就像真实世界里远处的东西会变模糊一样。

1. **找到雾气设置** — 点击菜单栏的"场景"→"地形雾气"[^2]
2. **开启雾气** — 勾选"启用雾气"选项
3. **调整雾气颜色** — 点击颜色框，选择和你的地图主题相符的颜色
   - 森林地图：试试浅绿色或灰蓝色
   - 沙漠地图：试试土黄色或橙色
   - 夜晚地图：试试深蓝色或紫色
4. **调节雾气密度** — 通过滑块控制雾气的"浓淡"，数值越大，远处的东西越模糊
5. **设置雾气高度** — 如果你想让雾气只出现在地面附近，可以调整"起始高度"和"终止高度"

> **⚠️ 常见错误**：很多新手把雾气密度调得太高，结果近处的东西也看不清了！记住，雾气的主要作用是让**远处**有层次感，**近处**应该保持清晰。一般密度设置在0.3~0.5之间就够了。

### 装饰物与场景细节

地图光有地形还不够，就像拍电影需要道具一样，你需要往地图上放**装饰物**（Doodads）来丰富画面。

1. **打开装饰物面板** — 在编辑器左侧找到"装饰物"标签页[^2]
2. **选择合适的分类** — 装饰物按类型分成树木、岩石、废墟、灯光等，先想好你需要什么
3. **在地图上放置** — 点击分类后，在右侧列表里选中想要的装饰物，然后在地图上点击放置
4. **调整装饰物大小和旋转** — 选中已放置的装饰物，按住**Ctrl**键拖动可以旋转，按住**Ctrl+Shift**可以缩放
5. **善用组合功能** — 如果你想反复使用一组装饰物组合（比如一个"篝火+木桶"的小场景），可以把它们全部选中，右键点击"作为装饰物保存"[^1]

> **💡 新手提示**：装饰物不是越多越好！新手最常犯的错误是往地图上塞满各种东西，结果看起来杂乱无章。建议每个区域有1~2个"主角装饰物"（比如一棵大树或一座石碑），再配几个小装饰物点缀就够了。

### 小结

完成以上三个部分后，你应该能看到你的地图有了明显的变化：
- 山坡、山崖的高低起伏让地形有了立体感
- 远处的雾气让地图产生了"景深"效果
- 精心布置的装饰物让画面不再空旷

现在就打开编辑器，选一块你正在做的地图，试着把今天学的三个技巧都用

## 天气效果增强场景感

天气效果是让地图"活起来"的重要工具。合适的雨雪雾效不仅能美化画面，还能配合剧情营造不同的情绪氛围。本节你将学会在地图中添加天气效果，并让它与灯光效果协调搭配。

### 操作步骤

1. **打开场景设置面板** — 在顶部菜单栏点击「场景」→「场景控制」（或直接按快捷键 F4）。[^3] 这里包含了地图的所有全局设置选项。

2. **切换到天气标签页** — 在场景控制面板中找到「环境」选项卡，然后选择「天气」子选项。你会看到一个下拉菜单，里面列出了所有可用的天气类型。

3. **选择天气类型** — 常见的天气效果包括：
   - **雨（Rain）** — 适合雨天战斗场景或悲伤剧情
   - **雪（Snow）** — 适合寒冷区域或节日氛围
   - **雾（ Fog）** — 适合神秘森林或恐怖地图
   - **无（None）** — 清除所有天气效果
   
   点击选中你想要的效果，地图会立即预览显示。

4. **调整天气密度和强度** — 选中天气后，下方会出现调节滑块。你可以调整「密度」控制天气粒子的数量，调整「速度」控制天气移动的快慢。根据实际效果反复调试，直到看起来自然。

5. **设置天气持续时间**（可选）— 如果你希望天气随时间变化，可以点击「随机」选项，系统会自动切换不同的天气状态。

6. **与灯光协调** — 返回「环境」面板的「灯光」标签页。[^1] 记住：雨天建议用偏冷的蓝色调灯光，雪天用明亮的白色或淡黄光，雾天则适合昏暗的散射光。灯光颜色和天气效果统一，场景才会真实。

> **💡 新手提示**：不要在同一张地图中使用太多类型的天气！新手常见错误是频繁切换各种天气，反而让玩家感到混乱。建议整张地图使用1-2种主天气，突出核心氛围。

> **⚠️ 常见错误**：有些玩家添加了雾效后发现什么都看不清了，这是因为雾的密度调得过高。解决方法是将密度降低到30%以下，或者减少雾的颜色不透明度。

### 小结

完成以上步骤后，你应该已经在地图中成功添加了天气效果，并且了解了如何通过调整灯光颜色来配合天气。如果你切换了不同的天气类型，地图会实时更新预览——多尝试几种组合，找到最适合你地图风格的搭配吧！

## 实践：创建一个有氛围的场景

本节将手把手教你从零开始，在地图编辑器中创建一个有场景感的区域。学完后，你将能制作出让人一眼就想进入探索的地图场景。

### 操作步骤

1. **第一步：确定地图主题** — 在动手之前，先问自己：这个地图想表达什么？是阴森的古堡废墟，还是阳光明媚的草原？打开地图后，按 `F4` 打开场景管理器，根据你的地图类型选择对应的主题风格[^1]。主题决定了后续所有视觉元素的统一方向。

2. **第二步：设置基础灯光** — 点击顶部菜单 `场景` → `设置地形的灯光`。在弹出的窗口中，你会看到"环境"、"高光"、"阴影"三个颜色选项[^2]。想营造神秘感？试试把环境光调成深蓝色；想要温暖氛围？橙黄色是不错的选择。别忘了调整"白天时长"，长夜会让人感到压抑危险。

3. **第三步：放置装饰物营造氛围** — 按 `F9` 打开对象编辑器，选择"装饰物"面板。不同主题有不同的装饰物分类：森林用树木和蘑菇，废墟用破石和枯木[^1]。选中装饰物后，在地形上点击放置，然后用 `R` 键旋转角度，`Delete` 删除放错的物件。

4. **第四步：添加大气效果** — 回到地形编辑界面，按 `F10` 打开高级地形选项，找到"雾效"设置[^2]。启用迷雾后，选择迷雾颜色——蓝色迷雾给人寒冷感，灰绿色则显得荒凉。迷雾浓度越高，远处景物越模糊，空间感越强。

> **💡 新手提示**：装饰物不要堆得太满！留白也是一种美。想象真实世界，再茂密的森林也有空地。再摆放前，可以先按 `Shift + D` 切换到俯视全图模式，规划整体布局。

> **⚠️ 常见错误**：新手容易把所有装饰物都放在地面上，忽略了"垂直感"。试试把一些小型装饰物（如花朵、草丛）放在斜坡或悬崖边缘，这样能增强地形的立体感。

### 小结

完成以上四步后，你的地图应该已经有了一个初步的场景感——从统一的灯光色调，到合理的装饰物分布，再到恰到好处的迷雾效果。你可以通过 `F5` 预览地图，观察在不同时间段（白天/黄昏/夜晚）下的氛围变化，这些细节决定了玩家是否能"沉浸"进你的世界。下一步，建议你尝试在场景中加入"触发器"（一种自动播放音效或特效的脚本）来进一步增强氛围感！

## 常见问题与解决方案

本节将帮你解决地图中**灯光过暗或过亮**以及**氛围不协调**这两个最常见的问题。学完以后，你就能让地图的视觉效果变得舒适自然，不再出现"黑漆漆一片"或"刺眼难耐"的情况。

### 灯光过暗或过亮

1. **第一步：打开环境设置面板** — 在WE界面菜单栏中点击"场景"→"环境"（或者直接按快捷键F9）[^3]。这里控制着整张地图的基础光照设置。

2. **第二步：调整基础光照强度** — 在环境面板中找到"白天亮度"和"夜晚亮度"滑块。如果你的地图偏暗，就把白天的亮度调高（比如从默认的0.7调到0.85）；如果是夜晚地图太黑，适当提高夜晚亮度[^1]。

3. **第三步：添加局部补光** — 地图上某些区域（如建筑内部、洞穴）可能特别暗。在这些地方放置"灯光"类型的装饰物（doodads），它们可以提供局部光源[^2]。

4. **第四步：预览验证效果** — 随时按F9或F10切换视角查看实际运行效果，因为编辑器里的显示和实际游戏中可能略有差异。

> **⚠️ 常见错误**：新手容易把亮度调到最大值（1.0），导致整个地图看起来"泛白"没有层次。正确做法是**微调**，每次增加0.05左右，直到看着舒服为止。

> **💡 新手提示**：如果你不确定什么亮度合适，可以先参考暴雪原版地图的数值。比如"失落之神庙"这张地图的夜晚亮度约为0.4左右。

### 氛围不协调的调试方法

1. **第一步：确定地图主题风格** — 先问自己："这张地图是阴森恐怖的地下城，还是阳光明媚的草原？"不同风格需要完全不同的配色方案[^1]。在环境面板中，"雾化颜色"和"环境色"要与主题匹配。

2. **第二步：统一灯光色温** — 在环境设置中，"白天色"和"夜晚色"决定了光的冷暖倾向。暖黄色调给人温暖感，冷蓝色调则显得阴郁神秘。**整个地图尽量使用同一色系**，不要东边暖西边冷。

3. **第三步：检查装饰物的光照冲突** — 有些装饰物模型自带光源，如果它们和你的主灯光色差太大，就会显得格格不入。解决方法是把这些自带光源的装饰物替换成不带光源的同类物品[^2]。

4. **第四步：逐步微调** — 氛围调整没有标准答案，需要你反复尝试。建议每次只改动一个参数（如只改雾颜色），保存后进游戏预览，感觉不对再回来调整。

> **💡 新手提示**：善用"地图→测试地图"功能（快捷键F10）来实时预览效果，比反复进出编辑器要快得多。

> **⚠️ 常见错误**：新手常常只关注"好不好看"，而忽略了**可读性**——玩家需要能看清地面上的路径、陷阱和道具。调试完氛围后，务必确认关键元素仍然清晰可见。

### 小结

完成以上步骤后，你应该能够：
- ✅ 将地图亮度调整到既不过暗也不过亮的舒适范围
- ✅ 让整体氛围风格统一协调，符合地图主题
- ✅ 在保持美观的同时，确保玩家能看清重要的游戏元素

如果调整后仍有局部区域不理想，记得用**局部补光**或**替换装饰物**的方式定点优化。灯光和氛围的调试需要耐心，多试几次就能找到最佳平衡点！
