# 地形纹理与贴图技巧

## 概述

地形纹理是决定地图外观的基础元素，它能让一片平淡的绿色"海洋"变成郁郁葱葱的森林、荒芜的沙漠或冰封的冻土。在魔兽争霸3地图编辑器中，**地形纹理（Terrain Texture）** 决定了地面看起来像什么——草地、泥土、岩石、雪地、水面等。本节将教你如何为你的地图选择和涂抹纹理，让你的地图从"毛坯房"变成有灵魂的虚拟世界。

### 操作步骤

1. **打开地形纹理面板** — 在世界编辑器（World Editor）顶部菜单栏中点击"地形（Terrain）"选项卡，然后选择"地形纹理（Terrain Texture）"[^1]。这一步会打开纹理工具栏，你会在左侧看到所有可用的地形纹理列表。

2. **了解纹理列表** — 列表中显示了当前地图可用的所有纹理缩略图。默认地图通常自带草地、荒地、岩石等基础纹理。你可以用鼠标左键点击选中一个纹理，选中后该纹理会高亮显示[^2]。

3. **使用画笔涂抹地形** — 选中纹理后，在右侧地图区域按住鼠标左键拖动即可涂抹该纹理。按住鼠标的时间越长，纹理覆盖面积越大。你可以通过调整工具栏下方的"画笔大小（Brush Size）"来控制每次涂抹的范围[^3]。

> **💡 新手提示**：在涂抹大面积区域时，建议先选择较大的画笔尺寸快速覆盖，再用小画笔修补边缘细节，这样效率更高。

> **⚠️ 常见错误**：新手容易忽略的是——地形纹理不是"画"上去的装饰，而是实际改变了地形的物理属性。比如用"深水"纹理涂抹的区域单位无法通行，用"悬崖"纹理会改变地形高度。涂抹前要明确你想要的地形效果。

### 小结

完成以上步骤后，你应该能够打开地形纹理面板、选择不同的纹理类型，并用画笔在地图上涂抹各种地形。掌握这些基础操作后，你的地图就不再是单调的绿色平面，而是开始有了地形的变化和视觉层次。建议你打开世界编辑器，新建一张空地图，实际操作练习一下这些步骤！
