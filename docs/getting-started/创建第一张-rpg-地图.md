# 创建第一张 RPG 地图

## 第一步：下载并启动World Editor（世界编辑器）

在本节中，你将学习如何找到并启动魔兽争霸III的世界编辑器（World Editor），这是我们制作地图的核心工具。学完本节后，你就能成功打开编辑器，看到那个由多个面板组成的创作界面啦！

### 操作步骤

1. **确认游戏版本** — 首先，你需要确保电脑安装的是**《魔兽争霸III：冰封王座》（The Frozen Throne，简称TFT）**，而不是原版的《混乱之治》。这是因为TFT版本包含了我们制作RPG地图所需的所有功能[^2]。你可以在战网客户端或游戏启动器中查看版本信息。

2. **找到World Editor入口** — 安装好游戏后，World Editor（世界编辑器）通常不会在桌面创建快捷方式。你需要进入游戏安装目录，找到`World Editor.exe`或`WorldEdit.exe`文件并双击打开[^4]。常见的路径是：`C:\Program Files\Warcraft III\_worldeditoool_Final`_（具体文件夹名可能因版本而异）。

3. **首次启动等待加载** — 双击后，程序可能需要几秒钟加载。第一次启动时，你会看到一个包含多个窗口和面板的复杂界面，别担心，这是正常的[^7]。

4. **认识主界面四大模块** — 打开后，你会看到编辑器的核心组成部分：[^2]
   - **地形编辑器（Terrain Editor）** — 用来绘制地图、放置树木和装饰物
   - **触发编辑器（Trigger Editor）** — 用来编写游戏逻辑，比如玩家触发事件
   - **物体管理器（Object Manager）** — 用来修改单位、技能和物品的属性
   - **音效编辑器（Sound Editor）** — 用来添加背景音乐和音效

> **💡 新手提示**：如果你的编辑器界面是英文的，不用担心！很多老玩家都是用英文版工作的。你可以在网上找到中文补丁，但英文界面其实不影响地图制作，因为重要的按钮和菜单都很直观。

> **⚠️ 常见错误**：很多新手会下载错误版本的编辑器，比如只安装了原版《混乱之治》就想打开World Editor。如果你在游戏目录里找不到World Editor.exe，那很可能就是版本不对，请重新下载TFT版本。

### 小结

完成以上步骤后，你应该能看到一个包含多个面板的编辑器界面，左边是地形工具栏，右边是物体列表。你可能会觉得界面有点复杂，但别担心——接下来的教程会一步步带你熟悉每个工具。现在，你可以先随意点击看看各个按钮，感受一下界面布局，这绝对是个好习惯！

## 第二步：创建新地图与基础地形

本节将带你完成创建地图的第一步工作——设置地图尺寸、绘制基础地形、放置玩家出生点。学完本节后，你将拥有一张可以进入测试的空旷世界，虽然还没有任何玩法，但这是所有RPG地图的起点。

### 选择合适的地图大小

1. **打开World Editor** — 双击桌面上的"World Editor"图标（或在魔兽争霸Ⅲ安装目录中找到）[^4]
2. **点击"新建地图"** — 启动后软件会自动弹出新建对话框，如果没有，手动点击左上角菜单栏的 **文件 → 新建**[^7]
3. **设置地图尺寸** — 在弹出的"地图大小"选项中，拖动滑块选择地图宽度和高度[^7]
   - **推荐尺寸**：对于第一张RPG地图，建议选择 **32×32** 或 **48×48**，这样既能容纳足够的场景，又不会让编辑器运行卡顿
   - 尺寸越大，编辑时消耗的电脑资源越多，新手容易遇到卡顿问题
4. **确认创建** — 点击"确定"后，你会看到一片灰白色的空白画布，这就是你的地图画布[^5]

> **⚠️ 常见错误**：很多新手一上来就设置 **128×128** 或更大的地图，觉得越大越厉害。实际上大地图会让编辑器变得非常卡顿，而且新手很难把大地图填满内容。建议从中小尺寸开始，等熟练后再尝试大地图。

### 使用地形刷绘制地面

1. **切换到地形编辑器** — 在编辑器左侧的标签栏中，点击 **"地形"（Terrain）** 图标，即可进入地形编辑模式[^2]
2. **选择地面类型** — 在左侧面板的"地形类型"列表中，选择你想绘制的地面[^8]：
   - **草地**（Grass）— 最常用的基础地面
   - **泥土**（Dirt）— 用于道路或荒野区域
   - **水面**（Water）— 湖泊、河流
   - **岩石/山地**（Rock/Cliff）— 制作高低落差
3. **选择地形刷** — 在工具栏中选择刷子大小[^8]：
   - **大刷子** — 快速填充大面积区域
   - **小刷子** — 精细雕刻边缘
4. **开始绘制** — 按住鼠标左键在地图上拖动，地面类型会随着你的绘制而改变[^5]
5. **使用地形变形工具** — 如果需要制作山坡，点击工具栏中的 **"升高/降低地形"** 按钮，然后拖动鼠标制造高低起伏[^8]

> **💡 新手提示**：绘制地形时不必追求一步到位。建议先铺一个大致的框架（比如哪里是草地、哪里是水面），之后再慢慢精细化调整。地形编辑器就像"数字画板"，可以反复修改。

### 放置出生点（玩家起始位置）

1. **打开单位放置工具** — 在地形编辑器右侧，找到 **"单位"（Units）** 面板[^2]
2. **筛选可放置单位** — 在搜索框中输入"起始位置"或直接滚动列表找到 **"玩家起始位置"（Player - Start Location）**[^1]
3. **放置出生点** — 点击该单位，然后在地图上点击你想放置的位置[^1]
   - 通常放置在地图中央偏下的位置，方便玩家向外探索
   - 每个玩家（红色、蓝色、绿色等）都需要各自的起始位置
4. **设置玩家颜色** — 选中放置的起始位置，在左侧属性面板中确认其所属玩家（默认是玩家1，红色）[^1]

> **💡 新手提示**：如果你只做单人RPG，可以只放置一个玩家起始位置。如果想做多人游戏，需要为每个玩家分别放置对应颜色的起始点。别忘了在 **场景属性（Scenario Properties）** 中设置玩家数量！

### 小结

完成以上步骤后，你的地图应该已经具备了：
- ✅ 确定尺寸的地图画布
- ✅ 基本的地形（草地、泥土、水面等）
- ✅ 至少一个玩家起始位置

现在你的地图已经"有地有边界"了！虽然看起来还很简单，但地基已经打好。下一节我们将学习如何添加NPC、敌人和基本任务，让这张空白地图开始变得有趣起来。

## 第三步：添加游戏单位与野怪

本节我们将学习如何使用**物体管理器**在地图上放置玩家单位和野怪，并设置野怪的属性与掉落物品。学完本节后，你的地图将不再是一片空地，而是有"活物"的虚拟世界了！

### 使用物体管理器放置单位

物体管理器是魔兽争霸III编辑器最核心的工具之一，它就像一个"魔法盒子"，可以让你把各种单位、建筑、物品放进地图里[^2]。

#### 操作步骤

1. **打开物体管理器** — 在WE界面顶部菜单栏点击"模块(Modules)" → 选择"物体管理器(Object Manager)"，或者直接按键盘上的 **F6** 键[^7]。管理器窗口会出现在屏幕右侧。

2. **选择要放置的单位类型** — 在管理器左侧的列表中，找到"单位(Units)"选项并展开。你可以按类别浏览：人族、兽族、亡灵、精灵等，也可以使用顶部的搜索框直接输入单位名称，比如"步兵(Footman)"或"弓箭手(Archer)"。

3. **在地图上放置单位** — 找到想要的单位后，直接拖动到地图上，或者先点击选中，再在地图上点击放置。放置后你会看到一个单位图标出现在地图上。

4. **设置单位所属队伍** — 选中已放置的单位，在右侧属性面板中找到"所有者(Owner)"选项[^1]。你想让玩家控制的单位，选择具体玩家编号（玩家1-12）；想让中立怪物攻击玩家，选择"中立敌对"；纯粹装饰用的单位，选择"中立被动"。

> **💡 新手提示**：放置单位时注意"高度"问题！如果你把单位放在悬崖边缘，可能会掉下去或无法攻击到其他单位。养成习惯：先在低地放置一个测试单位试试。

### 设置野怪的属性与掉落物品

野怪是RPG地图的灵魂！好的野怪设计能让玩家感受到挑战与奖励的乐趣[^3]。

#### 操作步骤

1. **选中地图上的野怪单位** — 点击你刚才放置的中立敌对单位，确保它处于选中状态（四周有蓝色边框）。

2. **调整野怪基础属性** — 在属性面板中找到以下常用设置：
   - **生命值(Life)**：野怪有多少血，建议根据地图等级设计调整
   - **攻击伤害(Attack Damage)**：野怪每次攻击造成多少伤害
   - **护甲(Armor)**：野怪的防御力
   - **攻击间隔(Attack Speed)**：野怪多久攻击一次

3. **设置掉落物品** — 在属性面板中找到"掉宝表(Treasure Table)"选项，点击右侧的编辑按钮。弹出窗口中，你可以添加多个掉落条目：
   - **物品类型**：选择要掉落的物品（如装备、药水、金币等）
   - **掉落几率**：输入1-100的数字，100代表100%掉落
   - **掉落数量**：每次掉落几个

> **⚠️ 常见错误**：新手最容易犯的错误是把野怪的攻击力设置得太高！建议初期野怪的伤害不要超过玩家生命值的20%，否则玩家还没来得及反应就被秒杀了。先用较低的数值测试，再慢慢调整。

### 小结

完成以上步骤后，你应该能看到：

- ✅ 地图上有代表玩家初始基地的单位（如城镇大厅）
- ✅ 野外有会主动攻击玩家的野怪
- ✅ 击败野怪后会有物品掉落

现在你的地图已经有"敌人"可以打了！下一步我们将学习如何给玩家设置初始资源和胜利条件。

## 第四步：学习触发器基础（实现RPG逻辑）

触发器（Trigger）是让地图"活起来"的核心工具。简单来说，触发器就是一段"如果...那么..."的逻辑规则。比如"如果敌人死亡，就给玩家经验值"——这就是一个完整的触发器逻辑。RPG游戏中的人物升级、任务系统、怪物掉落等功能，都离不开触发器。学会基础触发器后，你就能让地图拥有真正的游戏玩法了。

### 操作步骤

1. **打开触发编辑器** — 在世界编辑器顶部菜单栏点击"触发编辑器"按钮[^2]。如果你之前没有编辑过触发器，编辑器会显示一个空白列表。触发编辑器由左侧的触发器列表和右侧的代码编辑区组成。

2. **理解触发器的基本结构** — 每个触发器都由三个部分组成：
   - **事件（Event）**：触发的条件，比如"单位死亡"
   - **条件（Condition）**：额外的判断，比如"死亡的单位是怪物"
   - **动作（Action）**：满足条件后执行的操作，比如"给玩家加经验值"
   
   这个结构叫做"ECA架构"，记住它就能编写任何触发器。

3. **创建第一个触发器：单位死亡获得经验** — 在触发编辑器左侧空白处**右键点击**，选择"新建触发器"，命名为"怪物死亡加经验"。双击打开后，按照以下顺序添加：
   - 点击"新建事件"→ 选择"单位-单位死亡"[^2]
   - 点击"新建条件"→ 选择"单位-单位类型比较"，设置为"死亡单位类型等于 野猪"
   - 点击"新建动作"→ 选择"玩家组-添加经验"，设置为"击杀玩家获得 50 经验"

4. **测试触发器** — 按F6键或点击"测试地图"按钮[^4]。在游戏中放置一只野猪，让你的英雄攻击它。当野猪死亡时，你应该看到经验值增加的提示。

> **💡 新手提示**：触发器的执行顺序很重要！如果把"添加经验"放在"条件"之前，即使条件不满足也会加经验。养成先加事件、再加条件、最后加动作的习惯。

> **⚠️ 常见错误**：很多新手创建触发器后忘记启用它。检查触发器列表左侧是否有灯泡图标——灰色灯泡表示关闭，黄色灯泡表示开启。如果不亮，右键点击触发器选择"启用"。

### 小结

完成以上步骤后，你已经掌握了触发器的基本工作原理和"事件-条件-动作"结构，并且创建了第一个能实际运行的触发器。现在你可以尝试修改条件中的"野猪"换成其他单位，或者调整经验值数值来观察效果。建议再创建一个"玩家进入区域触发"的简单触发器，进一步熟悉触发编辑器的操作界面。

## 第五步：测试与调试地图

本节我们将学习如何运行自己创建的地图，以及如何将地图保存为可分享的 `.w3x` 文件。学完本节后，你就能亲眼看到自己搭建的地图世界，并把它分享给朋友测试了！

### 使用测试按钮运行地图

1. **找到测试按钮** — 在 World Editor 顶部工具栏的右侧，有一个绿色的三角形播放图标（▶），这就是"测试地图"按钮[^4]。如果找不到，注意看工具栏上有没有类似媒体播放器播放键的图标。

2. **点击测试按钮** — 点击这个绿色三角形按钮[^7]，World Editor 会自动启动 Warcraft III 游戏，并加载你当前编辑的地图。这就像按下了游戏机的电源开关！

3. **体验你的地图** — 游戏启动后，你可以用默认的玩家1（红色）身份控制单位，在地图上移动、探索你创建的地形[^4]。这时你可以检查：地形是否好看？单位是否在正确位置？地图大小是否合适？

4. **退出测试返回编辑器** — 按键盘上的 `Esc` 键或点击游戏右上角的 ✕ 关闭游戏窗口，即可回到 World Editor 继续修改。

> **💡 新手提示**：首次测试时，World Editor 可能会弹出"是否保存地图"的提示，建议先保存一次再测试，避免辛苦编辑的内容丢失。

> **⚠️ 常见错误**：点击测试按钮后游戏没反应？常见原因是地图还没有保存，或者地图存在严重的触发器错误导致无法加载。解决方法：先按 `Ctrl + S` 保存地图，然后再测试。

### 保存并导出地图为 .w3x 文件

1. **首次保存地图** — 按键盘快捷键 `Ctrl + S`，或者点击顶部菜单栏的"文件 → 保存"。如果是第一次保存，系统会弹出对话框让你输入地图名称和选择保存位置[^3]。

2. **导出为 .w3x 格式** — 点击"文件 → 另存为"，在保存类型中选择"Warcraft III 地图 (*.w3x)"[^3]。`.w3x` 是魔兽争霸3地图的标准格式，只有这个格式才能被游戏识别和加载。

3. **选择保存位置** — 建议在桌面或你喜欢的位置新建一个文件夹，专门存放你的地图文件。地图文件命名建议用英文或数字，避免使用中文名称导致兼容性问题。

4. **确认保存成功** — 保存完成后，找到你保存的 `.w3x` 文件，双击它应该能直接用 Warcraft III 打开测试。

> **💡 新手提示**：每次重要修改后，记得按 `Ctrl + S` 保存！World Editor 没有自动保存功能，如果突然崩溃或断电，未保存的内容会全部丢失。

### 小结

完成以上步骤后，你应该已经：
- ✅ 成功运行了自己的第一张地图
- ✅ 学会了快速保存（Ctrl+S）和导出为 .w3x 文件
- ✅ 了解了测试—修改—再测试的地图开发循环

现在你可以继续完善地图，添加更多地形、单位或触发器，然后重复测试流程，让你的地图越来越丰富！

## 第六步：提升RPG体验的进阶概念

在前几步中，你已经学会了创建地形、放置单位和设置简单的触发器。现在我们来学习两个让RPG地图真正"好玩"起来的核心系统——英雄系统与物品商店系统。学完本节后，你的地图将不再是"走路看风景"，而是真正有成长感、有挑战的RPG体验。

### 什么是英雄单位与技能系统

在RPG地图中，"英雄"是玩家控制的特殊单位，它们比普通单位更强，并且可以升级、学习技能[^3]。

#### 操作步骤

1. **创建英雄单位** — 打开"单位编辑器"（在WE顶部菜单的"模块"中找到），点击左侧列表选择一个你想要作为英雄的单位，比如"剑圣"或"山丘之王"[^2]
2. **设置英雄属性** — 选中该单位后，找到右侧属性面板中的"英雄"选项卡，勾选"是英雄"复选框，并为它设置初始力量、敏捷、智力等属性值
3. **添加技能给英雄** — 切换到"技能"标签页，点击"添加技能"按钮，从技能列表中选择你想让英雄学会的技能（如"风暴之锤"或"治疗"）
4. **配置技能等级** — 为每个技能设置最高等级（通常设为3级），这样玩家升级后可以提升技能效果
5. **放置英雄到地图** — 切换回地形编辑器，在玩家出生点附近放置你创建好的英雄单位，并将其"所有者"设置为你希望玩家控制的颜色[^1]

> **💡 新手提示**：英雄单位最好放置在地图中央偏下的位置，这样玩家开始游戏时能第一时间看到并选中它。别忘了给英雄单位起一个有意义的名字，这样玩家更容易记住自己的角色。

> **⚠️ 常见错误**：新手经常忘记给英雄设置"是英雄"属性，导致玩家无法通过击杀获得经验值升级。解决方法：回到单位编辑器，反复确认该选项已勾选。

### 物品系统与商店配置

物品系统是RPG地图的"甜味剂"——玩家通过购买或获得物品来提升角色实力，让战斗更有策略感[^3]。

#### 操作步骤

1. **创建商店单位** — 在单位编辑器中找到一个适合做商店的单位，比如"神秘宝藏"或"人类商店"，将其添加到你的地图中
2. **放置商店到地图** — 切换到地形编辑器，将商店单位放置在玩家经常经过的区域（如主城门口或城镇中心），并设置其"视野范围"确保玩家能看到它
3. **配置商店物品列表** — 打开"触发编辑器"，创建一个新触发器命名为"初始化商店物品"，添加事件"地图初始化"，然后使用"单位-添加出售物品"动作，为商店添加你想出售的物品（如"生命药水"、"力量手套"等）
4. **设置物品价格与属性** — 在"物品编辑器"中为每件物品设置价格、冷却时间和使用效果
5. **给玩家初始金钱** — 创建一个触发器"给予玩家初始金币"，设置事件为"地图初始化"，动作为"玩家-设置当前黄金"为你希望的数量（如500金）

> **💡 新手提示**：商店的物品不要一下子全放出来！建议按照游戏进度分批解锁。比如第一级商店只卖基础药水，中期才解锁高级装备，这样玩家才有奋斗目标。

> **⚠️ 常见错误**：新手设置完商店后发现玩家买不了东西，原因是物品没有正确"关联"到商店。请确保触发器中的"添加出售物品"目标单位名称与你放置的商店完全一致，包括大小写。

### 小结

完成以上步骤后，你的RPG地图已经具备了三大核心要素：一个可以升级学技能的英雄、一套完整的物品购买系统，以及让玩家持续探索的目标感。现在你可以运行地图测试，按下F6测试模式，感受一下英雄升级和购物的乐趣——这正是让玩家"上瘾"的关键体验！
