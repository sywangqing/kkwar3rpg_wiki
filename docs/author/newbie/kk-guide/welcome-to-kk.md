# 欢迎来到KK开发者平

> KK对战平台秉持 "无限创的理念，致力于打造一个开放的游戏创作平台，百万玩家自由共创，汇聚海量地图。平台提供完善的游戏创作工具和创作、上架、运营推广、激励扶持等一站式服务，激励游戏制作人创作，助力实现每一个游戏梦想
>

## KK官方对战平台优势

- **内容多样*——[KK官方对战](https://www.kkdzpt.com/)平台拥有庞大的作者群体以及UGC内容
- **快速体*——用户可以在KK平台上启动新地图并在几秒钟内开始游戏
- **社交联系**——KK平台在某种程度上是一个社交网络，玩家可以在上面与朋友开黑或对战。综合游戏配置，包括社区、活动、多人配对、排名、语音聊天功能等。在为玩家受众设计时，请始终牢记如何发挥这些优势
- **收益** ——头部作者月收入超过百万，腰部作者超过十万
