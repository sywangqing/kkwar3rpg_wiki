## KK官方对战平台用户画像

作者发布的地图将展示在KK官方对战平台上供玩家游玩，为了精准触达并持续吸引玩家，以下是对平台玩家群体的深入剖析，旨在帮助内容创作者打造出更具吸引力的游戏作品

### 数据

- 调研活跃用户数据显示，用户年龄分布中5-34岁用户占比较高；
- 城市等级分布中，一线城市、新一线城市占比较高；
- 进一步观察职业分布，平台用户中在职人员超5%，整体有消费能力，另外自由职业者的高达30%
- 讨厌上手门槛和挫折，喜爱易于理解，正反馈迅速的内容

### 玩家触媒偏好

平台的活跃人群更倾向于通过 **微信公众号、抖* 渠道获取官方游戏资讯和社交互动，据调查显示，我们的用户对官方社媒平台的关注度保持稳定，微信公众号和抖音占比近60%。这也从侧面提示我们，在游戏推广和社区建设方面，可以更多地利用这些平台与玩家建立联系，高效利用玩家稳定的关注度对游戏进行有效的宣传与互动
