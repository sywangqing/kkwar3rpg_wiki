## 平台热门游戏类型

- 防守、生存最热，用户数约占平台发行数量的50%+
- 塔防其次，用户数约占比约23%
- 平台玩家以PVE为主，PVP为辅

### 项目选品思路

- 项目是否是平台玩家所喜好的；
- 项目是否是团队成员所擅长且喜欢的类型
- 项目的特点及团队的创意能否通过编辑器发挥出来；
- 项目在平台上是否有同品类的热门地图，自己较于它能否做到一定的创新

### 创新项目思路

- 全新原创玩法，能够不受现有玩法基础的束缚进行新的探索；
- 在KK官方对战平台传统玩法的品类上进行突破和组合；
- 当前KK官方对战平台里比较少的玩法类型；
