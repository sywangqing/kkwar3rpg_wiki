# 技能编辑器与自定义技

## 技能编辑器简介与基础操作

本节将带你认识技能编辑器——这是制作RPG地图最重要的工具之一。学完本节，你就能独立打开技能编辑器，看懂界面上的各种设置，为后续创建自定义技能打下基础

### 什么是对象编辑

在正式介绍技能编辑器之前，我们先了解一*对象编辑*（Object Editor）

对象编辑器是World Editor中用于修改游戏数据的核心工具。你可以把它想象成一个巨大的“数据库”，里面存放着单位、物品、升级、技能等各类游戏元素的所有设定[^4]。每当你需要调整某个单位有多少生命值、某个技能造成多少伤害时，都要在对象编辑器里修改对应的数值

技能编辑器并不是一个独立的窗口，而是**对象编辑器中的一个标签页**[^5]。换句话说，对象编辑器就像一栋大楼，技能编辑器是其中的一层。在后续的学习中，你还会接触到“单位编辑器”“物品编辑器”等其他标签页，它们都遵循相同的操作逻辑

> **💡 新手提示**：理解“对象编辑器”和“技能编辑器”的关系很重要——所有编辑器操作都在同一个窗口内进行，只是切换不同的标签页而已

### 如何打开技能编辑器

下面我们一步步打开技能编辑器

1. **启动World Editor** 双击桌面或开始菜单中的“World Editor”图标，打开编辑器主界面[^5]
2. **进入对象编辑* 在顶部菜单栏找到*编辑**”菜单，点击后选择*对象编辑*
3. **切换到技能标* 在对象编辑器窗口中，找到并点击*技*”标签（通常显示为一个图标）

完成以上三步后，你就成功进入了技能编辑器界面

> **⚠️ 常见错误**：新手常常找不到“技能”标签的位置。请注意观察对象编辑器顶部的标签栏——技能标签上有一个类似于魔法棒的图标，点击它就能切换到技能编辑模式

### 编辑器界面布局介绍

技能编辑器的界面分为左右两个主要区域[^2]

- **左侧：技能列* 这里显示所有可用的技能。你可以看到暴雪自带的预设技能（如“闪电链”“治疗”等）。点击列表中的任意技能，右侧就会显示该技能的详细设置
- **右侧：技能属性面* 当你选中某个技能后，这里会显示该技能的所有参数，比如伤害值、冷却时间、魔法消耗、施法范围等。你可以直接修改这些数值来改变技能效果

此外，界面上方还*搜索**分类筛*功能。当你创建了很多自定义技能后，利用这些工具可以快速找到目标技能

> **💡 新手提示**：属性面板中的参数非常多，全部记住不现实。建议先关注几个常用的参数—*伤害**（Damage）*冷却时间**（Cooldown）*魔法消*（Mana Cost）*施法范围

## 技能的基础属性与参数

本节我们将学习如何在对象编辑器中设置技能的基础属性，包括技能名称、图标、魔法消耗、冷却时间以及施放方式。完成本节学习后，你将能够创建一个拥有完整基础属性的自定义技能，并理解这些参数如何影响游戏中的使用体验

### 技能名称与图标设置

技能名称和图标是玩家第一眼看到的内容，清晰的命名和直观的图标能帮助玩家快速理解技能用途

1. **第一步：打开对象编辑* Warcraft III 世界编辑器菜单栏中，点击“模块”然后选择“对象编辑器”。在左侧列表中找到并点击“技能”选项[^1]
2. **第二步：创建新技能或复制现有技* 点击“新建技能”按钮创建空白技能，或右键点击一个现有技能选择“复制”，然后粘贴为新技能（推荐新手使用复制方式，这样能保留基础参数结构）[^2]
3. **第三步：设置技能名* 在技能属性面板中找到“名称”字段，点击后在右侧输入你想要的技能名称。建议使用简洁明了的名称，如“火焰冲击”或“治疗术
4. **第四步：设置技能图* 找到“编辑器图标”或“图标”字段，点击右侧的省略号按钮打开资源选择器。在弹出的窗口中浏览已导入的图标，选中后点击“确定”[^2]

> **💡 新手提示**：为技能选择一个与功能相关的图标非常重要。例如治疗类技能使用绿色十字图标，伤害类技能使用红色火焰图标，这样玩家即使不看文字描述也能猜到技能效果

### 消耗属性：魔法值与冷却

魔法消耗和冷却时间是平衡技能强度的关键参数。合理的设置能让技能有使用价值但不至于过于强大

1. **第一步：设置魔法消* 在技能属性中找到“魔法消耗”或类似字段（如“数- 魔法消耗”），输入每次使用技能需要消耗的魔法值。如果设，则技能不消耗魔法[^1]
2. **第二步：设置冷却时间** 找到“冷却时间”或“数- 冷却时间”字段，输入以秒为单位的冷却时长。例如输.0表示使用后需要等秒才能再次使用[^1]
3. **第三步：调整消耗与冷却的平* 这是新手最难掌握的部分。一般原则是：效果越强大的技能，消耗和冷却应该越高。建议先设置较低的消耗和冷却进行测试，再根据实际效果逐步调整

> **⚠️ 常见错误**：新手容易把强力技能的冷却时间设为0，这样会导致该技能可以无限连续使用，破坏游戏平衡。正确的做法是给高伤害或高治疗技能设置至-5秒的冷却时间

### 技能施放方式与目标类型

技能的施放方式决定了玩家如何激活技能，目标类型则决定了技能可以作用于什么对象

1. **第一步：设置施放方式** 在技能属性中找到“施放方式”或“数- 施放方式”字段。常见选项包括
   - **主动施放**：玩家手动点击使
   - **被动**：技能自动生效，无需玩家操作
   - **触发施放**：通过触发器（一种脚本系统）控制何时生效[^3]
2. **第二步：设置目标类型** 找到“目标类型”或“数- 目标类型”字段。常见选项包括
   - **无目*：如范围性魔法，不需要选择目标
   - **单位目标**：必须点击指定单
   - **点目*：施放到指定位置
   - **自己**：只能对自己使用[^3]
3. **第三步：配置施法距离和范* 根据目标类型设置“施法距离”（最远能使用的距离）和“影响范围”（技能生效的区域大小）。这些参数在“数据”面板中可以找到

> **💡 新手提示**：如果你的技能是远程攻击但玩家使用时总是点不到目标，很可能是施法距离设置得太短了。新手建议将施法距离设置得稍长一些，这样会更易于操作

### 小结

完成以上三个部分的学习后，你应该已经掌握

- 如何在对象编辑器中创建和命名技
- 如何为技能设置图
- 如何配置魔法消耗和冷却时间
- 如何选择合适的施放方式和目标类

现在你可以尝试创建一个简单的自定义技能了。建议从复制一个现有技能开始，修改其名称和图标，然后调整消耗与冷却参数，最后进入游戏测试效果如何。实践是学习的最好方式！

## 技能等级与升级系统

本节将教你如何在技能编辑器中设置技能的等级上限、为每个等级配置不同的数值，以及如何设置技能升级的前置条件。学完本节后，你将能够创建可升级的技能，让英雄或单位随着游戏进程获得更强的技能效果

### 操作步骤

1. **打开对象编辑器并选择技* 在顶部菜单栏点击"高级"对象编辑（或直接F4 快捷键），在左侧分类中找技选项并展开，选中你想要设置等级的技能[^1]

2. **设置技能最大等* 在技能属性面板中找到"等级"（Level）相关字段，最大等（Max Level）设置为你想要的数值（通常设置1-5 级）[^2]。这个数值决定了技能可以被升级几次

3. **为每个等级设置独立参* 展开技能的不同等级标签（Level 1、Level 2、Level 3 等），分别修改每个等级下的数值，比如伤害值、冷却时间、魔法消耗、作用范围等[^3]。例如将1级火焰雨的伤害设00级设00，以此类推

4. **设置升级研究要求** 在技能属性中找到"升级要求"研究要求"字段，指定升级该技能需要的建筑和科技。比如设奥秘殿堂"建筑火焰掌握"科技才能升级级[^4]

> **💡 新手提示**：在设置多级技能时，注意观察每个等级右侧的预览图示，确保图标或特效也随等级变化，这样玩家才能直观看到技能升级了

> **⚠️ 常见错误**：新手经常忘记为每个等级单独修改数值，导致所有等级都使用相同的参数。请务必逐一检查每个等级的具体数值，确保升级确实带来增强效果

### 小结

完成以上步骤后，你的技能应该已经具备了等级系统和升级路径。保存地图后进入游戏测试，验证不同等级技能的实际效果是否符合预期。如果升级条件不满足，请检查建筑是否正确建造、科技是否已研究

## 常用技能类型实

本节将通过三个最常用的技能类型，教会你在对象编辑器中创建完整的自定义技能。学会后，你将能为任何单位添加具有实际效果的特殊能力

### 伤害技能的制作方法

伤害技能是最基础的技能类型，让我们从零开始创建

**操作步骤**

1. **打开对象编辑* 在顶部菜单栏点击"高级"对象编辑[^4]，或者直接按 **F6** 快捷
2. **切换到技能页* 在左侧分类中选择"技标签页，然后点击右上角的"新建技按钮
3. **选择基础模板** 在弹出的技能列表中，找医疗闪电等你想修改的基础技能，点击确定[^5]
4. **设置伤害数* 找到"数据"面板中的"伤害字段，将其改为你想要的数值（000等）
5. **调整施法范围** 数据"中找施法范围"有效范围"，设置技能能作用多远
6. **冷却时间设置** 数据"面板找到"冷却时间"，输入你希望技能冷却的秒数

> **💡 新手提示**：伤害值默认是"近战"伤害，如果你想让远程技能也生效，需要在"攻击类型"字段中选择正确的类型（如魔法攻击）

> **⚠️ 常见错误**：很多新手设置了伤害值但技能不生效，这是因为没有给单位添加这个技能。记得回单位"页面，选择你的英雄单位，在"技字段中添加你刚创建的技能

### Buff与Debuff技

Buff是增益效果，Debuff是减益效果，它们都通过"效果"系统来实现[^1]

**操作步骤**

1. **创建基础技* 同样在对象编辑器中新建一个技能，选择"返老还邪恶之光"作为模板[^1]
2. **设置持续时间** 数据"中找持续时间"（Buff生效多少秒）冷却时间"（多久能用一次）
3. **设置效果数* 找到"伤害/恢复"护甲加成/减免"等字段，输入每次生效的数
4. **设置生效间隔** 数据"中找缓冲时间间隔"，决定Buff多久生效一
5. **创建Buff图标** 常规"中找图标"，双击选择一个合适的图标文件

> **💡 新手提示**：Buff和Debuff的区别主要在效果类型"字段。选择"增益魔法"是Buff，选择"疾病"诅咒"则是Debuff，它们在单位头顶显示的图标颜色不同

### 召唤与幻象技

召唤技能可以创造新单位，幻象技能可以复制现有单位[^2]

**操作步骤**

1. **新建技* 选择"召唤"镜像"作为基础模板
2. **选择被召唤单* 数据"中找创建的单位类，点击右侧按钮选择你想召唤的单位[^2]
3. **设置召唤数量** 数据"中找最大创建数，决定一次能召唤几个
4. **设置持续时间** 找到"持续时间"，输入召唤物的存活时间（表示永久存在
5. **设置魔法消* 数据"中找魔法消，设置施法需要多少魔法

> **⚠️ 常见错误**：召唤出的单位没有攻击或移动，这可能是因为召唤单位本身的AI设置问题。检查一单位编辑中该单位游戏数据"单位分类"，确保不建筑"类型或者有正确武器类型"

### 小结

完成以上三个技能的制作后，你应该掌握了

- 创建带伤害数值的攻击技
- 创建持续恢复/减益的Buff/Debuff
- 创建能召唤单位或幻象的技

建议将这些技能分别添加到你的测试单位上进行实际测试，观察效果是否符合预期

## 技能的美术资源设置

Warcraft III 中，一个技能不仅要有效果，还要有漂亮的"外表"——玩家通过图标了解这是什么技能，通过特效感受技能的威力。本节将教你如何为自定义技能设置图标、施法动画和视觉特效，让你的技能既有实力又有颜值

### 技能图标与头像设置

技能图标（Icon）是玩家在界面中看到的小图标，头像（Portrait）则是单位施法时显示的大头像

1. **打开对象编辑* 在顶部菜单栏点击"高级"对象编辑（或直接F6 快捷键）[^1]
2. **进入技能列* 在左侧分类中选择"技，然后选中你要设置图标的那个技
3. **找到图标字段** 在右侧属性中找到"科技图标"（Art - Icon）字段，点击右侧的放大镜按钮[^2]
4. **选择图标文件** 在弹出的资源选择器中，选择一.blp 格式的图标文件，或者点导入"从你的电脑添加自定义图标
5. **设置施法头像** 同样在属性中找到"施法者背景模（Target - Cast Receiptacle），设置施法时显示的单位头像模型

> **💡 新手提示**Warcraft III 的图标文件格式必须是 .blp（暴雪专用格式）。如果你PNG 图片，可以用 World Editor 自带的导入功能自动转换为 blp 格式

> **⚠️ 常见错误**：很多新手设置完图标后游戏里显示的是空白问号，这是因为图标文件没有被正确"导入"到地图中。正确做法是：文件菜导入管理添加图片文件 使用自定义图选项改为"

### 施法动画与特

施法动画是单位施放技能时的表演动作，正确的动画能让玩家清楚知这个单位正在放大

1. **设置施法动画** 在技能属性中找到"施法动作"（Animation - Cast）字段，输入单位模型的动画名称[^3]
2. **设置施法时间** 找到"施法时间"（Cast Time）字段，设置0 表示瞬发，数值越大施法时间越
3. **添加施法特效** 施法特效"（Art - Effect)字段中输入特效路径，"Abilities\\Spells\\Human\\ThunderClap\\ThunderClapCaster.mdl"[^1]
4. **设置施法时能否移* 移动时可以施设为"关闭"可以防止移动施法

> **💡 新手提示**：每个单位模型的动画名称可能不同。你可以在对象编辑器中查看该单位动画 - 置换动画"字段获取可用动画列表。常用施法动画名包括 "spell channel"spell slam"attack" 等

### 弹道与冲击特

弹道特效是飞行中的技能（如火球、闪电），冲击特效则是命中目标时的爆炸效果

1. **设置弹道模型** 弹道"（Stats - Projectile）相关字段中找到"弹道模型"，设置为你的飞行物模型路径[^1]
2. **调整弹道速度** 设置"弹道速度"数值，数值越大飞行物移动越快
3. **设置影响范围** 找到"影响范围"（Stats - Area of Effect）设置技能的作用半径
4. **添加命中特效** 特效"（Stats - Effect）字段中设置目标受到攻击时的特效路径
5. **设置冲击特效** 冲击特效"（Art - Effect）字段中设置技能到达目标位置时的视觉效

> **⚠️ 常见错误**：新手常把弹道特效当成命中特效。记住：**弹道**是飞行中的视觉效果（如射出的箭）*冲击特效**是到达目标后的爆影响效果（如箭矢命中后的爆炸光环）

### 小结

完成以上设置后，你的技能现在应该拥有：
- 可以在界面中识别的技能图
- 施放时有对应的单位动作和光效
- 远程技能有飞行弹道，本地技能有冲击波纹

现在打开你的地图，选中拥有该技能的单位，进入游戏测试一下视觉效果吧！根据实际效果微调各项数值，直到看起来既酷炫又不会过于花哨影响游戏性能

## 技能测试与调试技

在本节中，你将学习如何在游戏中测试你创建的技能，并掌握排查和解决常见问题的方法。学完本节后，你就能自信地在地图中验证技能是否正常工作

### 在游戏中测试技

1. **第一步：保存并测试地* 在World Editor中按 `Ctrl+S` 保存地图，然后点击编辑器顶部菜单栏的"测试地图"按钮（绿色三角图标）[^1]。这会启Warcraft III 并自动加载你的地图

2. **第二步：进入游戏测试** 使用热键 `F6` 打开"触发编辑，在测试模式下运行你的地图[^4]。如果你还没添加触发器来测试，可以直接在"对象编辑中创建一个测试单位并给它装备你的技能

3. **第三步：观察技能效* 在游戏中选择一个拥有自定义技能的单位，按下技能快捷键（如 `Q`、`W`、`E`、`R`）来释放技能，仔细观察技能是否按预期工作

> **💡 新手提示**：测试前建议先保存一份干净的地图备份，这样如果修改出问题可以随时重来

### 常见问题与解决方

**问题一：技能不显示或无法使*
这通常是因为技能没有正确关联到单位上。在对象编辑器中，选中你的技能，查看"数据 - 科技等级"可用- 所属阵等字段是否正确配置[^1]

**问题二：技能数值不生效**
检查技能的具体数值字段（如伤害、范围、持续时间）是否已经修改。直接在技能列表中双击字段即可修改[^4]

**问题三：游戏崩溃**
如果修改技能后游戏直接崩溃，可能是数值设置过大或过小[^1]。新手常见错误是把伤害值设为负数或超过99999的数值

> **⚠️ 常见错误**：很多新手以为改完技能数值后不用保存就直接测试，这会导致改动丢失。每次修改后务必`Ctrl+S` 保存

### 使用触发器增强技能效

对象编辑器能创建基础的技能效果，但如果想要更复杂的逻辑（比如多段伤害、特殊视觉效果触发），就需要使*触发*（Trigger）[^3]

1. `F6` 打开触发编辑
2. 新建一个触发器，命名为"测试_技能触
3. 添加事件：`单位 - 单位使用技能`
4. 添加条件：`技能被使用等于 [你的技能名称]`
5. 动作"中添加你想实现的额外效果

> **💡 新手提示**：触发器听起来很复杂，但你可以先从简单的动作开始，比如"播放音效"显示提示文字"，逐步增加复杂度

### 小结

完成以上步骤后，你应该已经掌握了在地图中测试技能的基本流程，并能够识别并解决三个最常见的技能问题。如果你的技能需要更高级的功能，现在可以尝试用触发器为它添加额外效果了！
