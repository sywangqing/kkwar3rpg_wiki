# 红包功能

红包KK 开发者平台提供的地图营销工具，目的是KK 官方对战平台的地图进行推广，让更多玩家游玩，从而提升地图曝光和游玩量。推荐开发者在新地图上线或老图大版本更新等需要强化地图曝光的场景使用

## 红包优势

- **更多曝光**：获得更大范围的地图推荐，提升人
- **更多游玩**：获得更多兴趣玩家的游玩和关注，提升用户留存和活
- **即刻开*：创建并支付红包计划后，即可开

## 前提条件

- 必须是认证作
- 上架地图且申请开通了推广权限
- 付费后方可推

## 功能配置说明

1. 登录开发者平台，选择一张地
2. 在地图侧边栏选择「推广」，点击【红包

![选择红包](/kk-guide/202410160123.png)

3. 进入推广计划列表页，点击左侧按钮【创建红包计划

![创建红包计划](/kk-guide/202410160124.png)

4. 在推广配置页，填写完信息后，点击提交

![推广配置页](/kk-guide/202410160125.png)

![提交配置](/kk-guide/202410160126.png)
