# 公会配置

公会系统可以给玩家提供社交互动、团队合作、资源共享和精神归属感，使得玩家在游戏中能够获得更好的体验

## 配置说明

- 首次配置时，地图公会状态为「待更新」；没有配置道具时，道具栏标记为「待配置」，作者可添加道具作为福利
- 添加道具后，点击【立即更新】，道具将同步下发到平台并生
- 完成首次更新后，作者仍可更新道具，最新配置的道具将在**下周一 0 *自动生效

## 道具规则

- 只可添加「审核通过」的时效性商品道
- 默认时效为：当局
- 公会5 个等级，每个等级最多可添加 **4 个道*作为福利
- 默认给等Lv1 的玩家发1 *地图经验加速卡 10%** 作为奖励（不可删除）

## 配置流程

上架地图（游戏大测试大厅）→ 配置公会等级道具 激活公会道

## 功能说明

### 添加道具

道具添加成功后，出现在待生效的道具栏

![添加道具](/kk-guide/202410160115.png)

### 立即更新

立即更新仅在**首次配置**时出现，作者需要手动触发将配置的道具下发给平台。确定更新后将实时生效。更新前请先确认等级道具。（一旦确定提交后，后续更新需下周一才可生效

![立即更新](/kk-guide/202410160116.png)

### 更新道具

完成首次更新后，后续更新的道具将标记为「待生效」，**每周一 0 *会拉取最新的「待生效」道具组。若没有配置，则沿用已生效的道具组

![更新道具](/kk-guide/202410160117.png)

### 删除道具

可删除「待生效」的道具*已生效的则不可删*

![删除道具](/kk-guide/202410160118.png)

### 查看道具

点击查看，查看道具信息

![查看道具](/kk-guide/202410160119.png)
