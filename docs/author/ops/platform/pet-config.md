# 宠物探险

KK 宠物设置的初衷，旨在为玩家提供一个贴心的游戏玩伴，以减轻玩家在游戏中的负担，帮助玩家更轻松地挂机地图，从而有效降低游戏压力。KK 宠物不仅支持 PC 端，还兼容移动端，双端搭配促进回流，提升活跃度

## 功能包括

- **额外热度**：玩家探险的地图会提供额外热
- **商业*：新增地图商品探险特权收益与开发者分成。地图会自动生成 38 月的地图道具，新增地图商品探险特权收益将与开发者进行分成。建议作者设置一些消耗品作为探险奖励
- **数据查看**：提供开发者探险相关数
- **活动运营**：新增地图级探险任务辅助作者促活。建议创建一些宠物相关地图任

## 操作流程

**Step1**：在【推广】中找到【KK宠物】板块，点击【设置奖励】选择商品进行设置

![KK宠物](/kk-guide/202410100127.png)

开发者每次设*最5 个探险奖*，每次玩家探险概率掉落，可以控制每周产出上限。默认奖励为每周首次探险必获1 次（100 积分），开发者可以在此基础上增加奖励吸引玩家来地图上探险
