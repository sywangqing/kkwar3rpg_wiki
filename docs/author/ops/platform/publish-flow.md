# 作品发布流程

作品上传分为三个阶段*自测测试大厅 正式*

在地图制作完成后，大多数作者会选择在自测服进行测试。前期测试可以尽早发现地图中存在的问题（错误或不完善的部分），在测试大厅之前进行修复和改进，确保地图的质量和可玩性

提前上传到测试大厅还能收集玩家的建议和反馈，作者可以关注敏感数据（地图流畅度、玩家参与度等）
