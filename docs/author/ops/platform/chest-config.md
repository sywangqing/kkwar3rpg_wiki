# 宝箱配置

目前平台设置**2 种宝*：SSR 宝箱UR 宝箱

- **SSR 宝箱**：商城购买，1 张抽奖券 = 800 U币，可以使用地图积分和金币兑
- **UR 宝箱**：商城购买，1 张抽奖券 = 1200 U币，需要至少上架一SSR 宝箱才能创建

## 奖池概率

在【线上宝箱配置】中可以设置宝箱的中奖概率。注意：概率配置不能配置 N、R、SR、SSR 的整体概率，仅能配置每个等级中道具概率。宝箱每次上**七天内不能修*其中的道具及概率

| 道具库级| 道具件数要求 |
|-----------|-------------|
| N 级库 | 10 |
| R 级库 | 5 |
| SR 级库 | 3 |
| SSR 级库 | 1 |
| UR 级库 | 1 |

四大类别里的道具*权重**来区分概率，权重分布 1-9

- 权重设为 1：此道具价值更高，抽中概率越小
- 权重设为 9：此道具价值更低，抽中概率越大

## 配置流程

**Step1**：【工作台】→【运营能力】→【活动运营】→【地图宝箱

![地图宝箱](/kk-guide/202410160104.png)

**Step2**：创SSR/UR 宝箱之前需要创建基础宝箱，点击【编辑】创建基础宝箱，在宝箱设置中配置相关道具

![编辑宝箱](/kk-guide/202410160105.png)

**Step3**：基础宝箱配置完毕后，点击【创建宝箱】，选择【创建SSR宝箱【创建UR宝箱】进行配置

![创建宝箱](/kk-guide/202410160106.png)

**Step4**：开发者通过地图促活方式吸引更多玩家关注。通过签到、活动设置等措施激发用户参与热情。平台地图活动分为三种类型：**常规活动、抽奖券日常、抽奖券周常**。开发者可根据地图所需自行配置

![活动配置](/kk-guide/202410160107.png)
