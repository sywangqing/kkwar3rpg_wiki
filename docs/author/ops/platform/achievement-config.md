# 成就配置

## 什么是地图成就

成就是激励玩家深入挖掘游戏并提高游戏水平的一种富有趣味的方式，开发者可以定期添加成就，以保持玩家对游戏的新鲜感并鼓励玩家持续参与

## 界面介绍 & 规则

- **成就名称**：不能超25 个字
- **成就说明**：成就简要说明，告知玩家获取方式
- **成就 ID**：编辑器中接口上报成就的唯一标识，创建后自动生成随机 6 ID
- **成就类型**
  - **连续成就**：用户通过多次执行同一个任务事件来逐步解锁成就，进程开始后事件必须具有连续性。比连续赢得 20 次对局"，若玩家输掉游戏，其连续状态则不成
  - **累计成就**：用户通过单次或多次同一个任务事件累计来解锁成就，进程开始后事件累计计算，事件可不具有连续性。比累计赢得 20 次对局"，若玩家输掉游戏，成就仍会累计计
- **达成次数**：用户解锁成就前，需要完成的预设次数

## 概述

- 开发者通过 API 接入"成就系统"，可快速配置游戏内的成就体
- 当玩家达到开发者设定条件后，开启相应成就，且玩家取得的成就可以在平台集中展
- 成就创建后可在列表中获取成就 ID，此 ID 可用于后续的成就功能接口开
- 每张地图可配置的成就点数**最大为 30**，不限制成就数量，请合理策划地图成就

## 流程

接入准备 成就配置 提交审核 正式上架

## 管理操作

### 上架成就
审核通过后，成就会自动上架。当成就被开发者主动下架，可在列表中点击【上架】，状态变为：已上架

### 下架成就
已上架的成就可被下架。成就下架后，不影响已经获得成就的玩家，但没有获得成就或正在解锁成就的玩家将不会获得成就*建议谨慎下架成就**。下架后的成就可以更新成就信息，提交后通过审核再上架

### 重新编辑成就
没有上架过的成就，可重新编辑。点击【重新编辑】，所有字段均可重置。点击提交，重新进入审核

### 更新成就
已上架的成就可通过先下架成就进行更新。可更新的字段为*成就名称、成就说明和成就图标**。不可更新的字段*成就类型、成就任务初始状态和成就点数**（一旦初始设置，则不可再修改）。更新提交后进入审核，审核通过自动上架

### 删除成就
未上架过的成就可以删除，在列表中点击【删除】。成就一旦上架平台后*则不可删*

### 查看成就数据
地图成就上架后，可以查看已发布的成就数据"完成人数 & 比例"。通过分析成就数据，了解玩家的成就进度和成就的难易程度，从而优化游戏的设计方案
