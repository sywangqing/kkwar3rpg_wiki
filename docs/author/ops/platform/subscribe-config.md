# 预约配置

## 预约奖励

上架测试大厅前配置，包括预约礼包、测试礼包、等级礼包

![预约奖励](/kk-guide/yuyue.png)

- **预约礼包**：设置一个奖励提供给预约该游戏的玩家。当游戏正式上架后，玩家登录平台后会弹出提示，收件箱中也会收到游戏上架的消息。奖励可以是称号、特权，用来激励玩家预约
- **测试礼包**：吸引玩家参与地图测试的奖励手段，可选择是否开启。测试奖励可以是一个称号、礼包或任何道具，测试奖励将在地图正式上线时得以保留
- **等级礼包**：吸引玩家参与地图测试的奖励手段，可选择是否开启。等级礼包可以是一个称号、礼包或道具，将在地图正式上线时得以保留

## 预约有奖

上架测试大厅后配置。通过预约梯度奖励活动，可以提升用户预约量。在测试期间地图达到梯度后，用户有机会获得奖励（配合 API 使用）*至少配置 4 个预约节*

![预约有奖](/kk-guide/yuyueyoujiang.png)

**配置流程**：点击【运营能力】→【活动运营】→【预约活动

![步骤四](/kk-guide/s4.png)
