# 关于地图寻宝

地图寻宝是平台为地图提供的一*付费抽奖机制**，玩家消U币参与寻宝，有概率获得各类商品

## 核心逻辑

- 寻宝消U
- 寻宝奖品从作者配置的宝箱奖池中抽
- 重复获得永久商品自动分解为地图积

## 寻宝配置要求

- 需要先配置宝箱（SSR/UR 宝箱
- 宝箱上线后七天内不可修改道具及概
- 各等级库最少道具要求：N0、R、SR、SSR、UR
