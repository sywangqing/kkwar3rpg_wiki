# 策略制定框架梳理

商业化设计从问题解决的策略角度来看分别为*"用户策略"产品策略"场景策略"**

## 用户策略

首先需要做*用户分层细分**，再明确用户的偏好情况，通过在局内制造需求或痛点卡点，来设计对应的商品内容，最后结合促销活动或其他形式来促成用户的付费

在具体商品设计的时候，需要考虑如何打*易于感知的权益感受、收益感*，在用户购前信息收集的过程中，可以清楚了解商品带来的体验上变化

从满足用户情绪需要的方面看，需要满足用户的情绪需要，为其打造个性化的体验满足；在售卖环节要制买的合适、性价比高"的感受，才能有足够理由去强化他的付费动机

## 产品策略

- 商品整体定价要规划清晰，成体系方便用户找到自己需要商品对应的价格区间
- 地图第一批商品内容的整体定价，在一定程度上决定了用户对地图产品的口碑评价及观感
- 商品内容规划要成体系，有一定的一致性，方便用户购前挑
- 在效果与定价上可以有一定的对比，高价值与低价值商品效果区分足够明
- 商品的内容设计上要投放关键的属性和功能，投放的内容要清晰且与系统的关联度要

## 场景策略

- **局*：设计可激发用户购买需求的卡点、障碍点、竞争氛围。结合内容、排名、数值压力考核强化用户的付费动
- **商城**：通过定价策略激发用户的性价比考量，另外可结合限时收集、限量、限购等方式二次强化用户的付费欲
