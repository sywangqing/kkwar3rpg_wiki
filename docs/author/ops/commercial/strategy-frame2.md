# 商业化策略制定的框架

综上所述，"用户策略"产品策略"场景策略"分别有以下特点：

- **用户策略**：洞悉用+ 满足需+ 强化动机
- **产品策略**：定价策+ 体系规划 + 内容投放
- **场景策略**：制造痛+ 营造冲+ 感受对比

策略制定完成后，建议根据自身地图产品的情况，去分析下平台内的相关竞品的情况，再去推导自身产品商业化策略的可行性。最终结KK 官方对战平台商业化规则，去制定最终的落地方案
