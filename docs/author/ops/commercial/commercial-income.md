# 商业化规

地图创作者需要遵KK 官方对战平台商业化既有逻辑规则，去设计商业化内容。主要规则框架与逻辑涉及：商品类型、商品定价规则、宝箱抽卡机制、货币类型、货币转换规则等内容

## 收入组成

平台内创作者商业化收入的组成为*直售商品 + 地图寻宝 + 平台地图商品**
